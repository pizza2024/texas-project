## 竞品调研（CoinPoker / GGPoker / WSOP / 888poker）

### 主要竞品对比

| 平台          | 独特功能                          | 变现模式                  | UI 特点                |
| ------------- | --------------------------------- | ------------------------- | ---------------------- |
| **CoinPoker** | 区块链 + USDT 充值 / 提现 + 匿名  | 抽水 (5%) + 代币经济      | 现代深色主题，HUD 友好 |
| **GGPoker**   | SmartHUD / 牌局回顾 / AI 对手检测 | 抽水 (5-6%) + Jackpot SNG | 界面复杂，功能密集     |
| **WSOP**      | 品牌赛事授权 / 戒指赛 / 手镯象征  | 报名费抽水 + 内购         | 明亮视觉，金色元素     |
| **888poker**  | SNAP 快速桌 / 桌面共享            | 抽水 (3-5%)               | 简洁轻量，教程友好     |

### 关键竞品洞察

1. **变现模式共识**：主流平台均采用"抽水 + 报名费"双轨制，无纯免费模式
2. **区块链优势**：CoinPoker 的 USDT 充值即时到账，大幅降低摩擦
3. **GGPoker 的差异化**：智能工具（SmartHUD）增强粘性，但自研需要投入
4. **UI 设计趋势**：深色主题 + 最小化操作步骤 + 一键快速加入
