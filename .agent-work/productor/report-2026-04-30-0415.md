### 二、竞品资料页 UX 对标

#### GGPoker

**VIP 等级系统：**
- 多级 VIP：Bronze → Silver → Gold → Platinum → Diamond → Crown
- 每个等级有明确的下注量/手数门槛
- 等级特权：更高返水、专属客服、优先入场、生日奖金
- 进度条显示当前等级和下一级差距
- 实时显示本周期累计贡献和距离下一级还需贡献

**Equity 曲线 / 数据统计：**
- Dashboard 显示：总手数、赢率（BB/100）、总盈亏
- 按局种（Hold'em / Omaha）、按级别（盲注）分类统计
- 月度盈亏曲线图（可切换时间范围）
- 输赢分布直方图
- 最常见 Bad Beat 统计

**战绩分享：**
- 手牌分享：每手牌生成唯一链接（类似 `ggpoker.com/hand/xxxxx`）
- 分享到社交媒体：Facebook / Twitter / Reddit
- 手牌回放内嵌分享
- 玩家可生成本人战绩链接（公开 Profile 页面）

---

#### 888poker

**888 Club（VIP体系）：**
- 888poker Club：铜牌/银牌/金牌/白金/钻石 五级
- 通过 rake 积累提升等级
- 每月重置等级，需持续贡献维持
- 专属锦标赛门票、现金返还等奖励

**统计面板：**
- SNAP 统计（快速桌特有）：每秒出手速度、弃牌率
- 基础盈亏图（周/月视图）
- 赢率统计（按手数、按局种）

**战绩分享：**
- 手牌历史可导出（CSV/文本格式）
- 888poker 社区可发布战绩讨论
- 无原生社交分享链接

---

#### WSOP（Caesars Digital）

**WSOP Rewards 忠诚计划：**
- 积分制（Comps）：根据 rake/贡献计算
- 等级：Gold → Platinum → Diamond → Seven Star（顶级）
- Seven Star 为邀请制，稀缺性极高
- 积分可兑换：锦标赛门票、现金、WSOP 周边商品
- 跨品牌积分（M life Rewards）：可在 Caesars 全平台使用

**资料页统计：**
- 基本 P&L 图（累计盈亏）
- 锦标赛成绩：买入/回报/利润
- 现金桌统计：手数/赢率/总盈亏
- 相对简洁，无详细 equity 曲线

**战绩分享：**
- 局限性明显：无原生手牌分享链接
- 战绩主要通过截图传播
- 社区工具辅助（HHIntel 等第三方导入）

---

#### CoinPoker（Crypto 原生平台）

**CPD Coins 奖励体系：**
- 基于贡献（rake）派发 CPD Coins
- CPD 可兑换 USDT 或用于参赛/购买周边
- VIP 高达 60%+ rakeback（高流量玩家）
- 无传统等级制度，更侧重即时返利

**Equity 曲线 / 统计（最详尽）：**
- 完整 Dashboard：总手数、BB/100 赢率、累计盈亏（USDT）
- **原生 Equity Curve 图表**：累计利润随时间变化
- 按局种、按级别分类统计
- Session 历史记录：日期/时长/结果
- 日期范围筛选器（自由选择统计区间）

**手牌回放与分享（最具社交特色）：**
- 内置 **Hand Replayer**：复盘任意历史手牌
- **Shareable Hand Links**：每手牌生成唯一可分享 URL
- 分享链接展示所有 hole cards + 完整行动过程
- 支持社交媒体直接分享
- 内置 HH Converter：导出标准格式供第三方分析工具使用

---

#### BetMGM Poker

**MGM Rewards 忠诚体系：**
- MGM Resorts 跨品牌统一积分（与酒店/餐饮/娱乐打通）
- 等级：Sapphire → Pearl → Gold → Platinum → Diamond → Noir（邀请制）
- Poker 贡献计入 MGM Rewards 总积分
- 高级权益：免费停车、优先入住、专属活动邀请

**统计面板：**
- 基础统计：总手数、锦标赛成绩、现金桌盈亏
- 与 BetMGM 博彩平台整合，展示综合账户活动
- **无原生 Equity Curve** — 不如 CoinPoker 详尽

**手牌回放/分享：**
- 无内置手牌回放器
- 无原生分享功能
- 通过 Borgata Poker 网络兼容第三方 Tracker（HHNinja 等）

### 四、P2-PROFILE-001~003 规格建议

#### P2-PROFILE-001：VIP 等级系统

**建议方案（三选一）：**

**方案 A — 传统多级 VIP（对标 GGPoker）：**
```
等级：Bronze → Silver → Gold → Platinum → Diamond
门槛：月累计贡献（rake）划分
权益：更高返水% + 专属客服 + 优先入场 + 生日奖励
显示：个人资料卡 + 进度条（距下一级距离）
```

**方案 B — 积分累计制（对标 CoinPoker CPD）：**
```
积分：CPD Coins 累计，不设等级上限
兑换：USDT / 锦标赛门票 / 周边商品
实时：当前余额 + 本周期积分
```

**方案 C — 混合制（对标 WSOP MGM Rewards）：**
```
等级特权：独家活动邀请、专属锦标赛门票
积分通用：积分可在多个场景使用（充值折扣/兑换门票/提现手续费减免）
```

**推荐：** 方案 C 兼顾拉新和留存，积分通用性提升用户粘性

#### P2-PROFILE-003：战绩分享功能

**建议方案（核心链路）：**

1. **手牌分享链接：**
   - 生成唯一 URL：`/hands/[handId]/share`
   - 页面展示：所有 hole cards + community cards + 行动历史
   - 无需登录即可查看（公开数据）
   - 可生成分享图片（OG image / Twitter card 优化）

2. **社交分享按钮：**
   - 分享到 Twitter/X：自动生成文本 + 链接
   - 分享到 Facebook：Open Graph 标签支持
   - 复制链接：Click-to-copy 体验

3. **战绩页面（可选）：**
   - `/profile/[username]` 公开主页
   - 展示：总手数、赢率、盈亏曲线、近期大手牌
   - 可分享到社交媒体作为个人品牌

**数据结构：**
```ts
// HandShare 生成格式
GET /api/hands/[handId]/share
{
  handId: string;
  players: { username: string; holeCards: string[]; finalStack: number }[];
  communityCards: string[];
  actions: { username: string; action: string; amount: number }[];
  pot: number;
  winner: string;
  createdAt: string;
}
```

**竞品参考：** CoinPoker 的 shareable hand links 最成熟；GGPoker 提供完整 profile 页面
