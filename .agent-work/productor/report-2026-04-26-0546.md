## 竞品调研：BTC 赛事格式

### Beat the Clock（T BTC）是什么

**BTC（Beat the Clock）** 是 ACR Poker 独创的时间驱动锦标赛格式，后被多个平台借鉴。其核心机制：

1. **定时启动**：每 N 分钟开一组（ACR 通常 15/30/60 分钟）
2. **6人 SNG**：每组固定 6 人
3. **时间压力**：每个盲注级别时间固定，到时间自动升盲
4. **随机奖池倍数**：开赛前随机决定奖池乘数（1x~10000x），普通 SNG 变为"彩票"
5. **极高参与度**：因为奖池倍数随机，玩家愿意频繁参赛

### ACR Poker BTC 赛事特点

| 特性     | 描述                   |
| -------- | ---------------------- |
| 买入     | $1~$30 多档            |
| 格式     | 6人 SNG，时间驱动升盲  |
| 奖池倍数 | 随机 1x~10000x         |
| 节奏     | 极快（15~30 分钟/场）  |
| 目标用户 | 休闲玩家，追求"彩票感" |

### CoinPoker 的创新点

- **加密货币原生**：CHP token 作为游戏货币，USDT 充值即时到账
- **CoinRewards**：类似 rakeback 的忠诚计划
- **随机奖池**：类似 BTC 的彩票机制

### 对本项目的借鉴意义

BTC 赛事对 Texas Hold'em 项目的价值：

1. **用户留存**：高频短时赛事创造"再来一局"循环
2. **收入提升**：高参与度带来更多 rake
3. **差异化**：Crypto + BTC = 独特竞争力
