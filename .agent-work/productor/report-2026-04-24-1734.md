## 竞品调研：ClubGG 深度分析

### ClubGG 核心功能

ClubGG（Club Game）是一个以**俱乐部系统**为核心的去中心化德州扑克平台，允许用户创建和管理自己的私人扑克俱乐部。

#### 1. Club 系统（核心差异化）

| 功能         | 说明                                       |
| ------------ | ------------------------------------------ |
| 俱乐部创建   | 任何人可创建俱乐部，设置名称、logo、描述   |
| 邀请码/链接  | 每个俱乐部有唯一邀请码，管理员可分发给玩家 |
| 私有房间     | 俱乐部内创建私有桌，仅对俱乐部成员开放     |
| 俱乐部排行榜 | 按 ELO/盈利/游戏场次排名，增强社交竞争     |
| 俱乐部等级   | 基于活跃度/玩家数自动升级，解锁更多功能    |

#### 2. 俱乐部管理员功能

| 功能      | 说明                                                |
| --------- | --------------------------------------------------- |
| 成员管理  | 审批加入申请、踢出成员、设置角色（管理员/普通成员） |
| 房间配置  | 设定盲注级别、准入门槛、最大人数、密码保护          |
| Rake 设置 | 俱乐部可设置自己的 rake 比例（平台抽水外额外抽成）  |
| 公告/通知 | 向俱乐部成员发送公告                                |
| 数据统计  | 俱乐部总游戏场次、活跃度、成员增长趋势              |

#### 4. ClubGG 成功的关键因素

1. **社交黏性**：俱乐部形成封闭社交圈，玩家有归属感
2. **低准入门槛**：免费创建俱乐部，任何人可运营
3. **经济激励**：俱乐部 rake + 平台代币奖励，形成正向循环
4. **私密性**：私人牌局满足不想公开游戏的玩家需求
5. **病毒式传播**：邀请码机制驱动自然增长

#### 5. 对本项目的建议

**Club 系统 MVP 方案（Phase 1）**：

1. **Club 表结构**：id, name, description, avatar_url, owner_id, invite_code, created_at
2. **ClubMember 表**：club_id, user_id, role (OWNER/ADMIN/MEMBER), joined_at
3. **私有房间**：Room.is_private, Room.club_id, Room.invite_code
4. **邀请码**：6位字母数字，URL 分享格式 `/join/{invite_code}`
5. **俱乐部页面**：俱乐部主页、成员列表、排行榜、私有房间列表

**差异化建议**：

- ClubGG 无 Web3 充值，本项目可结合 USDT 充值优势
- 俱乐部等级体系 + USDT 奖励池，形成 Web3 原生社交扑克
