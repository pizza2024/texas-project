#### 发现的产品问题

**P-UX-1: 缺少 Pot Odds 显示**

- **严重程度**: 中
- **描述**: 当前动作栏没有显示 Pot Odds（跟注赔率），这是玩家决策的核心参考数据
- **竞品**: PokerStars、WSOP、GGPoker 均标配
- **计算公式**: `potOdds = callAmount / (pot + callAmount)`
- **建议**: 在 Call 按钮旁边或底部显示，如 "Odds: 33%" 或 "1:2"
- **工作量**: 0.5-1天

**P-UX-2: Sit-Out 位置过于显眼**

- **严重程度**: 低-中
- **描述**: Sit-Out 按钮与其他动作按钮并列在同一行，高级玩家才需要的选项对新手干扰
- **竞品**: WSOP/PokerStars 将 Sit-Out 放在设置菜单或玩家头像菜单内
- **建议**: 将 Sit-Out 移至右上角玩家菜单或设置齿轮图标内
- **工作量**: 0.5天

**P-UX-3: 缺少轮次指示器**

- **严重程度**: 低
- **描述**: 玩家无法快速判断当前是 Preflop/Flop/Turn/River
- **竞品**: WSOP 在牌桌上明显标注
- **建议**: 在牌桌顶部或计时器旁边显示 "FLOP" / "TURN" 等标签
- **工作量**: 0.5天

**P-UX-4: Raise Input 宽度受限**

- **严重程度**: 低
- **描述**: Raise Input (w-24 = 96px) 和 Raise 按钮在同一行，在小屏幕上可能显示不全
- **竞品**: GGPoker 使用右侧滑出面板放置 Raise 控制
- **建议**: 考虑将 Raise Input 移到第二行，或者在大屏幕上使用更大宽度
- **工作量**: 0.5天
