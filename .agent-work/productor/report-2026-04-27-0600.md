## 本轮调研主题：GGPoker / SPINS 即时赛事产品分析

### GGPoker SPINS 产品洞察

SPINS 是 GGPoker 发明的一种极速 SNG（Sit & Go）赛事格式，特点是：

**核心机制**

- 3人桌，极速开始（满员即开）
- 随机奖池：买入x2～x10000
- 时间短：通常3-5分钟一局
- 淘汰制：2人晋级即结束

**产品设计亮点**

1. **随机奖池揭晓动画**：开赛前一秒揭晓奖池倍率，极具悬念感
2. **极速匹配**：点击"SPINS"按钮，3秒内自动分配座位
3. **VIP Club**：Spin填充（必中大奖）/Jackpot SPINS（超高倍率）
4. **社交整合**：Club系统（工会/联盟），共享战绩

**UX 关键洞察**

- SPINS 入口在主导航栏第二位（仅次于"扑克"）
- 买入金额选择：$0.25 / $1 / $3 / $5 / $10 / $25
- 实时显示"正在等待"人数，强化参与感
