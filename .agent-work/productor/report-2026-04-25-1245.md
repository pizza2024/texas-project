## 竞品调研 — 本轮专题：GGPoker Squash 手牌复盘

### Squash 功能分析

GGPoker 的 **Squash**（也被称为 "Replayer"）是其手牌复盘系统的核心，提供以下能力：

| 功能 | 说明 |
|------|------|
| 时间轴播放 | 逐步回放每轮下注（Preflop → River） |
| 手牌对比 | 同时显示你和对手的手牌 |
| Pot Odds 叠加 | 每轮显示当前 pot odds |
| 胜率动态 | 每轮实时更新你 vs 对手的胜率百分比 |
| 决策标记 | 可标注"关键决策点"供后续学习 |
| 分享功能 | 生成复盘链接分享给好友 |

### Squash UI 设计要点

1. **双栏布局**: 左侧牌桌动画，右侧手牌+动作历史
2. **时间轴控制条**: 拖拽或点击跳转任意轮次
3. **胜率条**: 横向显示实时胜率变化曲线
4. **玩家标记**: 高亮当前行动玩家头像

### 对本项目的启示

本项目 P2-NEW-007（战后手牌复盘）可参考 Squash 设计：

**建议技术实现路径**:
- 后端：存储 `HandAction` 序列，前端重放
- 胜率：复用 `hand-evaluator.ts` 实时计算
- UI：参考 Squash 双栏，左侧牌桌状态，右侧动作时间轴

**优先级建议**:  MVP 版本优先实现"时间轴 + 手牌对比"，胜率曲线作为 V2
