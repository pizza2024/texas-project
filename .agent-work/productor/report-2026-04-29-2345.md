## 本轮调研专题

**专题：忠诚积分 / VIP 体系竞品深度分析**

本项目目前缺少忠诚积分和 VIP 体系，这是留存玩家的重要手段。对标竞品：

### 竞品忠诚积分体系

| 平台           | 体系名称   | 核心机制                        | 亮点                    |
| -------------- | ---------- | ------------------------------- | ----------------------- |
| **GGPoker**    | GGPoints   | 每耗佣金1美元=1分，月清零       | 78%用户参与，里程碑徽章 |
| **PokerStars** | StarsCoin  | 任务池+每周重置                 | 67%用户参与，可兑实物   |
| **888poker**   | 888 Club   | 三级VIP（铜/银/金）+Club Points | 50%渗透率，月度奖励     |
| **ACR Poker**  | Rewards    | 即时返利+赛季积分               | 100%覆盖，高额返利      |
| **CoinPoker**  | CHP Tokens | 链上积分，可交易                | Web3差异化              |
| **本项目**     | ❌ 无      | —                               | 差异化机会              |

### 积分体系核心组件（本项目建议）

```
1. 点数货币：CHIPS_Rakeback（每周返水）
2. VIP等级：Bronze/Silver/Gold/Platinum（基于月累计充值或游戏佣金）
3. 里程碑奖励：达成特定手数/盈利目标解锁徽章
4. 兑换商店：门票、筹码包、专属头像框
```

### 本项目已有基础（可复用）

- `Transaction` 模型：DEPOSIT/WITHDRAW/GAME_WIN/GAME_LOSS 已覆盖
- `User.role` + `User.status`：VIP等级字段可扩展
- 赛程系统（P1-SCHEDULE-001）：可与赛事绑定做"赛事积分双倍"
- 通知中心（规划中）：可推送"您的黄金VIP已解锁"等

### 实施建议优先级

| 优先级 | 功能               | 理由                             |
| ------ | ------------------ | -------------------------------- |
| 🔴 高  | 周返水（Rakeback） | 最直接的留存手段，竞品都有       |
| 🟡 中  | VIP 等级制度       | 基于月累计充值或佣金，提升忠诚度 |
| 🟢 低  | 兑换商店           | 需要商品库存管理，复杂度高       |
