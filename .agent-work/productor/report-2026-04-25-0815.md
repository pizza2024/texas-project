## Club 私人邀请码 — 产品设计草案

### DB Schema 建议

```prisma
model ClubInviteCode {
  id        String   @id @default(uuid())
  clubId    String
  code      String   @unique  // 6-8位字母数字
  creatorId String
  maxUses   Int      @default(1)
  usedCount Int      @default(0)
  expiresAt DateTime?
  createdAt DateTime @default(now())
  club      Club     @relation(fields: [clubId], references: [id])
}
```

### API 端点建议

| 方法   | 路径                              | 说明                 |
| ------ | --------------------------------- | -------------------- |
| POST   | `/clubs/:id/invite-codes`         | Owner 生成邀请码     |
| GET    | `/clubs/invite-codes/me`          | 用户查看自己的邀请码 |
| POST   | `/clubs/join-by-code`             | 用户凭邀请码加入     |
| DELETE | `/clubs/:id/invite-codes/:codeId` | Owner 删除邀请码     |

### 前端流程

1. Club Detail 弹窗新增 **"邀请"** 标签页（Owner/Admin 可见）
2. "生成邀请码" 按钮 → 可选 maxUses + expiresAt
3. 邀请码列表 + 复制链接
4. 未登录用户访问 `/club/join?code=XXXX` → 引导注册/登录后自动加入
