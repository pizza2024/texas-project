### 本项目 Blast 赛事建议规格

```
名称: BLAST
类型: 即时匹配 SNG
参赛费: 100 / 1,000 / 10,000 chips（对应微/小/中额）
人数: 3人
奖金池随机倍率:
  - 50%: 2× buy-in（保本）
  - 30%: 5× buy-in
  - 15%: 20× buy-in
  - 4.9%: 100× buy-in
  - 0.1%: 10,000× buy-in（Jackpot）
盲注结构: 每90秒涨一次（极速）
允许ante: 是
Straddle: 否
Re-entry: 否（单次生命）
取消条件: 3人未满则取消，退还报名费
```

### UI 差距

1. **牌桌信息密度** — GGPoker 在牌桌上方显示 ELO/VPIP/PFR HUD，本项目无玩家数据统计
2. **底池 Odds 实时显示** — GGPoker 每回合显示 pot odds 和 outs，本项目无此功能
3. **手牌历史热图** — GGPoker 的 "Hand History Heatmap" 可视化输赢，本项目仅文字记录
4. **赛事气泡提示** — GGPoker 赛事中悬浮气泡提示操作时限，本项目无
