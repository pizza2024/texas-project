### MTT 核心功能缺失（与竞品对比）

#### 1. 延迟注册（Late Registration）

WSOP/GGPoker 均支持比赛开始后一定时间内持续注册，典型为 Level 4-6（30-60分钟）。

| 功能         | 本项目 | WSOP             | GGPoker          |
| ------------ | ------ | ---------------- | ---------------- |
| 延迟注册     | ❌ 无  | ✅ 最多 6 个级别 | ✅ 最多 8 个级别 |
| 注册截止展示 | ❌ 无  | ✅ 倒计时        | ✅ 倒计时        |

**规格建议：**

- 新建 `Room.isLateRegistration = true` + `Room.lateRegLevel = number`
- 前端 tournament detail 页面显示 "Register available for X more levels"
- 延迟注册玩家初始筹码与原注册玩家相同（不设初始劣势）

#### 2. Re-Entry（再买入）

GGPoker/WSOP 均支持比赛淘汰后重新买入，典型限制为 1-3 次。

| 功能              | 本项目 | WSOP       | GGPoker        |
| ----------------- | ------ | ---------- | -------------- |
| Re-Entry          | ❌ 无  | ✅ 1次     | ✅ 2-3次       |
| Re-Entry 截止时间 | ❌ 无  | 注册截止时 | 延迟注册截止时 |
| Re-Entry 费用     | ❌ 无  | 同初始买入 | 同初始买入     |

**规格建议：**

- `TournamentRegistration.reEntryCount: number` 字段
- `TournamentPlayer.eliminatedAt: DateTime?` 标记淘汰时间
- 淘汰后弹窗："You've been eliminated. Re-enter for $X? (Y re-entries remaining)"
- `MAX_REENTRIES` 配置在 Room model

#### 3. 奖金结构（Payout Structure）

| 功能             | 本项目 | WSOP                  | GGPoker       |
| ---------------- | ------ | --------------------- | ------------- |
| 奖金分配         | 基础   | 9-max/10-max 独立配置 | 15 种结构可选 |
| GTD（保底奖池）  | ✅ 有  | ✅ 有                 | ✅ 有         |
| Bounties（赏金） | ❌ 无  | ❌ 无                 | ✅ 有         |

**规格建议：**

- `Room.payoutStructure: 'standard' | '8-max' | '10-max' | '15-max'`
- `Room.bounty: number` 淘汰奖励（可选）
- 前端 tournament card 显示奖金分布表（可展开）

#### 4. 重新报名（Re-Register）与 Late Join

| 功能                 | 本项目 | WSOP        | GGPoker         |
| -------------------- | ------ | ----------- | --------------- |
| Late Join（晚加入）  | ❌ 无  | ✅ 补偿筹码 | ✅ 等价起始筹码 |
| 重新报名（重新坐下） | ❌ 无  | ✅ 同一级别 | ✅ 同一级别     |

### 竞品 MTT 功能详细对比

#### WSOP Tournament Features

- 延迟注册：最多 6 个级别（每级 20-30 分钟）
- Re-Entry：单次，淘汰后立即可购买
- 重新坐下：淘汰后同一级别可重新报名
- 奖金分配：标准化 9 人桌结构，前 27% 获奖
- 卫星赛：可通过卫星赛获得主赛资格

#### GGPoker Tournament Features

- 延迟注册：最多 8 个级别
- Re-Entry：最多 3 次
- 多种比赛类型：Regular、Mystery Bounty、Progressive Bounty、Knockout
- 实时人数更新：展示当前注册人数和剩余名额
- 多日赛：主赛支持多天赛程
