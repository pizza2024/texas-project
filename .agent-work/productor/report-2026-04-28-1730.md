## 竞品动态：本轮专题

### GGPoker Spin & Gold 深度分析

**产品定位：**
- 3人极速 SNG，60秒超时强制出桌
- 随机奖池倍数 2x~12000x，入场费固定
- 核心变现：rake + 中途退出罚没

**轮盘设计（视觉参考）：**
```
档位分布：2x(36%) / 3x(28%) / 5x(20%) / 10x(10%) / 25x(4%) / 50x(1.5%) / 100x(0.5%) / 250x(0.1%) / 12000x(极稀有)
旋转机制：3D CSS transform，5-8秒减速动画 + 指针弹跳音效
视觉特效：光晕粒子、金属质感、12000x时全屏金色爆炸特效
```

**用户体验亮点：**
1. 匹配等待期间显示"等待对手..." + 倒计时
2. 轮盘结果触发"WINNER"大字 + 奖池数字飞涨动画
3. 被淘汰即时显示"You placed #2/3 — refunded $X"
4. 超级大奖触发全服广播（匿名化）

**对我们 BLAST 的启示：**
- SpinWheel 需要 9 档位 + 12000x 超级大奖
- 轮盘动画需要光效/粒子系统
- 淘汰结果页需要即时退款显示
- 12000x 结果应触发特殊庆祝动画（screen flash + confetti）

### ACR Poker "Beast Jackpot" 对比

**Jackpot 触发条件：**
- 必须是 bad beat（四条 vs 更大四条）
- 底池必须包含所有玩家的全押
- 双方必须亮牌

**分配机制：**
- 受伤方（输掉四条）：50%
- 获胜方（四条更大）：25%
- 桌上其他玩家：25%

**与 BLAST 的互补性：**
- BLAST = 快速博彩体验（2-5分钟一局）
- Bad Beat Jackpot = 长局现金桌附加奖励
- **建议：** BLAST 和 Bad Beat Jackpot 可共存于同一平台
