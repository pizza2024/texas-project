## 竞品调研 — 888poker

### 888poker 概览

888poker 是全球第三大在线扑克平台，主打休闲玩家市场，2024年进行了大规模 UI 改版，突出"更简单、更快、更社交"的定位。

### 888poker 关键产品特性

| 功能                      | 说明                     | 可借鉴性                |
| ------------------------- | ------------------------ | ----------------------- |
| **快速扑克 (Snap Poker)** | 一键弃牌后立即换到新桌   | ⭐⭐⭐⭐ 直接参考       |
| **Beat the Clock**        | 时间驱动的锦标赛门票活动 | ⭐⭐ 活动机制           |
| ** клуб 系统**            | 私人扑克俱乐部/社群      | ⭐⭐⭐ 与 Club 功能重叠 |
| **牌局回放**              | 历史手牌查看器           | ⭐⭐⭐ MVP 已覆盖       |
| **社交功能**              | 好友列表、聊天、成就徽章 | ⭐⭐ 增强留存           |

### Snap Poker vs 本项目 Solo 模式

| 特性     | 888poker Snap   | 本项目 Solo |
| -------- | --------------- | ----------- |
| 快速离桌 | ✅ 弃牌后秒换桌 | ✅ 已实现   |
| 筹码保留 | ❌ 不保留       | ⚠️ 待确认   |
| 匹配时间 | <5秒            | 待测        |
| UI 体验  | 一键操作        | ✅ 基础可用 |

### 对本项目的建议

1. **Solo 模式优化**: 对标 Snap Poker，确保离桌/入桌体验流畅（<5秒）
2. **Club 社群化**: 参考 888poker 俱乐部系统，强化社交属性（好友聊天、成就）
3. **Beat the Clock 活动**: 定时锦标赛门票发放，提升日活
