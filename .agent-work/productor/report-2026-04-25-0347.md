## 本轮关键发现

### Rakeback MVP 实现了什么

这是 P1-004 的后端核心，功能完整且测试覆盖充分：

```
每局结算时:
  1. 计算每个赢家的 rake share = (winAmount / totalWinAmount) * totalRake
  2. 更新 winner.totalRake += rakeShare
  3. 调用 RakebackService.creditRakeback(winnerId, rakeShare)
  4. winner.rakebackBalance += floor(rakeShare × tierRate)
```

### Rakeback MVP 缺少什么

**前端完全空白** — 这是一个完整的产品功能缺口。玩家打完几百局后 rakebackBalance 积累到 thousands，但完全看不到这个数字。没有 UI = 功能不存在。
