## 本轮专题调研：Rake（抽水）系统

### Rake 行业标准模式

| 模式                             | 说明                      | 竞品参考                   |
| -------------------------------- | ------------------------- | -------------------------- |
| **Pot Rake（比例抽水）**         | 每局结束后从底池抽取 2-5% | PokerStars、WSOP、888poker |
| **Time Collection（计时抽水）**  | 按座位时间收费（$/30min） | 部分线下场景               |
| **Tournament Fee（比赛报名费）** | 固定买入的 5-10% 作为费用 | 所有锦标赛                 |
| **No Rake（零抽水）**            | 俱乐部模式可能免抽        | CoinPoker 私人局           |

### 主流平台 Rake 结构

#### PokerStars（行业基准）

- **比例**: 3.5% - 5%（根据级别不同）
- **封顶**: $0.30 - $3.00 每手（根据级别）
- **特殊情况**: 如果只两人对战且无 Flop，rake 减半或免收
- **超时**: 无

#### 888poker

- **比例**: 3% - 5%
- **封顶**: $1 - $2 每手
- **特色**: "Direct Rakeback" 返还 30-50%

#### CoinPoker（Crypto）

- **比例**: 2% - 4%
- **封顶**: 动态封顶（CHP 代币持有者降低封顶）
- **特色**: 可用 CHP 代币支付 rake，获得额外返利

### 本项目 Rake 建议方案

#### 推荐：Pot Rake 模式（竞品最通用）

```
盲注级别 → Rake 比例 → 单手封顶
MICRO    → 5%       → $0.30
LOW      → 4%       → $0.50
MID      → 3.5%     → $1.00
HIGH     → 3%       → $2.00
PREMIUM  → 2.5%     → $3.00
```

#### 数据库变更需求

| 变更                    | 说明                                   |
| ----------------------- | -------------------------------------- |
| `Hand.rake`             | 本局总抽水金额                         |
| `Hand.rakePercent`      | 抽水比例（0.025 - 0.05）               |
| `Settlement.rakeAmount` | 用户承担抽水（分摊）                   |
| `User.totalRake`        | 用户累计贡献抽水（用于 Rakeback 计算） |

#### Rakeback（返水）建议

| 等级     | Rakeback 比例 | 月度 Rake 要求 |
| -------- | ------------- | -------------- |
| Bronze   | 10%           | $0             |
| Silver   | 20%           | $50            |
| Gold     | 30%           | $200           |
| Platinum | 40%           | $500           |

**注意**: Rakeback 是提升用户留存的关键机制，竞品 WSOP/888poker/CoinPoker 均提供 Rakeback。
