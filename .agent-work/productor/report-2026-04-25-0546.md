## 本轮产品洞察

### 1. SNG Phase 1 细化建议

基于 GGPoker SNG 功能研究，Phase 1 SNG 应包含：

**必须包含 (MVP)**
- 6人标准 SNG（无注册费/低买入）
- 盲注结构：Standard（15分钟）/ Turbo（7分钟）
- 自动排名与结算
- ELO 匹配（继承现有 matchmaking 服务）

**差异化功能（上线后迭代）**
- Hyper-Turbo（3分钟盲注）— GGPoker 标配
- 前3名奖励结构（替代全额奖励）
- 离座惩罚机制（防止中途退出）

### 2. 移动端优先级建议

当前移动端为 Expo React Native（SDK 55），但：
- Web 端已有 `/room-mobile/[id]` 路由
- 需确认移动端与 Web 游戏桌功能 parity
- 移动端市场份额 65%，是核心增长渠道

**建议下轮行动**：对 `/room-mobile` 与 `/room` 进行功能对比审计

### 3. Rakeback 体验观察

刚上线的 Rakeback MVP：
- ✅ 三档等级（BRONZE/SILVER/GOLD）
- ✅ API 完整（GET/POST）
- ❌ 玩家感知价值不明确 — 无进度条/预估收益展示
- ❌ 无推送通知 — 玩家不知何时达到下一等级

**改进建议**：在 `/rakeback` 页面增加"本周预计收益"预估和下一等级进度条
