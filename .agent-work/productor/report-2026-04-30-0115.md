### 立即可改进：首充奖励 Race Bug（P2-DEPOSIT-FIRST-BONUS-RACE）

**问题：** `hasReceivedFirstDepositBonus` 检查在事务外，并发充值可能重复触发首次 bonus。

**修复方案（Test Agent 建议）：**
```ts
// deposit.service.ts — 将检查移入 atomic transaction
await prisma.$transaction(async (tx) => {
  const user = await tx.user.findUnique({ where: { id: userId } });
  if (user.hasReceivedFirstDepositBonus) throw new BadRequestException('Already claimed');
  // ... bonus logic
  await tx.user.update({ data: { hasReceivedFirstDepositBonus: true }, where: { id: userId } });
});
```

**实施优先级：P2 → P1（涉及资金安全）**
