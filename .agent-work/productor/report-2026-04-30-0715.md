### GGPoker 房间 UX 最佳实践

**房间状态 Badge:**
- 绿色圆点 + "Waiting" — 等待玩家
- 蓝色脉冲 + "Playing" — 游戏中
- 灰色 + "Empty" — 无玩家
- 悬停显示：当前人数 / 最大人数 / 等待时长

**盲注结构可视化:**
- 每级盲注标注：Small Blind / Big Blind / Ante
- 倒计时时钟显示距离下次盲注升级
- 鼠标悬停展开完整盲注结构表
- 当前级别高亮

**空状态引导:**
- Ring Game 空状态：显示 "Be the first to start a game" + 创建房间 CTA
- Tournament 空状态：显示赛程表预览 + 注册按钮
- 最近房间区：一键快速加入

### 盲注结构可视化 — 详细规格

竞品普遍采用的结构（以 GGPoker $0.05/$0.10 Ring Game 为例）：

```
Level 1:  SB $0.02 / BB $0.05 / Ante $0.00 — 00:10:00
Level 2:  SB $0.03 / BB $0.05 / Ante $0.00 — 00:08:00
Level 3:  SB $0.05 / BB $0.10 / Ante $0.01 — 00:06:00
...
```

**建议实现方式：**
1. 房间卡片 hover 显示盲注结构弹窗
2. 游戏中顶部显示当前盲注倒计时
3. 管理后台可配置盲注结构表（Room.blindSchedule JSON）
