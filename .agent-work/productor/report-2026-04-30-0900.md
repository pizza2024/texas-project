## 本轮产品调研：房间列表 UX 竞品分析

### 竞品最佳实践（WSOP / 888poker / GGPoker / CoinPoker）

#### 1. 房间筛选与排序
- **GGPoker:** 左侧边栏多维筛选（ stakes / game type / limit / speed），顶部快捷 tab（All / Texas / Omaha / Short Deck）
- **888poker:** 滑动 Filter Chip（micro/midi/macro）+ Sort by（player count / stakes / waiting time）
- **收藏夹固定：** ⭐ 按钮常驻，点击即收藏并置顶常玩房间

#### 2. 空状态引导
- **WSOP:** "No games found — Create your own table" CTA 按钮，底部圆角卡片引导
- **GGPoker:** 倒计时自动匹配（"Starting in 2:30 — 4 more players needed"）
- **CoinPoker:** 幽灵桌（Ghost Table）预展示，刺激用户创建

#### 3. 房间卡片信息密度
- **WSOP:** 房间名 | 盲注/底注 | 当前人数/max | 等待时间（"2min"）| 🔒密码锁图标 | ELO 标签
- **888poker:** 图标区分 Ring/Tournament/Sit-and-Go；玩家头像小圆点叠加
- **GGPoker:** 动态筹码量彩色条（绿黄红表示满人程度）

#### 4. 私人房间邀请
- **WSOP:** 分享按钮 → 邀请码 + 复制链接 + 二维码
- **GGPoker:** Room Code 大字展示 + 一键复制
- **Telegram Bot:** 部分平台支持直接通过 TG 机器人邀请好友入座

#### 5. 实时状态刷新
- **无刷新更新:** 房间人数变动 WebSocket 推送，数字动态增减（+1 / -1 动画）
- **玩家头像队列:** 真人头像叠加显示，不只显示数字
