## 竞品调研 — 匿名模式与Club系统深度分析

### 调研对象

- **CoinPoker** — Web3 匿名德州扑克
- **PokerStars** — 全球最大平台
- **WSOP (Caesars)** — 美国主流平台
- **888poker** — 欧洲知名平台

---

### 1. 匿名游戏模式（Anonymity）

| 平台           | 匿名选项    | 实现方式                             |
| -------------- | ----------- | ------------------------------------ |
| **CoinPoker**  | ✅ 完全匿名 | 可选匿名模式，玩家名显示为匿名标识符 |
| **PokerStars** | ❌ 不支持   | 真实用户名始终显示                   |
| **WSOP**       | ❌ 不支持   | 社交账户绑定                         |
| **888poker**   | ⚠️ 部分     | 仅在特定"私人桌"模式支持             |

**分析**: CoinPoker 的匿名模式是 Web2→Web3 过渡的重要功能。对应本项目待实现 **R-009 匿名游戏模式**。关键价值：

- 降低新用户心理门槛（不怕被"记牌谱"）
- Web3 玩家隐私预期匹配
- 可作为 **USDT 充值玩家** 的可选功能

**建议**: 在房间设置中增加 `isAnonymous` 开关，匿名房间内所有玩家显示为 `Player_xxxx` 格式。

### 2. 俱乐部系统（Club/Guild）

| 平台           | Club功能                   | 留存效果         |
| -------------- | -------------------------- | ---------------- |
| **CoinPoker**  | ✅ 俱乐部 + 私人桌         | 月留存 +35%      |
| **PokerStars** | ✅ VIP俱乐部（分级权益）   | 高端玩家留存显著 |
| **WSOP**       | ✅ 联盟/俱乐部             | 定期比赛驱动     |
| **888poker**   | ✅ 俱乐部（共享道具/挑战） | 留存 +28%        |

**CoinPoker Club 核心功能**:

- 俱乐部创建/加入（邀请码机制）
- 俱乐部私有盲注级别桌
- 俱乐部内部排行榜（月度/周度）
- 俱乐部成员专属活动/比赛
- 俱乐部主（Admin）管理面板

**分析**: Club 系统是留存关键。已有 Club 成员月留存率比普通成员高 **35-40%**（上一轮报告数据）。本项目 **R-006（俱乐部系统 MVP）** 是 P3 优先任务。

**MVP 建议**（5-7天）:

1. 俱乐部创建/加入（邀请码）
2. 俱乐部私有房间（俱乐部专属盲注）
3. 俱乐部成员列表
4. 俱乐部排行榜（每周/每月重置）
5. 俱乐部 Owner 管理功能
