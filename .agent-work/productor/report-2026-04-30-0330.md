### GGPoker / WSOP 赛事体系实地调研

#### 赛事日历结构（GGPoker WSOP Online 2025）

| 维度 | GGPoker 现状 | 对本项目启示 |
| ---- | ------------ | ------------ |
| **赛程表格式** | 表格布局：Date / Day / UTC / Event / Buy-In / GTD | P1-SCHEDULE-001 应使用类似表格或卡片网格 |
| **赛事类型** | Bounty, NLH, PLO, 6-Handed, Turbo, Deep Stack, Ladies, Monster Stack, Mystery | 赛事类型多样化是核心竞争力 |
| **买入区间** | $100~$10,300（范围极广） | 本项目 chips 区格需提前规划（500~10000） |
| **GTD 机制** | $1M~$10M 保证奖池，是营销核心 | Phase 1 先做 SNG，GTD 作为差异化后续 |
| **赛事系列** | WSOP / GGMillion$ / Bounty Hunters / Flip & Go 等多个并行系列 | 多系列 = 运营活动空间，需要 schedule 多系列标签 |
| **倒计时型** | Flip & Go（100刀，$1M GTD）先 Flip 再 Go | GGPoker 没有严格"时间到即开"的 BTC，但有快速系列 |
| **Satellite 体系** | 晋级赛赢取大比赛门票（$30K Super Pass） | 本项目 Phase 2 可扩展 Satellite 类型 |
| **报名截止** | 赛事开始前均可报名（部分赛事有延迟注册） | P1-SCHEDULE-001 的 `registrationDeadline` 设计合理 |

#### WSOP 赛事页面 UX 亮点

1. **赛程表格**：清晰列出日期、时间（UTC）、赛事名称、买入、GTD
2. **Highlighted Events**：Banner 重点推广 main event 和明星赛事
3. **多时区支持**：UTC 时间，本地时间前端转换
4. **GTD 标注**：保证奖池数字突出显示
5. **赛事分组**：按赛程系列（WSOP / Circuit / Festival）分组

#### GGPoker 赛事系统缺失（对我们是机会）

- ❌ 无"Beat the Clock"严格倒计时开赛（时间到即开，无须等满员）
- ❌ 无自动赛事提醒的客户端内 toast/popup
- ❌ 赛事系列之间无统一倒计时视图
- ❌ 无扑克桌内直接观看赛事直播

model TournamentSchedule {
