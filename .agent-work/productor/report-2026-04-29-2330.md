## 竞品调研 — 本轮专题

### 专题：GGPoker / PokerStars / 888poker / ACR Poker 功能对比总览

**背景:** 本项目核心功能已接近主流平台水平，但留存机制（通知、忠诚计划、社区）仍有差距。本专题做一次全面横向对比。

#### 核心游戏功能

| 功能        | GGPoker | PokerStars | 888poker | ACR Poker | 本项目               |
| ----------- | ------- | ---------- | -------- | --------- | -------------------- |
| 德州扑克    | ✅      | ✅         | ✅       | ✅        | ✅                   |
| 奥马哈      | ✅      | ✅         | ✅       | ✅        | ❌ (规划中)          |
| Spin & Go   | ✅      | ❌         | ❌       | ❌        | ✅ BLAST             |
| MTT 锦标赛  | ✅      | ✅         | ✅       | ✅        | ✅ SNG/MTT Phase 1-2 |
| 现金桌      | ✅      | ✅         | ✅       | ✅        | ✅                   |
| 私人/密码桌 | ✅      | ✅         | ✅       | ✅        | ✅                   |
| ELO 匹配    | ❌      | ❌         | ❌       | ❌        | ✅                   |

**观察:** ELO 匹配是本项目差异化的技术特性，竞品普遍使用房间类型/盲注级别匹配，ELO 精准匹配是提升新手体验的亮点。

| ------------ | ------- | ---------- | -------- | --------- | ------------- |
| 暗色主题     | ✅      | ✅         | ✅       | ✅        | ✅            |
| 多语言       | ✅ 15+  | ✅ 20+     | ✅ 25+   | ✅        | ❌ 仅英语     |
| 手牌历史回顾 | ✅      | ✅         | ✅       | ✅        | ✅            |
| 牌局回放     | ✅      | ✅         | ❌       | ✅        | ❌            |

| 移动端 App   | ✅      | ✅         | ✅       | ✅        | Expo (开发中) |

**观察:** 多语言支持是国际化的门槛，888poker 支持 25+ 语言是行业最高水平。牌局回放（hand replay）是高端玩家的核心需求，建议作为 P3 功能规划。
