### Hand Replay UI 规格文档

#### 1. 功能概述

战后手牌复盘（Hand Replay）允许玩家在牌局结束后，回放整局的完整过程，包括每轮的发牌、玩家操作、底池变化和最终结果。

#### 2. 用户体验设计（GGPoker Squash 风格）

**核心交互元素**:

| 元素     | 描述                                                          |
| -------- | ------------------------------------------------------------- |
| 进度条   | 底部时间轴，显示每轮节点（Pre-flop/Flop/Turn/River/Showdown） |
| 决策节点 | 进度条上的圆点，标记玩家操作（FOLD/CHECK/CALL/RAISE/ALLIN）   |
| 回放控制 | 播放/暂停、加速（0.5x/1x/2x/4x）、逐轮跳转                    |
| 手牌展示 | 玩家底牌（仅自己可见）+ 公共牌                                |
| 胜率线   | 显示关键时刻的胜率变化（接池/加注时）                         |

**UI 布局**:

```
┌─────────────────────────────────────────────┐
│  [返回]   Hand #12345   [分享] [关闭]        │
├─────────────────────────────────────────────┤
│                                             │
```

| 接口                    | 方法 | 说明                                       |
| ----------------------- | ---- | ------------------------------------------ |
| `GET /hands/:id/replay` | GET  | 获取完整牌局数据（Hand + HandAction 列表） |

**响应数据结构**:

```typescript
interface HandReplayData {
  handId: string;
  tableId: string;
  timestamp: Date;
  smallBlind: number;
  bigBlind: number;
  communityCards: string[]; // 5张公共牌
  potSize: number;
  players: {
    id: string;
    nickname: string;
    avatar: string;
    finalHand: string; // 最终手牌（5张最优）
    finalHandRank: string; // 同花顺/四条等
    netChange: number; // 胜负金额
    isWinner: boolean;
    holeCards: string[]; // 底牌（仅自己可见）
  }[];
  actions: {
    round: "PREFLOP" | "FLOP" | "TURN" | "RIVER" | "SHOWDOWN";
    playerId: string;
    action: "FOLD" | "CHECK" | "CALL" | "RAISE" | "ALLIN" | "STRADDLE";
    amount?: number;
    potOdds?: number; // 加注时的赔率
    winProbability?: number; // 关键时刻胜率
    timestamp: Date;
  }[];
}
```
