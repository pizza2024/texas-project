### 关键功能详细分析

#### 1. 成就徽章系统（Achievement/Badge）

**行业最佳实践：**
- **解锁式 vs 展示式**：主流平台采用解锁式（完成条件后永久显示），部分平台允许选择展示哪些徽章
- **分类维度**：游戏成就（盈利/手数/BB数）、社交成就（添加好友/加入俱乐部）、特殊成就（连胜/爆冷）
- **视觉设计**：稀有度分层（普通/稀有/传说），传说徽章带特效动画
- **存档展示**：个人主页或专属页面，支持按类别筛选

**对本项目的启示（P2-PROFILE-001）：**
- 建议复用现有 `Mission` 系统扩展成就徽章模块
- 徽章数据模型：`Badge { id, name, iconUrl, rarity, description, unlockedAt }`
- 用户关联：`UserBadge { userId, badgeId, unlockedAt }`
- 稀有度分层：`COMMON / RARE / EPIC / LEGENDARY`

#### 2. 头像与自定义（Avatar/Frames）

**GGPoker 的实现：**
- 基础头像（平台提供或上传）
- **头像框（Avatar Frame）**：可解锁/购买，用于身份标识
- 表情包/小部件装饰

**WSOP 的实现：**
- 虚拟形象（Virtual Avatar）系统
- 可自定义发型/服装/配饰

**对本项目的建议：**
- 头像框可关联赛季/段位/Rakeback等级
- 复用现有 `User.avatar` 字段，新增 `avatarFrame` 枚举
- 低成本快速实现：静态图片帧（对比 3D 虚拟形象）
