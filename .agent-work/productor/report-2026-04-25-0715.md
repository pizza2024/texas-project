### 2.2 GGPoker Rakeback 体系（行业标杆）

GGPoker 的 rakeback 体系是行业最复杂但也最有效的：

- **GGClub 系统**：玩家可创建俱乐部，私人 ring game 桌，rake 分配给俱乐部成员
- **PartyShares**：集体奖励机制，俱乐部成员共享 rake
- **6级 VIP**：STAR → DIAMOND → DIAMOND+ → CROWN → COBALT → BLACK
- **顶级玩家实际可达 60%+**：普通玩家 20-30%，高量玩家 50%+
- **本项目差距**：缺少 PLATINUM/DIAMOND 等级，缺少 Club rake 分配机制

### 2.3 CoinPoker Token 模型（本项目可借鉴）

CoinPoker 的 CHP Token 模型对本项目有参考价值：

- **CHP 质押**：持有 CHP 可获得额外 rakeback（最高 +10%）
- **USDT 存取**：已实现，是本项目差异化优势
- **链上可验证**：庄家随机数公开可验证
- **本项目启示**：可考虑引入平台 Token（如 TCP Token）增加粘性

### 2.4 PokerStars 5星 VIP（留存机制参考）

- **按月重置**：高频激励，防止躺平
- **Star Reward 系统**：每一手牌计算星级分数
- **本项目启示**：GOLD 30% 可考虑加入月重置机制提升活跃度
