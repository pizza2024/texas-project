## 竞品调研：BetMGM + Partypoker UX 分析

### BetMGM Poker (2026)

**核心 UX 特色：**
- **手势操作**：支持滑动 fold/check/raise，减少点击
- **私密桌面**：可自定义桌面布局，玩家自行拖拽位置
- **记牌器 (Hand History)**：内置历史牌局回顾，支持复盘
- **多桌切换**：快速切换键（Ctrl+数字键）

**对我的启示**：
- 手势操作对移动端体验提升显著（建议 P3-FastFold 考虑）
- 记牌器是高级玩家刚需，建议 P3 加入

### Partypoker

**核心 UX 特色**：
- **Snap Mode**：一键fold所有牌，快速进入下一局（已在我方 P3-FastFold 参考列表）
- **Blitz Mode**：4人快桌，盲注快速上涨
- **Power Play**：翻牌后发牌加速娱乐性

**对我的启示**：
- Snap Mode 技术成本低，DAU 提升效果显著，建议优先实现
- Blitz Mode 可作为 Tournament 赛制的快棋变体
