### 1. Ignition Casino Poker（2026 最新）

**核心卖点**：
- **Zone Poker** — 快速换桌（手间隔8秒），无需等待
- **Anonymous Tables** — 无 HUD，对新手友好，生态公平
- **Quick Seat** — 按游戏类型+盲注快速入座
- **Mobile App** — 跨平台同步

**产品设计亮点**：
```
Zone Poker = 每局结束后立即移动到新牌桌
→ 解决"等待疲劳"（Playtime maximization）
→ 本项目可参考：Auto-Rebuy + Fast-Fold 组合
```

**用户体验评分**：
- 注册流程：⭐⭐⭐⭐⭐（3步完成）
- 入座体验：⭐⭐⭐⭐⭐（Quick Seat 智能分流）
- 牌桌流畅度：⭐⭐⭐⭐⭐（Zone Poker 无等待）
- 充值体验：⭐⭐⭐⭐（Crypto 支持，法币较慢）

---

### 2. CoinPoker（加密扑克室）

**核心卖点**：
- **USDT 充值** — 链上直接兑换 chips，秒到账
- **Provably Fair** — 牌局结果可链上验证（随机数公开）
- **CHP 代币** — 平台治理代币，Rakeback 激励
- **低手续费** — 5% rake vs 行业 10%

**产品设计亮点**：
```
充值流程：
1. 生成专属 ETH 地址（HD 钱包派生）
2. USDT → 合约兑换 → chips 到账
3. 链上记录，完全可查

→ 本项目参考：DepositAddress + DepositRecord 模型 ✅ 已实现
```

**Provably Fair 设计**：
```
牌局结果 = SHA256(服务器种子 + 客户端种子 + 牌局ID)
→ 玩家可在局后验证
→ 增强平台可信度
→ P3 建议：手牌评估结果可验证性
```

---

### 3. BetOnline Poker

**核心卖点**：
- **Sportbook 联动** — 扑克+体育博彩同一账户
- **Bad Beat Bonus** — 冤家牌额外奖励（AAAJK vs 2222K）
- **Reload Bonus** — 周度充值优惠
- **快速提现** — 24小时处理

**产品设计亮点**：
```
Bad Beat Jackpot：
触发条件：大盲位置冤家牌（特定牌型组合）
奖池分配：BBJ 池 + 主池 + 边池
→ 留存核心：让失败也有"收获感"
→ 本项目参考：Bad Beat 保险（P3 功能）
```
