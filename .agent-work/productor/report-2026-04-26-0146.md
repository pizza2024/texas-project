### 关键差异分析

#### 1. 虚拟币体系（Play Money / Free Chips）

所有主流竞品均提供虚拟货币游戏场景，用于：

- **降低新用户心理门槛**：无需真实资金即可体验完整游戏流程
- **游戏机制学习**：特别是 Blinds/Straddle/Raise 等概念
- **波动性隔离**：新用户不会因bad beat 流失

**本项目缺失的影响**：

- 新用户直接面对 USDT 真实资金，风险厌恶型用户直接流失
- 无法在不充值情况下验证游戏体验
- 与 CoinPoker（匿名+Bot对战）相比，差异化不足

#### 2. 首充奖励（First Deposit Bonus）

| 平台       | 奖励结构                   | 释放条件      |
| ---------- | -------------------------- | ------------- |
| GGPoker    | 200% up to $600            | 25x rake 解锁 |
| PokerStars | $20 Free immediately       | 无            |
| WSOP       | $50 Free + 100% up to $200 | 2x rake       |
| 888poker   | 200% up to $888            | 30x rake      |
| CoinPoker  | 150% CHP staking bonus     | 锁仓          |

**本项目现状**：仅有 5-tier Rakeback（最高 50% DIAMOND），无首充奖金引导页。

**建议**：评估首充奖金机制（参考 GGPoker 200% up to $600），可显著提升首次转化率。

#### 3. Daily Missions / 每日任务

竞品标准留存机制，通过每日任务引导玩家返回：

- **任务类型**：完成 X 局 / 赢得 Y 底池 / 参与特定赛事
- **奖励**：免费比赛资格 / 虚拟币 / rakeback 额外百分比
- **设计目的**：将"无聊"转化为"每日目标"

**本项目缺失**：无任务系统，P2-NOTIFY-001（通知中心）已就绪但未实施，无法形成召回闭环。

#### 4. Freeroll 免费赛

所有竞品均提供免费参加的锦标赛（Freeroll），是：

- 新用户第一个"真实奖励"体验
- 社交/竞争氛围初次接触
- 低成本验证平台稳定性

**本项目**：无 Freeroll 机制。
