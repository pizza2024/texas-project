## 本轮调研 — VIP Rakeback 分段设计分析

### 当前系统

本项目使用 **总抽水累积金额（totalRake）** 决定层级：

| Tier     | 最低累计抽水 | Rakeback 比例 |
| -------- | ------------ | ------------- |
| BRONZE   | $0           | 15%           |
| SILVER   | $500         | 22%           |
| GOLD     | $2,000       | 30%           |
| PLATINUM | $10,000      | 40%           |
| DIAMOND  | $50,000      | 50%           |

**计算方式:** `rakebackAmount = floor(rakeAmount × rate)`

---

### 竞品 Rakeback 系统对比

#### GGPoker — Fish Points（7级）

| VIP 等级  | Rakeback 范围 | 说明            |
| --------- | ------------- | --------------- |
| Level 1   | ~15%          | 新玩家入门      |
| Level 2-3 | 20-30%        | 休闲玩家        |
| Level 4-5 | 35-50%        | 常规玩家        |
| Level 6-7 | 55-65%+       | 高额桌/职业玩家 |

- **Fish Points** 1:1 兑换 rake，跨游戏通用
- **EGC（Events）** 额外奖励，积分可兑换赛事门票
- **赛季重置:** 无年度重置（长期累积）

#### CoinPoker — CHP 质押模型

| 方案     | 年化 Rakeback | 说明                |
| -------- | ------------- | ------------------- |
| 不质押   | 0%            | 仅使用平台          |
| 基础质押 | ~15%          | 持有 CHP            |
| 积极质押 | 30-45%        | 长期锁仓 + 高持有量 |
| 超额质押 | 50%+          | 平台利润回购分发    |

- CHP 质押解锁rakeback，通过代币升值叠加收益
- **隐私:** 无 KYC（部分管辖区）

#### ACR (Americas Cardroom) — 27-65%+

| VIP 等级   | Rakeback | 特殊奖励         |
| ---------- | -------- | ---------------- |
| Silver     | 27%      | -                |
| Gold       | 35%      | BCC 存款免手续费 |
| Platinum   | 42%      | 专属赛事资格     |
| Black      | 55%+     | 1对1支持         |
| Super Nova | 65%+     | 定制奖励计划     |

### ELO 映射分析

| 平台          | ELO 用途        | Rakeback 关联                |
| ------------- | --------------- | ---------------------------- |
| **本项目**    | 匹配对手（mmr） | ❌ 无关，仅按 totalRake 分层 |
| **GGPoker**   | 头像框/成就显示 | 部分赛事资格相关             |
| **CoinPoker** | 排行榜          | 质押量决定，与 ELO 解耦      |

**发现:** 本项目 ELO 仅用于匹配（matchmaking），与 rakeback tier 完全独立。这是合理设计——防止玩家刷 ELO 跳 tier。

### 改进建议

#### 1. 当前系统的问题

| 问题            | 说明                                         |
| --------------- | -------------------------------------------- |
| **无 ELO 关联** | 优点：防止刷分；缺点：ELO 无长期价值感       |
| **无赛季重置**  | DIAMOND 玩家一旦达成永久享受 50%，无维护成本 |
| **升级提示弱**  | 用户不知道距下一级还需多少 rake              |

#### 2. 建议功能：VIP 赛季制

参考 GGPoker 年度赛季，维护成本：

```
赛季长度: 12个月
保级要求: 赛季内累计 rake 达到当前 tier 最低门槛的 50%
降级: 赛季结束时未达保级线 → 降一级
```

**示例:**

- 当前 GOLD（$2,000），赛季结束前需累计 $1,000 续命
- 否则降至 SILVER（$250 续命）
- 彻底脱离 → 降至 BRONZE

#### 3. 建议功能：ELO 联动奖励（Cosmetic Only）

| ELO 范围  | 称号    | 头像框样式 |
| --------- | ------- | ---------- |
| < 1000    | Fish    | 铜边框     |
| 1000-1500 | Regular | 银边框     |
| 1500-2000 | Shark   | 金边框     |
| 2000+     | Whale   | 钻石边框   |

**注意:** 仅 Cosmetic，不影响 rakeback，保证公平性。
