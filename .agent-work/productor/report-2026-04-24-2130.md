## 竞品调研：加密扑克平台深度分析

### CoinPoker — 加密原生扑克平台

**平台概况**
| 维度 | 数据 |
|------|------|
| 成立时间 | 2017年 |
| 区块链 | ETH、MATIC、TRON |
| 牌照 | Curacao eGaming |
| 玩家规模 | 中型（Crypto Poker Staff 估算 5-10万 MAU） |

**核心产品特性**
| 功能 | 实现方式 | 对本项目启示 |
|------|----------|--------------|
| USDT 充值 | 链上转账，自动确认 | ✅ 本项目已有类似架构 |
| 匿名游戏 | 无需 KYC（小额） | P3 — 考虑隐私模式 |
| 加密钱包集成 | 内置 Web3 钱包 | P3 — 钱包抽象层 |
| Rakeback | 5-55% 阶梯 rakeback | P3 — Rakeback 体系参考 |
| 天才国际象棋式匹配 | ELO + 段位系统 | P2 — Sit-Out 重构可参考 |

**商业模式**
- Rake 抽成（5-10% pot）
- Rakeback 消耗（玩家返利）
- 充值/提现手续费（1-2%）
- NFT 会员卡（高级功能解锁）

**用户体验亮点**
1. **即时充值** — 链上确认后 5 分钟内到账
2. **全球无障碍** — 支持 100+ 国家
3. **低佣金比赛** — 相比 WSOP 佣金低 50%
4. **移动端适配** — 滑动弃牌、紧凑布局
