### GGPoker Time Bank Card — 付费时间银行深度分析

**背景：** 2025年3月 GGPoker 重新设计了 Time Bank 功能，推出付费 "Time Bank Card"

**机制：**
- 玩家可购买额外时间（卡片形式），在需要时使用
- 购买方式：真实货币 / 平台积分购买
- 目的：减少强制站起的频率，提升用户体验

**与本项目的关联：**

| 维度 | GGPoker | 本项目 |
|------|---------|--------|
| 时间银行 | 付费扩展（货币化） | 免费基础时间银行 ✅ 已实现 |
| 购买入口 | 游戏中内购 | 暂无 |
| UI 呈现 | 时间条延长 | 需确认是否有视觉反馈 |

**实施建议：**
- P2 功能：Time Bank Card 购买 UI（先做免费版）
- 用户摩擦点：玩家在思考时不愿被站起，付费买时间比重新入桌更划算
- 商业模式参考：可作为 Rakeback 之外的辅助变现手段

### GGPoker Streamer Mode — 直播模式

**机制：**
- 主播启用后，牌桌数据（其他玩家手牌、历史）不显示给主播
- 防止主播泄露手牌给观众（Ghost play）
- 100% Rakeback 激励直播

**与本项目的关联：**
- 本项目已有匿名桌（Anonymous Mode）调研记录
- **Streamer Mode = Anonymous Mode + 附加限制**：主播账号特殊标记，隐藏所有对手数据
- 实现路径：
  1. ✅ Anonymous Mode（Room.isAnonymous）
  2. P2：User.isStreamer + Streamer Mode 权限过滤
