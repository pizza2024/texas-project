## 竞品调研：GGPoker / WSOP / 888poker / CoinPoker 横向对比

### GGPoker — 行业标杆

| 维度 | 详情 |
|------|------|
| 即时赛事 | **SPINS** — 3人极速赛，随机奖池最高 50,000X |
| UI/特效 | 3D 轮盘旋转动画，倍数从高到低递减免停 |
| 时长 | 约 3 分钟，碎片化体验 |
| 报名费 | $0.25 ~ $100，8档 |
| 商业模式 | 高 rake + 门票差价 + 抽奖池 |
| 特色功能 | SNAP Mode（快速fold）、AIO Shortcuts、Card Runners |

### WSOP — 品牌驱动

| 维度 | 详情 |
|------|------|
| 核心吸引力 | WSOP bracelet 文化联动，大奖池造势 |
| Blast 赛事 | 好莱坞风格特效，3-5人场次 |
| 跨平台 | PS5/PC/Mobile 多端同步 |
| 社交功能 | 玩家牌桌互动、表情反应 |

### 888poker — 差异化竞争

| 维度 | 详情 |
|------|------|
| FLIP & GO | 前3张牌强制 all-in，1-2分钟极短赛制 |
| SNAP Mode | 快速 fold 加速传统 cash game |
| 特色UI | 暗色主题，3D 赛事大厅 |
| 免费赛 | 大量 freeroll 吸引入门玩家 |

### CoinPoker — 加密原生

| 维度 | 详情 |
|------|------|
| 入金方式 | USDT/ETH/CHP (自有代币) 直接充值 |
| 隐私保护 | 无 KYC 或轻度 KYC |
| 游戏特色 | 加密 shuffle 算法（ verifiable fair） |
| Rakeback | CHP 代币持有者享有更高 rakeback |

1. **Blast 即时赛事是用户增长核心** — GGPoker SPINS 已成为行业标配，3分钟碎片化体验填补传统 MTT/SNG 空白
2. **UI 差异化靠特效和动画** — 888poker 暗色主题 + 3D 赛事大厅，WSOP 好莱坞级特效
3. **加密入金是竞争力** — CoinPoker 的 USDT 直充 + 无 KYC 对特定用户群有强吸引力
