## 本轮竞品调研：VIP 忠诚度体系 & Rakeback

### 为什么这个课题重要
Rakeback 是德州扑克平台最重要的收入留存机制。平台从每局牌中抽取 rake（服务费），部分返还给玩家形成 Rakeback。高效的 Rakeback 体系能：
- 提升玩家留存率（核心玩家依赖 Rakeback 维持游戏时间）
- 降低获客成本（现有玩家带新玩家）
- 形成竞争护城河（玩家在平台间迁移成本）

### 推荐 Rakeback MVP 方案（3 级 VIP）

```
┌─────────────────────────────────────────────────────┐
│  铜牌（Bronze）    银牌（Silver）    金牌（Gold）    │
│  月度 Rake ≥ 100   月度 Rake ≥ 500   月度 Rake ≥ 2000  │
│  Rakeback 10%      Rakeback 20%      Rakeback 35%     │
│  专属客服 ✗        专属客服 ✗        专属客服 ✅       │
│  专属房间 ✗        专属房间 ✗        专属房间 ✅       │
└─────────────────────────────────────────────────────┘
```

**月度 Rakeback 计算**：
```
每月 1 日结算：
  可领取 Rakeback = 月度 totalRake × VIP等级对应比例
  筹码自动发放至钱包
```
