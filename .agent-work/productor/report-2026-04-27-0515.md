### 🔴 P0-E2E-001 — 必须立即解决

**问题:** `Cannot find module '@texas/shared/types/tournament'` — E2E 测试无法启动
**影响:** 完整的 Playwright E2E 测试套件（app.e2e + rakeback.e2e）全部失败，CI/CD 发布前验收阻断
**根因:** `packages/shared/package.json` 的 `exports` 字段缺少 `./types/tournament` 映射，jest-e2e 的 module resolver 无法解析

**产品风险:**
- 每次功能变更无法通过 E2E 验证，导致潜在 Bug 进入生产
- 财务相关测试（rakeback、deposit、withdraw）无法端到端验证
- Tournament 功能（已实现的 Phase 1+2）完全没有 E2E 覆盖

**建议行动（立即）:**
```json

"./types/tournament": "./src/types/tournament.ts"
```
或在 `jest-e2e.json` 添加 `moduleNameMapper`。
