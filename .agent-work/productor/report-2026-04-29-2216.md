## 竞品动态（深度调研）

### 888poker — 核心差异化

| 功能                         | 详情                                    | 对本项目参考价值          |
| ---------------------------- | --------------------------------------- | ------------------------- |
| **Blast Jackpot Sit-and-Go** | 3人超快 SNG，随机 2x~10000x 彩池        | 🔴 高 — P2-NEW-024 已关注 |
| **PokerSchool**              | 免费教学平台（视频+文章+测验）          | 🟡 中 — 可作新手引导参考  |
| **888 Club 忠诚计划**        | 多级积分（Gold→Platinum→Elite→Diamond） | 🔴 高 — 本项目缺失        |
| **PokerCam**                 | 牌桌视频社交                            | 🟡 中 — 社交差异化        |
| **Snap Poker**               | 快速弃牌换桌（类似 Zoom/Rush）          | 🟡 中 — 已有匹配系统      |

### WSOP — 生态打法

- **Caesars Rewards 跨品牌积分**：与酒店/赌场业务联动
- **Clubs 社交系统**：玩家自建俱乐部
- **赛事日历**：锦标赛预告 + 倒计时报名
- **金手链文化**：成就体系标杆

### CoinPoker — Crypto 原生

- **USDT 即时充值**：链上确认秒到账
- **CHP 平台币 rakeback**：比 USDT 充值额外优惠
- **匿名注册**：无 KYC，本项目已有 HD 钱包充值

### 关键竞品洞察

1. **忠诚积分是标配**：所有主流平台都有多级 VIP/积分体系
2. **Jackpot SNG 是引流利器**：低成本高回报的彩票感
3. **教学内容是留存抓手**：PokerSchool 类产品有效降低新手流失
4. **俱乐部系统是社交核心**：WSOP Clubs 证明玩家社区价值
