## 竞品调研：ClubGG 与 BetMGM Poker

### ClubGG 概览

ClubGG 是全球最大的社交扑克俱乐部平台之一，采用"俱乐部+订阅"模式，玩家可创建私人俱乐部并邀请好友。

#### 核心功能
| 功能 | ClubGG | 本项目参考 |
|------|--------|-----------|
| **Club 系统** | ✅ 完整俱乐部体系 | P3-Club |
| **Gold 会员订阅** | ✅ $14.99/月 独家内容 | P3-Rakeback |
| **私人邀请桌** | ✅ | P3 |
| **免费游戏** | ✅ | ✅ 已实现 |
| **Texas Hold'em** | ✅ 核心游戏 | ✅ 已实现 |
| **Omaha** | ✅ | ❌ |
| **Tournament** | ✅ | P3-Tournament |
| **好友系统** | ✅ | P3 |
| **聊天功能** | ✅ 俱乐部/牌桌内聊天 | P3 |

#### Club 系统深度解析
- **创建俱乐部**: 任何玩家可创建，设置俱乐部名称、图标、描述
- **邀请机制**: 生成邀请码或直接邀请好友
- **俱乐部管理员**: 管理成员、设置准入门槛、踢出玩家
- **俱乐部排行榜**: 每周/每月盈利排行
- **俱乐部活动**: 定期举办的俱乐部内部锦标赛

#### 商业模式
- **Gold 会员**: $14.99/月，享受：
  - 独家 ClubGG 锦标赛参赛资格
  - 更高的 Rakeback 比例
  - 专属客服支持
  - 独家服装/头像道具
- **非会员**: 可免费创建俱乐部、使用基础功能

#### 关键洞察
1. **社交隔离设计**: ClubGG 核心是"私人空间"感，俱乐部是私密社交圈
2. **UGC 驱动**: 俱乐部管理员自发组织活动，降低平台运营成本
3. **订阅留存**: Gold 会员形成稳定订阅收入流

### BetMGM Poker 概览

BetMGM Poker 是 MGM Resorts 旗下的真钱扑克平台，在美国多个州合法运营。

#### 核心功能
| 功能 | BetMGM | 本项目参考 |
|------|--------|-----------|
| **真实货币** | ✅ | Chips ✅ |
| **多州运营** | ✅ NJ/PA/MI 等 | Web3 全球 |
| **Mlife Rewards** | ✅ 忠诚积分体系 | P3-Rakeback |
| **锦标赛** | ✅ 大型系列赛 | P3-Tournament |
| **Cash Games** | ✅ | ✅ 已实现 |
| **SNG (Sit & Go)** | ✅ | P3-JackpotSNG |
| **Casino 整合** | ✅ 老虎机/体育博彩 | P3 |
| **Pot Odds HUD** | ✅ | P-UX-1 |
| **FastFold (Blaze)** | ✅ | P3-FastFold |
| **Jackpot SNG** | ✅ | P3-JackpotSNG |

#### Mlife Rewards 忠诚体系
- **积分获取**: 每一手牌根据 rake 生成积分
- **等级**: Sapphire → Pearl → Gold → Platinum → Platinum Plus → Diamond → Black
- **兑换**: 免费客房、餐饮、Spa、赛事门票
- **跨平台**: 与 MGM 赌场、体育博彩共用积分体系

#### 锦标赛生态
- **World Series of Poker (WSOP)**: BetMGM 举办 WSOP 线上赛事
- **MGM Grand 线下赛**: 线上资格赛 → 线下决赛
- **日常锦标赛**: 每日多场不同买入赛事

#### 关键洞察
1. **品牌信任**: MGM Resorts 品牌背书降低新用户注册门槛
2. **跨产品协同**: 扑克/赌场/体育博彩用户互通，LTV 最大化
3. **线下联动**: 线上资格赛 → 线下实体赛事，增强仪式感

## 竞品调研小结

**ClubGG 核心洞察**:
- "私人俱乐部"模式创造社交隔离感，好友圈私密游戏是核心吸引力
- Gold 订阅 ($14.99/月) 提供稳定订阅收入
- 俱乐部联赛激活UGC内容，平台运营成本低

**BetMGM 核心洞察**:
- Mlife Rewards 积分体系打通线下 MGM 生态，跨产品协同
- 线下赛事联动（线上资格赛 → 线下决赛）增强仪式感
- Pot Odds HUD 是唯一具备 HUD 功能的主流平台

**差异化定位**: 本项目定位"无 KYC + Web3 + 全球化"德州扑克，与 ClubGG（封闭社交）、BetMGM（品牌授权真钱）形成错位竞争。
