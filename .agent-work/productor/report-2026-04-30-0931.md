## 本轮产品观察

### 从 Test r53 发现的架构问题 — P2-NOTIFY-WS-BRIDGE

**问题描述:** `notifyFriendsOfStatusChange` 通过 `wsManager.emitToUser` 推送，多实例部署时通知只推送到当前实例连接的客户端。

**产品影响:** 多实例扩容后，在线状态变更通知、好友上线提醒等功能会失效。

**建议:** 这是 Phase 3 前必须解决的基础设施问题，建议 Productor 标记为 P2 基础设施优先级。
