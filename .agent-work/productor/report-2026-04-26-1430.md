### 核心功能分析：玩家笔记系统

**什么是玩家笔记？**
在牌桌上对特定对手的实时标注，如"松弱"、"紧凶"、"跟注站"等，供后续参考。

**CoinPoker Smart Notes 特色：**
- 内置预设标签（Loose, Tight, Aggressive, Passive, Calling Station, Shark 等）
- 每手牌结束后可添加笔记
- 笔记与手牌历史关联，可追溯查看

**GGPoker 笔记功能：**
- 颜色标签 + 文字笔记
- 按房间筛选
- 导入/导出（给高阶玩家）

### 本项目 P2-NOTESYS 规格建议

基于竞品分析，P2-NOTESYS 规格应为：

```
**Phase 1 核心规格：**

后端：
- 新增 `PlayerNote` Model：userId（笔记主人）/ targetUsername / note / color（标签颜色）/ createdAt / updatedAt
- API: POST /notes（创建）/ GET /notes?target=（查询）/ DELETE /notes/:id

前端：
- 牌桌内：点击对手用户名 → 弹出笔记面板
- 快速标签：5 个预设标签（松/紧/凶/弱/跟注站）+ 自定义文字
- 颜色标识：对手名字显示对应颜色标签
- 笔记历史：profile 页可查看所有笔记

技术难点：
- 笔记隐私：只能看到自己写的笔记（userId + targetUsername 复合唯一索引）
- 性能：大量笔记时需分页查询
```
