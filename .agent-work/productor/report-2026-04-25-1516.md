## 竞品调研补充 — CoinPoker P2P 市场

### CoinPoker P2P 筹码市场核心机制

| 要素 | 说明 |
|------|------|
| 机制 | 玩家间点对点买卖 USDT 换筹码 |
| 价格 | 市场定价，有 slight premium/discount |
| 手续费 | 平台收取 1-3% 交易费 |
| 支付 | USDT TRC20 / ETH ERC20 |
| 限额 | 每日/每周限额防止洗钱 |
| KYC | 大额交易需 KYC 验证 |

### UX 设计要点

1. **即时性**: P2P 订单匹配后快速成交
2. **信任机制**: 筹码托管（Escrow）+ 争议处理
3. **价格发现**: 玩家自主定价，浮动汇率
4. **限制**: 新账户限额低，防止机器人刷量

### 对本项目的借鉴意义

| 优先级 | 功能 | 实现复杂度 |
|--------|------|-----------|
| ⭐⭐⭐ 高 | P2P 筹码托管 Escrow 系统 | 高（需要交易引擎） |
| ⭐⭐⭐ 高 | USDT P2P 买卖 UI/UX | 中 |
| ⭐⭐ 中 | KYC 集成（高额度） | 中 |
| ⭐⭐ 中 | 交易手续费系统 | 中 |

**建议**: P2P 市场是 CoinPoker 的差异化核心，可作为 Phase 1 Tournament 后的商业化功能候选。

## Tournament SNG Phase 1 — 规格框架（续）

### SNG 核心参数

| 参数 | 建议值 |
|------|--------|
| 买入 | 固定（$10 / $25 / $50） |
| 初始筹码 | 1000 / 2500 / 5000 |
| 玩家数 | 8 或 9 人 |
| 盲注结构 | 10分钟/级，2x 增长 |
| 奖励分布 | 第1名 60% / 第2名 30% / 第3名 10% |

### Phase 1 最小功能集

1. **固定买入入口**: 选择 SNG 买入级别 → 筹码自动入桌
2. **满人开始**: 8/9 人后倒计时（60秒）开始
3. **淘汰机制**: 筹码归零 → 自动出局
4. **奖励分配**: 决赛桌（3人）后结束，前3名按比例分配
5. **倒计时器**: 每手之间 30 秒操作时间

### 与 Cash Game 的技术差异

| 维度 | Cash Game | SNG |
|------|-----------|-----|
| 游戏状态 | 常驻 Table | 临时 Hand 序列 |
| 座位管理 | 动态加入/离开 | 固定座位，淘汰减少 |
| 盲注 | 固定位置轮换 | 随玩家减少轮换 |
| 结束条件 | 手动离桌 | 3人/6人时分配奖励 |
