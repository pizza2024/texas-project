## 三、P1-PRIZE-DISPLAY 规格建议（来自 Test 第272轮）

### 问题确认

当前 `TournamentCard`（`schedule/components/TournamentCard.tsx`）缺少：

1. **总奖池金额** — 只显示 buyin，无总奖池数字
2. **GTD 标签** — 无 "GTD" 标识（对 MTT 比赛至关重要）
3. **已注册人数** — 无 "XXX/YYY registered" 实时数据

### 当前 `ScheduleEntry` DTO 字段（`schedule.dto.ts`）

```ts
export interface ScheduleEntry {
  id: string;
  name: string;
  type: TournamentType; // BTC | SNG | MTT
  buyin: number;
  maxPlayers: number;
  smallBlind: number;
  clockIntervalSeconds: number;
  scheduledStartTime: string | null;
  prizeDistribution: readonly [number, number, number]; // 仅 [1st%, 2nd%, 3rd%]
  status: TournamentScheduleStatus;
  createdAt: string;
  updatedAt: string;
}
```

### 建议扩展字段

```ts
export interface ScheduleEntry {
  // ...现有字段...

  // 新增
  prizePool?: number; // 总奖池金额（chips/USD）
  isGuarantee?: boolean; // 是否有 GTD 保障
  registeredCount?: number; // 当前报名人数
  currency?: "chips" | "usdt"; // 货币类型
}
```

### 前端 TournamentCard 变更建议

```tsx
// 显示总奖池 + GTD 标签（参考 GGPoker 风格）
{
  prizePool && (
    <div className="flex items-center gap-2">
      <span className="text-yellow-400 font-bold text-lg">
        {formatChips(prizePool)} chips
      </span>
      {isGuarantee && (
        <span className="bg-red-600 text-white text-xs px-2 py-0.5 rounded font-bold">
          GTD
        </span>
      )}
    </div>
  );
}

// 显示已注册人数
{
  registeredCount !== undefined && (
    <span className="text-gray-400 text-sm">
      {registeredCount}/{maxPlayers} registered
    </span>
  );
}
```
