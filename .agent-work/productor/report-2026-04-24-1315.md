## 竞品调研 — P3 功能可行性分析

### 每日登录奖励（Daily Bonus）

**行业标准做法**：

| 竞品 | 机制 |
|------|------|
| CoinPoker | 连续7天阶梯奖励（$0.50→$5），断签重置 |
| GGPoker/Natural8 | Ocean Points 每日任务，额外签到奖励 |
| WSOP.com | 每日免费转盘，VIP 等级加成 |

**建议实现方案**：
```
签到天数 → 奖励 chips
1天: minBuyIn * 1
3天: minBuyIn * 3
7天: minBuyIn * 7（本周大奖）

触发：用户登录时检查 lastLoginDate
存储：User.lastLoginDate + User.consecutiveLoginDays
限制：UTC 0点重置，每日一次
```

**技术评估**：
- 后端：User 模型增加 2 字段，Auth login 时更新
- 前端：登录后弹窗显示奖励，无需新页面
- 风险：低，30 分钟可完成 MVP

### 表情互动系统（Emoji Reactions）

**行业标准做法**：

| 竞品 | 实现 |
|------|------|
| GGPoker | 桌内 8 个表情（😎🤔😮😂😘🔥👋🙏），非打断式 |
| PokerStars | 手牌结束后表情气泡 |
| CoinPoker | 仅文字快捷短语 |

**建议 MVP**：
```
非打断式 emoji reaction
- 位置：玩家头像旁（不遮挡牌面）
- 触发：点击玩家头像弹出选择器
- 可见范围：同桌玩家
- 动画：0.5s fade-out 气泡
- 频率限制：每 10 秒最多 3 次
```

**表情集合**（6个，扑克主题）：
😮 — 惊讶 | 😂 — 好牌 | 🤔 — 思考 | 😎 — 自信 | 🔥 — 热手 | 👋 — 你好
