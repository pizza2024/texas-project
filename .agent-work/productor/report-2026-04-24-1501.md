## 本轮新发现：新手引导（Onboarding）严重缺失

### 问题描述

翻阅代码库（AGENTS.md + 目录结构），**完全没有新手引导/教程实现**：

- 无 onboarding/tour/guide 目录或文件
- 无 new player tutorial 逻辑
- 无 interactive tour/guided walkthrough
- 无 tooltip/hint 系统引导新用户

### 竞品对比

| 竞品               | 新手引导实现                               |
| ------------------ | ------------------------------------------ |
| WSOP (PlayStation) | 强制新手教程关卡，解锁机制，逐步开放桌类型 |
| 888poker           | 引导式 tutorial，覆盖德州基础规则+操作     |
| CoinPoker          | 新手任务引导，AI 对战教学                  |
| PlayStation Poker  | interactive tutorial mode                  |

### 影响

- **新用户留存率**：无引导的情况下，新用户很可能在第一局就流失
- **学习曲线**：德州扑克本身有门槛（HUD 信息量、行动时机等），没有引导会加剧流失
- **与竞品差距**：竞品标配功能，这是一项 P1 级别的缺失

### 建议方案

```
Phase 1 — 最简引导（建议 P1）
1. 强制第一局 tutorial 模式（Solo AI 对局）
2. 分步骤高亮 UI 元素（底池、行动按钮、弃牌/跟注/加注）
3. 每步配文字说明（1-2 句）
4. 完成后解锁正式房间

Phase 2 — 渐进解锁
1. 解锁 MICRO → LOW → MEDIUM 房间（根据新手完成情况）
2. 每日任务引导（与 P3-DailyBonus 结合）
3. 教练模式（观察 AI 对局）
```
