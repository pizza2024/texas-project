### API 数据来源

```
GET /hands/:id/replay → HandReplayData
  - handId, roomName, date, smallBlind, bigBlind
  - initialPot, finalPot
  - players[]: { id, nickname, holeCards, netProfit, handName }
  - communityCards[], winnerId
  - timeline[]: ReplayActionNode[]
```
