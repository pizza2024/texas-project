### 竞品调研补充 — PlayStation Poker 不存在

**结论：PlayStation 平台无官方德州扑克游戏。**

搜索结果确认：PlayStation Store 中不存在独立的德州扑克应用。PlayStation 玩家若想玩扑克，只能通过：
1. PC/移动端在线扑克平台
2. 主机聚会游戏（如 Jackbox Party Packs 中偶尔包含扑克类游戏）

**→ 本项目移动端（Expo React Native）是接触 PlayStation 玩家群体的最优路径**

**竞品参照：** WSOP、888poker 均支持从好友列表直接邀请入桌

**当前状态：** 好友可以看到状态（online/playing），但无法从好友页直接邀请入桌

**用户体验闭环：**
```
好友在线 → 点击邀请 → 选择房间/直接带入我所在房间 → 好友加入同桌
```

**影响：** 好友系统留存率提升，社交驱动回归

**优先级建议：** P2
