## 二、竞品调研 — Spin & Go 家族横向对比

### 规格对比

| 平台 | 名称 | 人数 | 最高倍数 | 时长 | 特点 |
|------|------|------|----------|------|------|
| GGPoker | SPINS | 3 | 10,000x | ~3min | 轮盘动画，80% rakeback |
| PokerStars | Spin & Go | 3 | 12,000x | ~3min | 无穷奖池，1v1v1 |
| 888poker | BLAST | 3 | 10,000x | ~3min | 即时匹配 |
| 本项目 | Blast | 3 | 50,000x | ~3min | **规格已定义，待实施** |

### 奖池分布算法对比

**PokerStars Spin & Go 典型分布（$1 场）：**

| 倍数 | 概率 |
|------|------|
| 2x | ~60% |
| 4x | ~25% |
| 10x | ~10% |
| 100x | ~4% |
| 1,000x | ~0.5% |
| 12,000x | ~0.05% |

**技术实现要点：**
- 服务端使用 crypto secure RNG（`crypto.randomBytes()` 或 Chainlink VRF）
- 客户端先显示加载动画，服务端随机数到达后再展示结果
- 防止作弊：客户端无法预测或操控倍数

### GGPoker SPINS 额外机制

| 机制 | 说明 |
|------|------|
| Satellite 导向 | SPINS 奖金可转换为更高级别赛事门票 |
| 二等奖 | 前两名有时获得部分奖金（根据倍数） |
| 加速模式 | 高倍数场次有额外加速盲注结构 |
