## Tournament SNG Phase 1 规格 — 初步框架

> 下轮完整输出，此处先记录框架

### SNG vs Cash Game 核心差异

| 维度 | Cash Game        | SNG                  |
| ---- | ---------------- | -------------------- |
| 入场 | 随时，随金额买入 | 固定买入（固定筹码） |
| 离场 | 任意时刻         | 泡沫前不可离开       |
| 时间 | 无限制           | 倒计时/Turbo         |
| 奖励 | 即时输赢         | 名次奖励池分配       |

### SNG Phase 1 需求

1. **固定买入**: 固定筹码量（如 $10 买入 = 1000 筹码）
2. **满人开局**: 8/9 人满桌开始
3. **延迟开始**: N 分钟倒计时等待
4. **淘汰出局**: 筹码归零者出局
5. **奖励分配**: 前3名按比例分配奖池
