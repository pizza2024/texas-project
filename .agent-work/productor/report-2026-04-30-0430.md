**P2-DEPOSIT-UX-002 / P2-WITHDRAW-UX-001~004 充提 UX 改进方案 — 深度竞品分析**

### 一、充值地址簿（P2-DEPOSIT-UX-002）

**竞品参考：**

| 平台 | 地址簿实现 | 特点 |
| ---- | --------- | ---- |
| CoinPoker | ✅ 完整地址簿 | 标签 + 最近使用 + 链选择（TRC20/ERC20） |
| GGPoker | ✅ 地址簿 | 仅限提现，支持"常用地址"标记 |
| 888poker | ❌ 无地址簿 | 每次手动输入 |
| Binance | ✅ 完整地址簿 | 白名单管理 + 链名 + 备注 |

**本项目方案：**
```
DepositAddress {
  id, userId, address, chain: 'TRC20'|'ERC20',
  label: string（如"我的 TP 钱包"）,
  isDefault: boolean, createdAt
}
```

- API: `GET/POST/DELETE /api/deposit/addresses`
- 充值页面：下拉选择已有地址 + "新增地址"表单
- 支持设为默认地址
- 标签本地化（前端存 i18n key）

### 二、提现地址簿（P2-WITHDRAW-UX-001）

**竞品参考：**

| 平台 | 地址簿 | 额外功能 |
| ---- | ------ | -------- |
| CoinPoker | ✅ 完整 | 提现前双重确认（邮件通知） |
| Kraken | ✅ 完整 | 地址白名单 + 设备认证 |
| BetMGM | ✅ 完整 | 24h 冷却期（新地址） |

**本项目方案：**
```
WithdrawAddress {
  id, userId, address, chain,
  label: string, isTrusted: boolean（30天后自动升为 trusted）,
  createdAt
}
```

- API: `GET/POST/DELETE /api/withdraw/addresses`
- 新地址 24h 冷却（不可提现）
- 提现历史可"复用地址"
- 提现确认页显示地址标签（如「我的 TP 钱包 0x...1234」）

### 三、提现手续费透明化（P2-WITHDRAW-UX-002）

**竞品参考：**

| 平台 | 手续费展示 | 透明度 |
| ---- | --------- | ------ |
| CoinPoker | 固定 1 USDT 或 0.5% | 明确 |
| GGPoker | 按 VIP 等级：0.5%~2% | 分级 |
| Binance | 固定 0.8 USDT（TRC20） | 明确 |
| 888poker | 最高 3% | 不透明 |

**本项目建议：**
- 提现表单内实时计算：「实际到账 = 输入金额 - 手续费」
- 显示：手续费率 + 固定费 + 网络费拆解
- VIP 等级显示专属费率
- 每日/每月提现限额（admin 可配置）
- API: `GET /api/withdraw/fee-estimate?amount=X&chain=Y`

### 四、提现进度追踪（P2-WITHDRAW-UX-003）

**竞品参考：**

| 平台 | 状态流 | 追踪粒度 |
| ---- | ------ | -------- |
| CoinPoker | PENDING → PROCESSING → ON-CHAIN → CONFIRMED | 每步时间戳 |
| Binance | PENDING → 审核中 → 完成 | 粗粒度 |
| Kraken | Submitted → Pending → Processing → Complete | 4 步 |

**本项目方案：**
```
WithdrawRequest {
  status: PENDING | PROCESSING | ON_CHAIN | CONFIRMED | FAILED
  statusHistory: [{ status, timestamp, note? }]
  txHash?: string
  estimatedTime?: string
}
```

- 每个状态变更推送 WebSocket `withdraw_status_update` 事件
- PROCESSING 阶段显示预计完成时间
- ON_CHAIN 阶段直接链接到区块浏览器
- FAILED 状态显示原因（链拒绝/风控拦截）

### 五、提现二次确认安全（P2-WITHDRAW-UX-004）

**竞品参考：**

| 平台 | 二次确认 | 大额验证 |
| ---- | ------- | -------- |
| Kraken | 邮件确认链接 | Yubikey/TOTP |
| GGPoker | 支付密码确认 | TOTP（>$500） |
| CoinPoker | 无（邮件通知） | 无 |

**本项目建议：**
- 提现确认页：「您即将提现 X USDT 到地址 [标签] 0x...」
- 大额（>$200 等值）触发 Google Authenticator TOTP 验证
- 提现成功邮件 + 站内通知双通道
