## 规格确认更新（P1-BLAST-012）

**本轮确认结论（最终版）：**

| 规格项          | 确认值                 | 说明                                                     |
| --------------- | ---------------------- | -------------------------------------------------------- |
| 游戏总时长      | 固定 3 分钟            | `BLAST_TOTAL_DURATION_MS = 180000`                       |
| 买入筹码        | 18,000（9,000 BB）     | `BLAST_INITIAL_CHIPS`                                    |
| 最大玩家数      | 3 人                   | `BLAST_MAX_PLAYERS`                                      |
| 奖池公式        | buyin × 3 × multiplier | 每人买入 18,000                                          |
| 奖池分布        | 70% / 20% / 10%        | `BLAST_PRIZE_BASIS_POINTS`                               |
| Multiplier 范围 | **简化版 2x/5x/10x**   | 决策：Phase 4 先用简化版，完整版（2x~12,000x）作后续扩展 |
| 提前出局        | 触发 forfeit           | `forfeitBlast()` 调用                                    |
| 出局惩罚        | 退还剩余 chips         | `returnFrozenChips()` 有事务保护 ✅                      |
| 无退款机制      | 是                     | forfeit 玩家拿回剩余 chips，无额外退款                   |
| 游戏提前结束    | 无                     | 固定 3 分钟，Phase 3 设计如此                            |

**决策记录：** Multiplier 范围采用简化版（2/5/10x），理由：

1. Phase 4 前端开发需要确定性规格
2. GGPoker 完整版（12,000x）实际触发率极低（<0.001%），对用户体验影响小
3. 后续可通过配置化扩展为完整版
