## 本轮调研主题：竞品变现与留存机制深度分析

### WSOP App 变现体系

**虚拟货币系统：**

- Gold Coins (GC) — 免费金币，每日登录/任务获取
- Poker Stars (PS) — 付费货币，用于高级赛事
- Ring Games 抽水 + MTT 报名费 为主要收入来源

**留存机制：**
| 功能 | 说明 |
|------|------|
| Daily Bonus | 每日首次登录奖励，连击递增 |
| Bonus Chase | 完成特定牌局任务获取额外奖励 |
| Achievement System | 达成里程碑解锁头像框/背景 |
| Club System | 私人俱乐部促进社交留存 |
| WSOP Circuit | 真实世界赛事卫星赛 |

**赛事结构：**

- Daily Tournaments（每日多场）
- Weekly Events（周日大赏）
- Satellite Tournaments（卫星赛晋级路径）
- WSOP Main Event（年度最高奖池）

### CoinPoker 变现体系

**核心差异：**

- USDT 真实货币交易，规避应用商店支付分成（30%）
- Crypto shuffle 验证 —— 区块链可验证公平性（差异化卖点）
- Rakeback 高达 55%（vs 行业平均 20-30%）

**Rakeback Tier 结构：**
| 等级 | Rakeback | 月累计 Rake |
|------|----------|-------------|
| Bronze | 15% | $0+ |
| Silver | 25% | $500+ |
| Gold | 35% | $2,000+ |
| Platinum | 40% | $10,000+ |
| Diamond | 55% | $50,000+ |

**赛事系统：**

- Crypto Series of Poker (CSOP) — 自有品牌赛事系列
- 每日/每周 MTT + SNG
- Freeroll 免费赛（拉新）
