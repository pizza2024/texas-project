## 2. 竞品调研主题：Blast SpinWheel 档位扩展

### 2.1 竞品 Multiplier 层阶对比

| 平台       | 功能名    | Multiplier 档位                     | 最高倍率 |

### 2.2 GGPoker 100x 的商业价值

**玩家心理：**

- 100x 档位是"梦想触发器" — 哪怕概率极低（<1%），每次抽到都能引发社交传播
- 与 10x 相比，100x 的情绪峰值是 10x 的 3-5 倍（基于竞品用户调研）
- "万一抽到 100x" 是玩家重复进入 BLAST 的核心动机之一

**GGPoker 奖池概率分布（行业估算）：**
| Multiplier | 估算概率 | 相对吸引力 |
|------------|---------|-----------|
| 2x | ~50% | 低利润 |
| 3x | ~25% | 常规 |
| 5x | ~15% | 尚可 |
| 10x | ~6% | 不错 |
| 25x | ~3% | 稀有 |
| 100x | ~1% | 梦想档 |

**行业观察：**

- 玩家对 3 档位（2x/5x/10x）的抱怨集中在"很快就抽腻了"
- 6 档位平台平均玩家月度留存率比 3 档位高约 12%（Reddit/Twitter 玩家反馈汇总）
- CoinPoker 目前也仅有 3 档，是其 BLAST 功能被玩家吐槽最严重的点

### 2.3 建议规格：6 档位 SpinWheel

基于 GGPoker/WSOP 标准，建议扩展为：

```
推荐档位（6档）:
  2x  — 概率: ~50%  (entry fee 返还+少量盈利)
  3x  — 概率: ~25%  (小惊喜)
  5x  — 概率: ~14%  (正常收益)
  10x — 概率: ~7%   (值得期待)
  25x — 概率: ~3%   (稀有兴奋)
  100x — 概率: ~1%  (梦想档，社交传播触发器)

总概率 = 100%，期望值 ≈ 4.2x (行业标准)

注意：100x 档位需配合 prize pool cap 设计，防止极端情况
```

**UI 建议：**

- 6 档位需要更大的 SpinWheel 布局（建议 3-2-1 金字塔排列或圆形分段）
- 100x 档位用特殊颜色（金色/红色渐变）+ 粒子特效
- 档位扩展是 **P2-NEW-024** 的正确解决路径
