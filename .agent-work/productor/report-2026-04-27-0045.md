## 本轮调研 — GGPoker SPINS (Blast) 机制深度解析

> **背景:** Blast/SPINS 是 GGPoker/PokerStars 的 3人即时赛事产品，奖池随机化（最高 10000x），单桌即开即玩，时长 5-10 分钟。已在 task-queue 列为 P1-BLAST-001 规格待定义。本轮深度研究竞品机制，为 Coding 提供规格输入。

| 维度     | GGPoker SPINS                    | PokerStars Blast      | 本项目现状  |

| 参赛人数 | 3 人                             | 3 人                  | ❌ 不支持   |
| 参赛费用 | $0.25 / $1 / $3 / $5 / $10 / $25 | $0.25~$10             | ✅ 需扩展   |

| ------ | ------------- | -------------- |
| 2x     | 74.0%         | 69.0%          |

## 改进建议

### P1 — Blast/SPINS 即时赛事规格（P1-BLAST-001）

**核心设计决策（需 Coding 确认）:**

1. **奖池模型:** 固定 5x 最低奖池 + 随机乘数 (2x~10000x)，或纯随机 (2x~10000x)
2. **参赛人数:** 3 人固定，不接受第 4 人
3. **随机算法:** 使用 `crypto.randomInt()` 生成乘数，对应概率表
4. **Jackpot 池:** 每手提取 0.5% 计入 jackpot 池，满触发条件后全服广播
5. **强制 showdown:** 5 分钟计时器归零时，所有 active 玩家自动 all-in

**建议字段扩展:**

```ts
// Room model 扩展
interface BlastConfig {
  buyin: number; // 参赛费用
  minPlayers: 3; // 固定 3 人
  maxPlayers: 3;
  prizeMultiplier: number; // 随机乘数（2~10000）
  isJackpot: boolean; // 是否为 jackpot 局
  timerSeconds: number; // 强制 showdown 倒计时
}

// ScheduleEntry 扩展
interface ScheduleBlastEntry {
  type: "BLITZ" | "SPINS";
  buyinLevels: number[];
  prizeDistribution: { minMultiplier: number; probability: number }[];
}
```
