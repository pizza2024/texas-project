## 本轮主题：Blast 即时赛事竞品深度调研 + 移动端体验新趋势

### 1. Blast / SPINS 即时赛事竞品分析

#### GGPoker SPINS（行业标杆）

| 参数         | 值                               |
| ------------ | -------------------------------- |
| 最低买入     | $0.25 / $1 / $3 / $5 / $10 / $25 |
| 奖池倍率     | 2x ~ 12,000x                     |
| 初始筹码     | 500                              |
| 盲注递增     | 3 分钟                           |
| 最低开赛人数 | **3人**（满即开，无等待）        |
| 特点         | 随机奖池在发牌前揭示（悬念机制） |

#### CoinPoker BLAST（即时赛事）

| 参数         | 值              |
| ------------ | --------------- |
| 最低买入     | $0.25 / $1 / $3 |
| 奖池倍率     | 2x ~ 10,000x    |
| 初始筹码     | 500             |
| 盲注递增     | 2 分钟          |
| 最低开赛人数 | 3人             |

#### WSOP即时赛（World Series of Poker）

- 主要以 $1/$3/$5 买入为主
- 与 SPINS 机制类似但 UI 更传统
- 有 wsop.com 独立 App

#### 888poker BLAST

- $0.10/$0.25/$1 三档
- 2x~1000x 奖池
- 更适合微注额玩家

#### 共同设计模式

```
1. 3人即开（0等待）→ 立即开始
2. 随机奖池揭示后开始（悬念 + 社交分享动机）
3. 2分钟/3分钟短盲注周期 → 快速赛局
4. 初始500筹码 → 快速 all-in 决策
5. 奖池 2x~10000x → 高波动性（娱乐性）
```

**对 P1-BLAST-001 的建议规格:**

| -------- | -------------------- | ---------------------------- |
| 买入     | $0.25 / $1 / $3 三档 | 与竞品对齐                   |

| 开赛人数 | 3人                  | 竞品标准                     |
| 初始筹码 | 500                  | 竞品标准                     |

### 2. 移动端体验新趋势（2026）

#### GGPoker 移动端亮点

- **手势操作**: 滑动 + 按住拖拽，替代所有按钮操作
- **快速操作栏**: 底部固定，半透明，不遮挡牌桌
- **多桌切换**: 桌卡预览 + 滑动画廊
- **表情系统**: 桌内 emoji 实时广播（已完成 ✅ P2-EMOJI-001）

#### PokerStars 隐私创新

- **匿名桌**: 隐藏用户名，只显示头像
- **HUD 防护**: 自动屏蔽第三方追踪软件
- **无压力模式**: 新手可开启，隐藏对手数据

#### CoinPoker Web3 原生优势

- 链上即时充值（平均 < 5 分钟确认）
- 提现 30 分钟到账（核心竞争点）
- USDT-TRC20 无 Gas 费
