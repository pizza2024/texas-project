## MVP 方案（Phase 1）

### 数据库模型

```typescript
model VipTier {
  id            String   @id @default(uuid())
  name          String   // BRONZE, SILVER, GOLD, PLATINUM, DIAMOND
  minRakeMonthly Int     // 晋级门槛（本月累计 rake）
  rakebackRate  Float    // 返利比例：0.05 = 5%
  color         String   // UI 显示色
  icon          String   // emoji 或 icon name
  sortOrder     Int
}

model UserVipProgress {
  id              String   @id @default(uuid())
  userId          String   @unique
  currentTierId   String
  currentRake     Int      @default(0)   // 本周期累计 rake
  lifetimeRake    Int      @default(0)   // 终身 rake（用于徽章）
  updatedAt       DateTime @updatedAt
}

model RakebackLog {
  id        String   @id @default(uuid())
  userId    String
  amount    Int      // 返还筹码
  period    String   // "2026-W17"
  tierId    String   // 当时的 tier
  createdAt DateTime @default(now())
  @@index([userId, period])
}
```


### Rake 计算规则

```typescript
// 每手牌 rake = min(pot * 0.05, maxRakePerHand)
// 玩家贡献 rake = 玩家投入 pot / 桌总 pot * 本局总 rake
// 月度 rake = sum(玩家贡献 rake)
```
