### SNG 核心结构（本项目 Phase 1 参考）

```
SNG Phase 1 = 单桌赛（Sit & Go）
├── 参赛人数：6人 / 9人
├── 买入结构：固定买入（无 Rebuy）
├── 座位：随机分配
├── 赛制：常规 / Turbo（盲注加速）
├── 盲注上涨：固定时间间隔（如 10分钟）
├── 奖励结构：Top 2 或 Top 3 派发（典型 70/30 或 60/30/10）
└── 抽水：平台从买入费中抽取 5-10%
```

### Jackpot SNG 特别研究（GGPoker Mega Spin）

| 特性       | 规格                                     |
| ---------- | ---------------------------------------- |
| 基础买入   | $1 / $3 / $5 / $10 / $25                 |
| 奖池倍数   | 2x ~ 12,000x 随机                        |
| 概率分布   | 高倍数极低概率（12,000x 约 1/1,000,000） |
| 实现复杂度 | 高（需随机倍数生成 + 独立奖池账本）      |
| 建议阶段   | Phase 2（6个月后）                       |

**对 Phase 1 的启发**：Jackpot SNG 的核心是"小概率大奖励"心理设计，Phase 1 应先做标准 SNG，验证用户参与度后再叠加。

### 变现结构参考

| 场景        | 玩家付费点             | 平台收入 |
| ----------- | ---------------------- | -------- |
| 常规桌      | 抽水（每局 pot 的 5%） | 持续小额 |
| SNG         | 报名费（买入的 5-10%） | 赛前预收 |
| Jackpot SNG | 报名费 + 奖池叠加      | 最高杠杆 |
| MTT         | 报名费 + Late Reg      | 峰值集中 |

**本项目当前**：常规桌抽水架构已就绪（TierConfig.rake），缺少 SNG/MTT 的赛事管理模块。

## Phase 1 Tournament PRD 草稿

### 概述

**目标**：在现有 cash game 基础上，新增 SNG 单桌赛模式，作为 Phase 1 变现扩展。

### 用户故事

```
作为玩家，我希望快速加入一场 6 人 SNG，
这样我可以在 15-30 分钟内完成一场比赛并获得排名奖励。

作为玩家，我希望看到我的 SNG 历史战绩，
这样我可以分析我的锦标赛表现。
```

### 核心功能

| 功能           | 描述                                      | 优先级 |
| -------------- | ----------------------------------------- | ------ |
| SNG 房间创建   | 管理员创建 SNG 房间（配置买入/人数/速度） | P0     |
| SNG 房间列表   | 前端新增 SNG 房间 Tab，与 cash 房间并列   | P0     |
| SNG 加入队列   | 玩家支付买入 → 进入 SNG 等待队列          | P0     |
| 人数齐自动开赛 | 6人齐 → 随机座位 → 开始 preflop           | P0     |
| 盲注定时上涨   | 每 N 分钟盲注自动上涨（如 10min）         | P0     |
| SNG 结算       | 比赛结束，按排名分配奖池（扣除抽水）      | P0     |
| Speed 变体     | Turbo（盲注上涨时间减半）                 | P1     |
| SNG 历史记录   | 玩家查看过往 SNG 战绩                     | P2     |

### 数据库模型扩展建议

```prisma
model SngRoom {
  id          String   @id @default(uuid())
  name        String
  buyIn       Int      // 买入金额（chips）
  maxPlayers  Int      // 6 或 9
  speed       String   // REGULAR / TURBO / HYPER
  prizeTop2   Int      @default(70)  // 第一名获得奖池百分比
  prizeTop3   Int?     // 如果 maxPlayers > 6，第三名百分比
  blindMinutes Int     // 盲注上涨间隔（分钟）
  status      String   // WAITING / IN_PROGRESS / COMPLETED
  startsAt    DateTime?
  createdAt   DateTime @default(now())
}

model SngEntry {
  id        String   @id @default(uuid())
  roomId    String
  oderId    String
  buyIn     Int
  rank      Int?     // 比赛结束时的排名
  prizeWon  Int      @default(0)  // 获得的奖金（扣除抽水后）
  createdAt DateTime @default(now())
}
```
