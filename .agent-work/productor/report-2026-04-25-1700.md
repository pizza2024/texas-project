## 本轮深度竞品调研：手牌复盘功能

### 为什么关注手牌复盘？

手牌复盘（Hand Replay）是提升玩家技艺的核心功能，也是竞品差异化的关键战场。WSOP、888poker、CoinPoker 均投入大量工程资源打造各自特色的复盘体验。

### 手牌复盘功能分级

| 功能层级    | 描述                                               | 代表竞品                              |
| ----------- | -------------------------------------------------- | ------------------------------------- |
| **L1 基础** | 历史手牌列表 + 单局回放（按操作步骤前进/后退）     | 本项目 Phase 1 ✅                     |
| **L2 进阶** | equity 曲线、Pot Odds 浮窗、播放速度控制、自动播放 | 本项目 Phase 2 📋                     |
| **L3 高级** | 决策点高亮、胜率模拟、教练提示                     | WSOP、888poker                        |
| **L4 社交** | 复盘导出分享、视频生成、社区讨论                   | WSOP（视频）、CoinPoker（区块链验证） |

### 竞品复盘功能详细对比

#### WSOP（World Series of Poker）

| 功能                      | 状态      |
| ------------------------- | --------- |
| 基础回放（按步前进/后退） | ✅        |
| Equity 曲线图             | ✅        |
| Pot Odds 计算器           | ✅        |
| 自动播放模式              | ✅        |

**亮点：** WSOP 允许玩家将复盘导出为短视频，一键分享到社交媒体。这是本项目缺失的核心差异化功能。

### 本项目 Hand Replay Phase 2 规格确认

基于竞品分析，Phase 2 应聚焦以下功能（优先级排序）：

#### ✅ 必须实现（L2 核心）

1. **Equity 曲线图**
   - 显示每名玩家在每个回合的胜率变化
   - 参考 WSOP 样式：折线图，覆盖率可视化
   - 技术方案：Recharts `LineChart`，数据来自 `GET /hands/:id/replay`

2. **Pot Odds 浮窗**
   - 悬停/点击显示当前 pot odds
   - 底池大小、跟注额、赔率计算
   - 技术方案：React 浮层组件，实时计算

3. **播放速度控制**
   - 0.5x / 1x / 2x / 4x 速度切换
   - 影响自动播放间隔时间
   - 技术方案：PlaybackControls 组件

4. **自动播放模式**
   - 启动后按设定速度自动前进
   - 可暂停/中断
   - 技术方案：useEffect + setInterval，Phase 2 核心功能

### 关键发现：房间内聊天是 P1 优先级

**现状：**

- WebSocket `chat-message` 事件已实现（game.handler.ts）
- 前端 `/room/[id]` 页面**没有**聊天 UI
- 这是本项目与所有主流竞品差距最明显的地方

**竞品参考（WSOP）：**

- 聊天消息气泡显示在牌桌底部
- 支持 emoji 快捷输入
- 有"折叠/展开"切换，不干扰游戏界面

**建议实现方案：**

1. 在 `/room/[id]` 牌桌区域底部新增 `ChatPanel` 组件
2. 复用 WebSocket `chat-message` 事件（已有后端支持）
3. MVP：仅文字聊天 + emoji 快捷按钮
4. 后续迭代：添加聊天历史、@mention、禁言管理

## Tournament SNG Phase 1 规格确认

### 8人 SNG 锦标赛 — 规格文档

```
赛制：
- 参赛人数：8人
- 满员开赛（无需等待）
- 不可重买（Phase 1 简化版）

Buy-in 选项：
- 500 chips（微额）
- 1,000 chips（标准）
- 5,000 chips（高额）

奖金分配（#1/#2/#3）：
- 第一名：60%
- 第二名：30%
- 第三名：10%

游戏规则：
- 盲注结构：每 3 分钟涨盲
- 前两轮为 Freeze-out（出局后无法重买）
- 桌上每名玩家必须有筹码才能参与

UI/UX 流程：
1. 从 `/rooms` 进入 SNG专区
2. 选择 buy-in 级别
3. 进入 SNG 大厅等待（最多8人）
4. 满8人自动开始
5. 逐步淘汰，直到决出前三
6. 显示最终排名和奖金
```

### 技术注意事项

- **Phase 1 简化**：暂不支持 re-entry、re-buy、late registration
- **与现有房间系统整合**：复用 `Room` 和 `Table` 数据模型
- **新字段**：`Room.isTournament = true`，`Room.tournamentType = 'SNG'`
- **Phase 2** 扩展：Jackpot SNG（随机奖金池）、MTT 多桌赛
