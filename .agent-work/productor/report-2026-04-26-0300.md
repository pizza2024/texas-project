## 本轮调研：产品路线图分析 — "首次充血" + "每日任务" 机制

### 为什么要关注首充和每日任务？

根据竞品分析（WSOP、888poker、CoinPoker），**首充奖励 + 每日任务** 是 Freemium 扑克游戏最核心的留存和变现机制：

- **WSOP**：Gold Chips 内购 + 每日登录奖励 + 连胜奖励
- **888poker**：每日任务（"今天赢3手牌"）+ 经验值等级系统
- **CoinPoker**：首充 100% match + USDT 即时到账

### 建议规格：P1 首充奖励（P1-FIRSTDEP-001）

**规格草案**：

```
首充 100% Match
├── 最小充值: 10 USDT
├── 最大bonus: 100 USDT（即充100得200）
├── 释放方式: 线性释放
│   └── 每日释放 10%，持续10天
├── 解锁条件:
│   └── 需完成 KYC（手机验证）防薅羊毛
└── 有效期: 30天未用完则没收

数据库模型：
model FirstDepositBonus {
  id            String   @id @default(cuid())
  userId        String   @unique
  depositAmount Int      // 原始充值
  bonusAmount   Int      // 奖励金额
  released      Int      @default(0)  // 已释放
  expiresAt     DateTime // 30天后
  createdAt     DateTime @default(now())
}
```

### 建议规格：P1 每日任务（P1-DAILYMISSION-001）

**规格草案**：

```
每日3个任务，随机抽取：
├── M1: 赢下1手牌      → 50 chips
├── M2: 玩满10分钟     → 100 chips
├── M3: 淘汰1名玩家    → 75 chips
└── M4: 赢下3手牌      → 200 chips

Streak 机制：
├── 连续7天完成全部3任务 → 第8天奖励 2x
└── 中断则重置为 1x

数据库模型：
model DailyMission {
  id        String          @id @default(cuid())
  userId    String
  date      DateTime        // 每日重置
  missions  Json            // { id, type, progress, completed }[]
  streak    Int      @default(0)
  createdAt DateTime @default(now())
}
```
