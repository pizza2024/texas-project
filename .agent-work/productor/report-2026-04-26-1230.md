## 竞品调研：GGPoker 新功能 — "Snapcam"

### 功能概述

GGPoker 2024年推出的 Snapcam 是一种低成本玩家互动功能：

| 特性 | 描述 |
|------|------|
| 形式 | 3秒短视频/静态头像框反应 |
| 触发 | 牌局关键时刻（all-in、获胜、清台） |
| 存储 | GGPoker 云端，7天有效期 |
| 带宽 | 低 — 仅关键时刻上传 |
| 竞品参考 | WSOP 表情反应、Partypoker 语音气泡 |

### 对本项目的参考价值

**已规划为 P2-EMOJI-001（表情反应系统）**，但规格尚未定义。建议：

1. **Phase 1**：静态头像框切换（低实现成本）
   - 预设 10-20 个头像框样式
   - 玩家在设置页选择
   - 牌桌上显示（代替当前纯文字昵称）

2. **Phase 2**：3秒语音气泡
   - 预设音效（笑声、"GG"、"Nice call"）
   - WebSocket 广播到同桌玩家

**建议优先级**：可作为 P2 优化，建议先完成 P1-SCHEDULE-001 前端 UI。
