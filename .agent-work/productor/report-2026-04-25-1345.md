## 竞品调研 — GGPoker Squash Hand Replay

### GGPoker Squash 是什么

GGPoker Squash 是 GGPoker 推出的移动端应用，主打休闲/社交德州扑克，与 Zynga Poker 定位类似。2024年上线，主攻亚洲和拉丁美洲市场。

### Squash 手牌复盘功能分析

| 功能模块 | GGPoker Squash 实现 | 可借鉴性 |
|----------|-------------------|----------|
| **进度控制** | 滑块控制 + 决策节点标记 | ⭐⭐⭐⭐ 直接参考 |
| **决策点高亮** | 每轮操作用色块标注（FOLD=红/RAISE=绿/CALL=蓝） | ⭐⭐⭐⭐ UI 友好 |
| **胜率显示** | Showdown 前实时胜率%（基于剩余手牌范围估算） | ⭐⭐⭐ 技术成本中等 |
| **动画过渡** | 公共牌发牌时有展开动画 | ⭐⭐⭐ 提升体验 |
| **手牌对比** | Showdown 强制展示所有未摊牌玩家手牌 | ⭐⭐⭐ 减少疑惑 |
| **分享功能** | 可生成手牌截图/链接分享 | ⭐⭐⭐ 社交传播 |

### Squash vs WSOP vs CoinPoker 对比

| 特性 | GGPoker Squash | WSOP | CoinPoker |
|------|---------------|------|-----------|
| 手牌历史 | ✅ 完整 | ✅ 完整 | ✅ 有限 |
| 进度条 | ✅ 精细 | ✅ 基础 | ❌ 无 |
| 胜率实时 | ✅ | ❌ | ❌ |
| 导出/分享 | ✅ 截图 | ✅ 截图 | ❌ |
| 动画 | ✅ 流畅 | ✅ 一般 | ✅ 基础 |

### 对本项目 Hand Replay 的建议

**必须实现（MVP）：**
1. 进度条 + 决策节点跳转
2. 每轮牌桌快照（底池、玩家状态、公共牌）
3. 操作时间线（FOLD/CALL/RAISE/ALLIN + 耗时）
4. Showdown 结果展示 + 胜负高亮

**可选增强：**
1. 实时胜率%（需接入手牌强度评估器）
2. 分享截图
3. 动画过渡

**技术路径（维持上轮判断）：**
- 前端 `/hands/[id]/replay` 新页面
- 复用 `Hand` + `HandAction` 数据
- 决策点 = `HandAction` 时间戳聚合
