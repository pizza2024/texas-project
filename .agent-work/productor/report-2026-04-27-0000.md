## 竞品调研更新

### 本轮聚焦：主流平台 Rakeback 与 VIP 体系

德州扑克平台的 Rakeback 和 VIP 忠诚度体系是玩家留存的核心机制。本轮深入调研各平台rake结构：

**GGPoker Rake 结构：**

- 现金桌 Rake：2%-5%（根据 stakes 不同）
- Tournament Fee：固定 10%（买入的 10% 作为 fee）
- Rakeback：通过 FishBuff 计划返还，最高可达 60%
- VIP 分级：6级（Copper → Silver → Gold → Platinum → Diamond → Royal）

**PokerStars Rake 结构：**

- 现金桌 Rake：3.5%-5%（分级收取）
- Tournament Fee：10%（标准）
- Rakeback：StarsCoins 兑换系统（非直接 rakeback）
- VIP：6级 Club（Bronze → Chrome → Silver → Gold → Platinum → Supernova）

**WSOP (Caesars) Rake 结构：**

- 现金桌 Rake：3%-5%
- Tournament Fee：10%
- Rakeback：Caesars Rewards 积分（非扑克专用）
- VIP：Caesars Rewards 等级制

**888poker Rake 结构：**

- 现金桌 Rake：3%-5%（分级别）
- Tournament Fee：10%-15%
- Rakeback：888poker Club（5级，从 Bronze 到 Diamond）
- 特殊：Beat the Clock 免费赛作为留存机制

**CoinPoker Rake 结构：**

- 现金桌 Rake：2%-4%（行业最低之一）
- Tournament Fee：10%
- Rakeback：CHP 代币质押机制（非直接返现）
- VIP：4级会员体系

**本项目 Rakeback 评估：**

| 等级     | Rakeback | 对标     |
| -------- | -------- | -------- |
| BRONZE   | 10%      | 行业入门 |
| SILVER   | 15%      | 标准     |
| GOLD     | 20%      | 良好     |
| PLATINUM | 40%      | 优秀     |
| DIAMOND  | 50%      | 顶级     |

**对比结论：**

- 本项目 PLATINUM (40%) / DIAMOND (50%) 已达行业顶级水准
- BRONZE/SILVER/GOLD 三档偏低，建议提升至 15%/25%/35%
- 可参考 GGPoker FishBuff 的多维度返现（每周抽签 + 固定返还）
