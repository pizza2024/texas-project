## 竞品调研（聚焦 Rakeback 体系 & Club 私人邀请码）

### Rakeback 竞品对比

| 平台       | 最高 Rakeback  | Tier 数量                  | 特色                                           |
| ---------- | -------------- | -------------------------- | ---------------------------------------------- |
| GGPoker    | 60%+           | 6+ (Bronze→Diamond)        | CHP 持有者额外叠加，GGClub 私人局专属 rakeback |
| PokerStars | 40-50%         | 5星 VIP 系统               | 按月重置，Spin & Go 专属加成                   |
| CoinPoker  | 40%            | 4 tiers                    | CHP 质押量决定 rakeback 比例，链上可验证       |
| 888poker   | 36%            | 3 tiers                    | BLAST 赛制，PokerCraft 战绩统计                |
| **本项目** | **30% (GOLD)** | **3 (BRONZE/SILVER/GOLD)** | USDT 充值已有，**无 PLATINUM**                 |

**行业趋势：**

- Rakeback 上限持续提高，GGPoker 已达 60%+
- 高端玩家（VIP） rakeback 显著高于普通玩家（差 20-30 个百分点）
- 自动周结成为行业标准（替代手动领取）
- PLATINUM(40%) / DIAMOND(50%) 是 2024-2025 年主流平台标配

### GGPoker Club 私人邀请码机制

**核心机制（基于公开资料）：**

1. **Club 创建者**生成唯一邀请码（或多个）
2. 被邀请玩家**凭邀请码加入**私人 Club
3. Club 成员只能在**私人房间**内游戏（不对公众开放）
4. Club 可设置**专属 rakeback 比例**（高于公开值）
5. Club 管理者可查看**成员战绩统计**
6. 私人 Club 是 GGPoker 最核心的**差异化竞争力**——高 rakeback + 私密局 + 社交

**对标到本项目的设计缺口：**

- 本项目 Club 无邀请码系统 → 无法实现私人局
- 无 Club 专属房间 → Club 成员只能混在公共房间
- 无 Club 战绩统计 → Club 价值大幅降低
