## 本项目现存差距分析

### 高优先级差距

| 差距                 | 影响                               | 紧迫度 | 竞品参考           |
| -------------------- | ---------------------------------- | ------ | ------------------ |
| **无站内通知中心**   | 用户无法及时获知朋友上线、赛事开始 | 🟡 中  | 全部主流平台       |
| **无忠诚/VIP 计划**  | 难以长期留存高价值玩家             | 🟡 中  | 全部主流平台       |
| **无 Club/公会系统** | 社交深度不足，难以形成玩家社区     | 🟡 中  | GGPoker/PokerStars |
| **无成就徽章系统**   | 任务完成后缺乏成就感反馈           | 🟢 低  | GGPoker/PokerStars |

### P2-NOTIFY-001 站内通知中心 — 建议规格

基于竞品分析，站内通知应包含：

| 通知类型       | 触发时机                    | 优先级 |
| -------------- | --------------------------- | ------ |
| 朋友上线       | friend_status → ONLINE      | 高     |
| 朋友开始游戏   | friend_status → PLAYING     | 高     |
| 锦标赛即将开始 | tournament.startsAt - 15min | 高     |
| 充值到账       | deposit.confirmed           | 高     |
| 提现状态更新   | withdraw.statusChanged      | 中     |
| 每日任务完成   | mission.completed           | 低     |
| 新好友申请     | friend.requestReceived      | 中     |
