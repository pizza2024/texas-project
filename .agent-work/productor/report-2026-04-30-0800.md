## P2-WITHDRAW-UX-004 规格草案

**缺失：** 提现历史追踪（txHash + 状态时间线）

竞品参考：
- **Coinbase**: 每笔提现显示状态（Pending → Processing → On-chain → Complete）+ txHash 链接 + 时间戳
- **Binance**: 提现历史包含：时间、币种、数量、状态、目标地址（脱敏）、txHash（可复制）、预计到账
- **GGPoker**: 提现记录显示申请时间、金额、手续费、实际到账、状态

**建议规格：**

```
提现历史页面（/wallet/withdraw/history 或 /withdraw）
每条记录显示：
- 申请时间（本地时区）
- 提现金额（USDT）
- 手续费（分离显示，与 P2-WITHDRAW-UX-002/003 一致）
- 实际到账金额
- 状态 + 时间线：
  PENDING → PROCESSING → CONFIRMED（链上成功）/ FAILED（链上失败）
- txHash（可点击跳转 Etherscan）
- 目标地址（脱敏：前6位...后4位）
```
