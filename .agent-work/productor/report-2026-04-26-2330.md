## 本轮调研 — Blast 即时赛事深度分析

### 核心定位

Blast（参考 GGPoker SPINS）是 **3人极速即时赛事**，报名后 30 秒内开赛，随机奖池倍数，最高 50,000X，约 3 分钟完成。

### 竞品对比

| 平台       | 产品名       | 人数 | 时长   | 最高倍数 | 特色              |
| ---------- | ------------ | ---- | ------ | -------- | ----------------- |
| GGPoker    | SPINS        | 3    | 3-5min | 12,000X  | 3D 轮盘动画       |
| CoinPoker  | Flipout      | 3    | ~3min  | 1,000X   | 加密货币积分      |
| ACR        | On Demand    | 3-4  | ~3min  | 7,500X   | 经典设计          |

### Blast 与常规 SNG 的关键差异

| 维度 | SNG                | Blast              |
| ---- | ------------------ | ------------------ |
| 人数 | 6-9人              | 3人                |
| 时长 | 15-30min           | ~3min              |
| 奖池 | 固定（1.5-2X买入） | 随机（1X-50,000X） |
| 匹配 | 等待满人           | 即时（最长30秒）   |
| 盲注 | 常规递增           | 2分钟加速结构      |
