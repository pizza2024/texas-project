## 竞品调研：WSOP App 重点分析

### WSOP (World Series of Poker) — 行业标杆

**平台概况**
| 维度 | 数据 |
|------|------|
| 成立时间 | 2005年（线下1970年） |
| 移动端 | iOS/Android 双平台 |
| 特色 | WSOP 线下巡回赛资格 + 金币/真钱双模式 |
| 玩家规模 | 全球头部（估算 50万+ MAU） |

**核心产品特性**
| 功能 | 实现方式 | 对本项目启示 |
|------|----------|--------------|
| 金币/真钱双模式 | 虚拟货币 + USDT 充值分离 | ✅ 本项目已有 chips/wallet 分离 |
| 线下资格赛 | Satellite 锦标赛晋级体系 | P3 — Tournament |
| 每日签到奖励 | 连续登录激励 | P3 — 每日奖励 |
| VIP 忠诚计划 | 基于充值/游戏时长分级 | P3 — Rakeback 体系 |
| 表情互动 | 牌桌内表情包 | P3 — 表情系统已有基础 |
| 记牌器 | 历史手牌回顾 | P3 — hands 页面已有 |

**UX 设计亮点**
1. **双货币隔离** — 金币（社交）/真钱（竞争）心理分区明确
2. **线下资格赛包装** — 线上游戏 → 线下大赛的梦想激励
3. **签到连续奖励** — 第1/3/7天不同奖励档位，DAU 利器
4. **锦标赛卫星赛** — 低买入晋级高奖金，付费转化强

**本项目对齐度评估**
