## 本轮产品调研：锦标赛盲注结构可视化

### 竞品最佳实践分析

基于对 WSOP、888poker、CoinPoker、PartyPoker 的公开产品调研：

#### 1. 倒计时计时器设计
- **顶部固定条：** 当前盲注（BB/SB）+ 剩余时间 + 进度条，始终可见
- **最后 30 秒：** 背景色温渐变（蓝 → 黄 → 红）作为视觉预警
- **WSOP 移动端：** 计时器内嵌在牌桌顶部栏，不可被滚动移除

#### 2. 盲注结构表（预告知）
- **Hover/Tap 预览：** 鼠标悬停或点击盲注数字，显示完整结构表
  - 级别 | 大盲 | 小盲 | 暂停时间 | 底注（ante）
  - 下一级别预览高亮标注
- **GTOWizard：** 可视化时间轴，鼠标移到某级别显示对应筹码量
- **CoinPoker：** 锦标赛大厅直接展示 "Level 12/24 | 8:32 remaining"

#### 3. 锦标赛大厅展示
- 当前级别 + 下一个级别并排（"Level 5 → Level 6"）
- 预计结束时间（基于当前结构）
- 奖池分布饼图（直接集成，不跳页）

#### 4. 特殊状态标注
- **Break 提醒：** "Level 9 后 15 分钟休息" — 提前 2 分钟弹出
- **Day 切换：** 多日锦标的 Day 1 → Day 2 切换提示
- **Late Registration：** 截止时间倒计时高亮

#### 5. 进阶功能（高价值差异化）
- **Chip EV 提示：** 当前级别下 100BB 筹码的期望价值
- **ICM 计算器：** 决赛桌阶段的独立 ICM 实时估算弹窗
- **Payout Structure：** 一键查看奖池分布表

### 本项目现状评估

当前 `Table` 数据模型中，`Room.blindLevel` 仅存储单一字符串，没有：
- 级别时间（每级持续多久）
- 下一级别预告
- 暂停/break 时间
- 底注（ante）结构

### 改进建议规格

```
UI 组件：<BlindStructureBadge> + <BlindCountdownBar> + <BlindStructureModal>

数据扩展（Room 或 Tournament）：
- levelCount: number        // 总级别数
- currentLevel: number     // 当前级别
- levelDurationMinutes: number   // 每级持续分钟
- breakDurationMinutes: number    // 休息时长
- anteEnabled: boolean     // 是否有底注
- levels: Array<{level, bb, sb, ante?, isBreak?}>

UI 行为：
1. 顶部栏固定显示：BB | 倒计时 | 进度条
2. 点击/悬停显示完整结构表（含下一级预览）
3. 最后 30 秒：进度条颜色蓝→红
4. Break 前 2 分钟：toast 提示
```
