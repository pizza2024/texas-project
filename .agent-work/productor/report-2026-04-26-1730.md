## P1 问题产品影响评估

### P1-WALLET-001 — `exchangeBalanceToChips` 非原子事务 ⚠️

**风险评级: 高（资金安全）**

当前代码模式（wallet.service.ts:338-380）：

```
await this.prisma.wallet.upsert({ ... });   // ✅ 成功
await this.prisma.user.update({ ... });       // ❌ 失败 → wallet 扣款，chips 未到账
await this.prisma.transaction.create({ ... }); // ❌ 失败 → 审计日志丢失
```

**对比:** `exchangeChipsToBalance` 正确使用了 `$transaction`，`exchangeBalanceToChips` 则没有。

**产品影响:**

- 用户充值 USDT → chips，扣款成功但筹码未到账（资金凭空消失）
- 用户投诉率上升
- 建议: **立即用 `$transaction` 包裹，与反向操作保持一致**

### P1-WITHDRAW-005/006/007 — 提现事务问题

**产品影响:** 提现数据不一致（DB 状态与链上状态），可能导致重复转账或资金冻结用户账户。
