## 本轮专题：P2-NEW-024 SpinWheel 档位扩展规格

**背景:** 当前 SpinWheel 仅 3 档（2x/5x/10x），竞品 GGPoker Spin & Gold 有 9 档（最高 12000x），档位过少导致大奖感不足、用户留存动力弱。

### 档位规格（9档，参考 GGPoker 概率分布）

```typescript
// apps/web/components/blast/SpinWheel.tsx
const SEGMENTS = [
  { multiplier: 2, color: "#22c55e", label: "2X", probability: 0.36 }, // 绿色 — 常见
  { multiplier: 3, color: "#3b82f6", label: "3X", probability: 0.28 }, // 蓝色
  { multiplier: 5, color: "#f59e0b", label: "5X", probability: 0.2 }, // 琥珀
  { multiplier: 10, color: "#ef4444", label: "10X", probability: 0.1 }, // 红色
  { multiplier: 25, color: "#a855f7", label: "25X", probability: 0.04 }, // 紫色
  { multiplier: 50, color: "#ec4899", label: "50X", probability: 0.015 }, // 粉色
  { multiplier: 100, color: "#f97316", label: "100X", probability: 0.005 }, // 橙色
  { multiplier: 250, color: "#1e3a5f", label: "250X", probability: 0.001 }, // 深蓝
  {
    multiplier: 12000,
    color: "#ffd700",
    label: "12000X",
    probability: 0.00005,
  }, // 金色 — 超级 jackpot
];
```

**概率设计说明:**

- 2x~5x 合计占 84%：确保低额奖金高频出现，保持游戏节奏
- 10x~50x 合计占 15.5%：中等惊喜，适合娱乐性留存
- 100x~250x 合计占 0.6%：高价值目标，驱动重玩动机
- 12000x 概率 0.005%（即 1/20000）：超级 jackpot，核心营销噱头

**档位概率不等于旋转指针停留概率** — 需要 `crypto.getRandomValues()` 生成 0~1 随机数后二分查找对应档位。

### 后端修改

**`apps/backend/src/table-engine/blast.service.ts`** — `drawBlastMultiplier()`:

```typescript
// 权重数组（按概率排序）
const MULTIPLIERS = [2, 3, 5, 10, 25, 50, 100, 250, 12000];
const CUMULATIVE = [0.36, 0.64, 0.84, 0.94, 0.98, 0.995, 0.999, 0.9999, 1.0];

// 密码学安全随机数
const rand = crypto.randomBytes(4).readUInt32BE() / 0xffffffff;
for (let i = 0; i < CUMULATIVE.length; i++) {
  if (rand < CUMULATIVE[i]) return MULTIPLIERS[i];
}
return MULTIPLIERS[0];
```

### 奖池计算

```
奖池 = (入场费 × 3) × multiplier
例：买入 100 chips × 3 × 12000 = 3,600,000 chips 奖池
```

**建议每个 multiplier 独立配置房间**（Room 表 `multiplier` 字段），而不是单一 Blast 模式动态切换。
