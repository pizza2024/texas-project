## 竞品调研（更新）

### GGPoker Spin & Gold 详细规格

| 特征     | 数据                                  |
| -------- | ------------------------------------- |
| 产品名称 | Spin & Gold（原 SPINS）               |
| 人数     | 3 人                                  |
| 报名费   | $1, $3, $7, $15, $25                  |
| 时长     | 约 3-7 分钟                           |
| 最高倍数 | 50,000X（$25 报名 → $1,250,000 奖池） |
| 最低倍数 | 2X                                    |
| 常见倍数 | 2X, 5X（高概率）                      |

**奖池倍数分布（概率递减）：**

| 倍数        | 概率                 |
| ----------- | -------------------- |
| 2X          | 高（最常见）         |
| 5X          | 中                   |
| 10X         | 低                   |
| 25X         | 很低                 |
| 120X        | 极低                 |
| 360X        | 极低                 |
| 1,000X      | jackpot tier         |
| **50,000X** | **头奖（极度稀有）** |

**视觉设计：**

- 霓虹灯风格，金/橙配色
- 3D 旋转奖轮，开奖动画戏剧化
- 大奖时弹出动画 + 动态镜头效果
- 轮盘分段大小不一（高倍数 = 小分段）

**游戏流程：**

1. 选择买入级别，进入 Spin & Gold 队列
2. 3 名玩家自动入座
3. 奖轮旋转，揭示倍数
4. 盲注开始，快节奏 push/fold 策略
5. 淘汰至只剩 1 人
6. 赢家获得乘数奖金
