## 本轮竞品调研 — CoinPoker 深度分析

### CoinPoker 核心特性（基于官方资料）

#### 1. 实时在线人数显示

- **主页 Header 显示**："X players online"，实时更新
- 作为社交证明：高人数 = 游戏不缺对手
- CHIPS 缺失此功能（P1 优先级）

#### 2. 视觉设计语言

- **Dark Theme**：深色背景 + 绿色高亮（扑克桌传统色）
- **Crypto 元素**：BTC/ETH/USDT 图标融入界面
- **高质量视觉资产**：卡牌图像、筹码堆、赛事宣传图
- **CHIPS 当前**：深色主题方向正确，但缺少产品截图

#### 3. 赛事体系（SNG/MTT）

| 赛事类型            | CoinPoker | CHIPS |
| ------------------- | --------- | ----- |
| SNG（Sit & Go）     | ✅        | ❌    |
| MTT（多桌锦标赛）   | ✅        | ❌    |
| Satellite（卫星赛） | ✅        | ❌    |
| Freeroll（免费赛）  | ✅        | ❌    |
| Knockout/Bounty     | ✅        | ❌    |

**CHIPS 仅有 Cash Game，赛事功能完全缺失。**

#### 4. Crypto-Native 功能

| 功能             | CoinPoker | CHIPS                |
| ---------------- | --------- | -------------------- |
| 内置钱包         | ✅        | ⚠️ 基础（USDT 充值） |
