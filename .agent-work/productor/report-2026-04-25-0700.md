## 二、竞品调研（本周聚焦：Rakeback + Club 系统）

### 2.1 GGPoker（行业标杆）

| 维度          | 内容                                                       |
| ------------- | ---------------------------------------------------------- |
| **Rakeback**  | 顶级玩家可达 60%+，通过 GGClub 系统计算                    |
| **Club 系统** | GGClub — 玩家可创建俱乐部，私人 ring game 桌， rake 共享   |
| **独特功能**  | SnapCam（游戏中摄像头）、PartyShares（集体奖励）、智能投币 |
| **VIP 等级**  | 6 级（STAR/DIAMOND/DIAMOND+/CROWN/COBBALT/BLACK）          |
| **生态**      | CHP（GGPoker Token）持有者可享额外 rakeback                |

### 2.2 CoinPoker（区块链标杆）

| 维度         | 内容                                          |
| ------------ | --------------------------------------------- |
| **核心卖点** | USDT 存取，链上透明庄家验证，随机数公开可验证 |
| **Rakeback** | CHP Token 持有者最高 40% rakeback             |
| **Token**    | CHP 可质押，质押量决定 rakeback 比例          |
| **特色**     | 无需 KYC（部分地区），快速存取                |

### 2.3 PokerStars（规模最大）

| 维度         | 内容                                     |
| ------------ | ---------------------------------------- |
| **Rakeback** | VIP 顶级玩家 40-50%（Star Reward 系统）  |
| **特色**     | Spin & Go（随机奖池 Sit & Go）、FastPlay |
| **VIP**      | 5 星等级，按月重置                       |

### 2.4 888poker

| 维度         | 内容                                               |
| ------------ | -------------------------------------------------- |
| **Rakeback** | 最高 36%                                           |
| **特色**     | BLAST 赛制（倒计时 Sit & Go），PokerCraft 统计工具 |
| **Club**     | 888poker Club（入门/高级/精英三级）                |

### 2.5 竞品关键洞察

| 洞察                                                       | 对本项目启示                                      |
| ---------------------------------------------------------- | ------------------------------------------------- |
| GGPoker 的 Club 允许私人局 + rake 分配，是最强社交粘性功能 | **私人邀请码是 Club 核心缺失**                    |
| CoinPoker 的 Token 质押 model 可将 rakeback 最高推到 40%   | 本项目 USDT 充值已实现，可考虑 CHP-like token     |
| PokerStars 5星按月重置 → 高频激励                          | 本项目 GOLD 30% 略低，建议增加 PLATINUM（40-35%） |
| 所有头部平台都有详细战绩统计 / PokerCraft                  | Club 战绩统计需优先实现                           |
