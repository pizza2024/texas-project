## 本轮产品洞察

### 1. UX 体验差距分析

| 功能       | 竞品标杆                        | 本项目            | 差距          |
| ---------- | ------------------------------- | ----------------- | ------------- |
| 入座流程   | Quick Seat (一键入座)           | 手动选择房间→座位 | 中等 — 需优化 |
| 下注交互   | 手势+预置按钮(1/2 pot, 3/4 pot) | 仅按钮            | 小 — 补充预置 |
| 盲注倒计时 | 动态进度条 + 颜色变化           | 未知              | 待前端验证    |
| 爆冷展示   | Bad Beat 动画+奖励广播          | 无                | 大 — Phase 2  |
| 玩家笔记   | 游戏中悬浮笔记                  | 无                | 小 — P3       |

### 2. 变现模式对比

| 模式        | GGPoker             | PokerStars | 本项目     |
| ----------- | ------------------- | ---------- | ---------- |
| 传统抽水    | ✅ 5%               | ✅ 5%      | ✅ 已有    |
| Rakeback    | ✅ Stake Back       | ✅ VIP     | ✅ 刚上线  |
| Battle Pass | ✅ 赛季活动         | ✅         | ❌         |
| 保险产品    | ✅ All-In Insurance | ✅         | ❌         |
| Jackpot SNG | ✅ Mega Spin        | ✅ Blast   | ❌ Phase 2 |
| 皮肤商城    | ✅ Avatar/Theme     | ✅         | ❌         |

**结论**：本项目抽水架构完整，rakeback 已上线，但缺乏高溢价产品（保险、Jackpot）和留存驱动（Battle Pass）。

### 3. SNG 产品机会

竞品 SNG 功能矩阵显示：

- **Hyper-Turbo**（60秒盲注）是标配，非亮点
- **Progressive KO**（渐进淘汰）是差异化重点
- **Jackpot SNG**（随机奖池）是获客利器
- **Fractional Staking**（分数所有权）是小众但高粘性功能

**对 Phase 1 SNG 的启发**：先做标准 6人 SNG + Turbo 变体，验证核心体验后再叠加 KO/Jackpot。
