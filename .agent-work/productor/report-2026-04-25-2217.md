## 产品体验观察

### 项目当前阶段

项目处于 **"赛事生态建设期"** — 核心扑克逻辑稳定，社交功能落地，缺失最大的一块是 **赛事系统（SNG/定时赛）**。

### 关键缺口（与竞品对比）

1. **极速桌品类空白**：SNAP/AoF/SNG Hero 是当前行业增速最快的细分市场，本项目仅有 P2-FASTFOLD Phase 3 调研，无实际实现。GGPoker 的 AoF 和 888poker 的 SNAP 是核心增长引擎。
2. **移动端进度**：GGPoker 8表同开已是行业标杆，本项目 React Native 移动端仍在开发中。
3. **差异化社交**：888poker 的 Webcam 是业界唯一社交创新，本项目聊天已落地但缺乏更深度互动（表情 P2-EMOJI 储备中）。

### Rakeback 对比

| 平台 | Max Rakeback | 机制 |
|------|-------------|------|
| GGPoker | 60%+ | VIP + Happy Hours + Club |
| CoinPoker | 40-50% | CHP Token |
| 888poker | ~36% | 888 Club 忠诚计划 |
| WSOP | ~20% | Caesars Rewards |
| **本项目** | **可变（5 tier）** | Rakeback 系统已实现 |
