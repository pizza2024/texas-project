## 竞品调研（深度更新）

### 1. Natural8（GGPoker 生态）— 80% Rakeback

Natural8 是 GGPoker 亚洲市场品牌，提供高达 **80% rakeback**（"Ocean Rewards"计划），是本项目 5-tier rakeback（最高 50%）的直接竞品。

| 维度          | Natural8/GGPoker          | 本项目              |
| ------------- | ------------------------- | ------------------- |
| Rakeback 上限 | 80%（长期玩家）           | 50%（DIAMOND tier） |
| 赛季形式      | Ocean Rewards（持续积分） | 5 tier 静态等级     |
| 赛事保证金    | $200M/月                  | 待规划              |
| 新人奖励      | 价值 $3,358 的欢迎礼包    | 150% 首充（待实现） |
| 即时客服      | 24/7                      | 待实现              |

**关键洞察**：80% rakeback 需要玩家达到极高 volume，本质上是通过高返现锁定 hardcore 玩家。本项目 5-tier 系统（最高 50%）对 casual-mid tier 玩家已有吸引力，但建议规划 VIP专属更高返水的可能性。

### 2. CoinPoker — CoinRewards 代币经济体系

CoinPoker 使用 **CHP（CoinPoker Token）** 构建独特的经济体系（基于 Octopus Network/NEAR 链）：

| 维度      | CoinPoker                  | 本项目                 |
| --------- | -------------------------- | ---------------------- |
| 忠诚体系  | CHP 代币 + CoinRewards     | Chips + Rakeback 积分  |
| 首充优惠  | 150% Poker Bonus           | 待实现（已有基础）     |
| USDT 充值 | ✅ 原生支持                | ✅ 已实现（HD 钱包）   |
| 提现      | USDT/TRC20                 | USDT（待完成提现流程） |
| 赛事品牌  | World Poker Masters / CGWC | 待 SNG Phase 1         |
| 大使      | Bencb、Patrick Leonard     | 待实现                 |

**CoinRewards 体系**（来自社交媒体追踪，2026年4月）：

- 基于 CHP 代币的玩家 rewards
- 联赛积分系统（CoinRaces）
- 联赛排行榜（CoinMasters）

**关键洞察**：CoinPoker 的代币经济是差异化核心，但 CHP 代币价格波动可能影响玩家体验。本项目纯 Chips 体系更稳定，但可考虑引入"积分 token"作为 rakeback 的承载层。
