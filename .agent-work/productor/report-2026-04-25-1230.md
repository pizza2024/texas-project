## 竞品调研

### 平台对比

| 平台            | 目标用户      | 变现方式           | UX 亮点                                        |
| --------------- | ------------- | ------------------ | ---------------------------------------------- |
| **WSOP**        | 休闲/免费玩家 | 虚拟筹码 + 订阅    | 品牌权威性、Gamification（手镯收藏）、新手引导 |
| **GGPoker**     | 认真玩家      | Rake + 赌场        | 现代 UI、多桌配置、定制化牌桌主题              |
| **888poker**    | 中级玩家      | Rake + 交叉销售    | 简洁界面、易用性、跨平台同步                   |
| **CoinPoker**   | 加密用户      | Token + Rake + NFT | 隐私保护、加密提现、社区治理                   |
| **PlayStation** | 主机玩家      | 一次性购买         | 主机级画面、离线游玩                           |

### 商业模式模式

1. **Freemium/微交易 (WSOP)**: 免费游玩，虚拟货币购买 + 订阅
2. **Rake 抽水 (GGPoker, 888, CoinPoker)**: 每局抽成
3. **加密代币 (CoinPoker)**: CHP 代币经济 + rakeback + 治理
4. **交叉销售 (GGPoker, 888, CoinPoker)**: 扑克 + 赌场/体育博彩
5. **一次性购买 (PlayStation)**: 买断制，无持续变现
