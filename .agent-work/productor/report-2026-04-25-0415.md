## 竞品调研 — Tournament 功能

> 下一阶段：SNG 单桌赛 → Jackpot SNG → MTT 多桌赛

### GGPoker Tournament 产品分析

| 功能                              | 描述                                                  |
| --------------------------------- | ----------------------------------------------------- |
| **STT（Sit & Go）**               | 6-max / 9-max，单桌淘汰赛，参赛费 $1~$1000            |
| **Jackpot Sit & Go**              | 3-max，随机奖池（最小 $2，最大 $100,000），运气成分高 |
| **MTT（Multi-Table Tournament）** | 多桌锦标赛，Day1→Day2→Final，参赛费 $1~$10,000        |
| **BLINDS**                        | 5分钟/10分钟/15分钟可选                               |
| **Late Registration**             | 开赛后 1-12 levels 可补买（re-entry）                 |
| **Bounty（赏金赛）**              | 每淘汰一个玩家获得固定赏金                            |

**GGPoker 独特 UX**：

- `Snapshark` — 实时牌局回顾（每手牌自动录像）
- `NodeJS" style prize distribution` — 锦标赛奖金池分配计算器

### WSOP Tournament 产品分析

| 功能               | 描述                        |
| ------------------ | --------------------------- |
| **WSOP Circuit**   | 线下巡回赛资格赛            |
| **Mega Spin & Go** | 3人桌，最高 $1,000,000 奖池 |
| **Sit & Go**       | 常规 SNG，BOOM! 惊喜奖金    |
| **MTT**            | 每日多场，$0.10~$500 参赛费 |

### CoinPoker Tournament 产品分析

| 功能                     | 描述                                    |
| ------------------------ | --------------------------------------- |
| **CHP Satellites**       | 用 CHP 代币参加卫星赛获取大型锦标赛资格 |
| **Rakeback Race**        | 每周前 100 名额外 5%~30% 额外返水       |
| **Tournament Protector** | 锦标赛"保险"功能 — 淘汰后返还部分参赛费 |
