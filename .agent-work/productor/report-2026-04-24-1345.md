## 竞品调研 — Tournament 赛制专项分析

### 主流平台 Tournament 特性

| 平台 | Tournament 类型 | 特点 | Buy-in Range |
|------|----------------|------|--------------|
| **WSOP** | Main Event, Satellites, Sit-and-Go, Spin-and-Go | 真实 WSOP  bracelet 奖励 | Free - $10,000 |
| **GGPoker** | MTT, Sprint, Windfall, Battle | 短牌(6+), 赏金赛, Flipout | $0.25 - $25,000 |
| **888poker** | SNAP Cash, Tournament, 888 Club | SNAP (快速扑克), 888xl系列 | $0.10 - $1,000 |
| **CoinPoker** | Classic MTT, CNP Tour, Freeroll | 加密货币赛事, CHP 质押 | Free - $500 |
| **PokerStars** | SCOOP, WCOOP, Spin-and-Go | 最大规模系列赛, Spin马拉松 | $0.10 - $25,000 |

### Tournament UX 设计要点

#### 1. 赛程日历 / Lobby
```
推荐布局：
├── 今日赛事（按时间排序）
├── 明日赛事（提前报名）
├── 系列赛进度（当前进度 + 下一阶段）
└── 免费赛专区（每日 Freeroll）
```

#### 2. 报名交互
| 阶段 | UI 状态 |
|------|--------|
| 可报名 | 绿色"立即报名"按钮 |
| 等待开始 | 黄色"排队中 (X/Y)" |
| 已满 | 灰色"已满" |
| 进行中 | 蓝色"进入赛场" |

#### 3. 盲注结构展示
```
♠️ $5+0.50 Turbo
├── 当前: 100/200
├── 剩余时间: 8:32
├── 下一级别: 150/300 (2 min)
└── 级别: 5/12
```

#### 4. 晋级/淘汰动画
- **晋级**: 金色边框 + "晋级!" 弹出动画 + 排名变化
- **淘汰**: 灰度处理 + 手牌翻开动画 + 补偿金展示
- **Final Table**: 特殊舞台效果 + 聚光灯动画

### Tournament 与常规桌游的差异体验

| 维度 | 常规桌游 (Ring Game) | Tournament |
|------|---------------------|------------|
| 目标 | 持续盈利 | 存活 + 排名 |
| 筹码 | 可充值 | 固定参赛金 |
| 时间 | 无限 | 计时盲注 |
| 返水 | 按 rake 计算 | 按报名费计算 |
| 玩家池 | 固定 | 动态增删 |
| 最优策略 | Pot Odds 驱动 | ICM 驱动 |

### 建议 Tournament 功能分级

#### Phase 1 (MVP - 最低复杂度)
```
1. 注册截止时间前可随时注册
2. 固定结构 (不同时段不同盲注表)
3. 单一 stakes (如 $10+1)
4. 前30%名次获奖
5. 无 Rebuy/Addon
```

#### Phase 2 (标准功能)
```
1. 多 stakes 选择
2. 赏金赛 (Knockout Bounty)
3. 计时盲注 (30s/级别)
4. Rebuy 窗口期
5. Late Registration 延长
```

#### Phase 3 (高级)
```
1. Freezeout
2. Satellite 赛事 (晋级资格)
3. SNG 万人赛 (Spin-and-Go 变体)
4. 俱乐部 Private Tournament
5. 系列赛追踪
```
