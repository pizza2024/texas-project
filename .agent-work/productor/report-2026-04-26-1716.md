## 本轮产品调研：Natural8 (GGPoker 生态) UX 分析

### Natural8 产品定位

Natural8 是 GGPoker 旗下品牌，定位为**亚洲市场最大的在线扑克室**。通过本轮浏览观察到的关键产品特性：

#### 1. 新手引导与入职

| 特性 | Natural8 实现 | 本项目现状 | 差距 |
|------|--------------|-----------|------|
| 首次存款奖励 | 200% 红利或免费门票 | ❌ 无 | **大** |
| 免费锦标赛门票 | 每周 $100k 免费锦标赛 | ❌ 无 | **大** |
| Rakeback | 最高 80%（Ocean Rewards） | ✅ 有（5 tier） | 中等 |
| 新手教程 | 入门页面引导 | ❌ 无 | **大** |
| 即时客服 | 24/7 支持 + 即时回复 | ❌ 无 | **大** |

#### 2. 社交与留存功能

| 特性 | Natural8 实现 | 本项目现状 | 差距 |
|------|--------------|-----------|------|
| 表情反应 | 桌内 emoji | ✅ Web 已实现 | 移动端缺失 |
| 好友邀请奖励 | 推荐好友获免费门票 | P1-FIRST-DEPOSIT-spec 已规划 | 中 |
| 社区活动 | Twitch 直播、每月赠品 | ❌ 无 | **大** |
| 返水计划 | 80% rakeback 固定返现 | ✅ 已有 tier 系统 | 接近 |

#### 3. 商业化模式

- **收入来源：** Rake（抽水）+ 锦标赛报名费 + 充值/提现手续费
- **用户激励：** 高 rakeback（80%）作为核心留存手段
- **生态整合：** 直播平台（Twitch）合作、社区运营

### 对本项目的启示

1. **新手引导优先级提升** — 建议参考 Natural8 的「首次存款 200%」设计 P1-FIRST-DEPOSIT-spec
2. **免费锦标赛门票作为拉新** — 可作为 Blast 即时赛事（P1-BLAST-001）的激励机制
3. **Twitch/直播生态** — 长期可考虑 Twitch 集成或赛事直播功能
