### Hand Replay UI 规格（草案）

**参考竞品**: GGPoker Squash, WSOP, Zynga Poker

**核心组件**:

1. **进度条控制** — 拖动查看每轮（Preflop/Flop/Turn/River/Showdown）决策点
2. **牌桌快照** — 每个决策点展示完整牌桌状态（底池、玩家手牌、公共牌）
3. **操作时间线** — 每位玩家操作（FOLD/CALL/RAISE/ALLIN）+ 耗时
4. **胜率面板** — Showdown 时的手牌强度 + 胜负结果
5. **动画过渡** — 轮次切换时的平滑动画

**技术实现方向**:

- 前端新页面 `/hands/[id]/replay` 或弹层面板
- 复用现有 `Hand` + `HandAction` 数据模型
- WebSocket `game-state` 事件已有完整状态快照

## 竞品调研 — CoinPoker（续）

### CoinPoker 反 AI 探测机制

CoinPoker 采用以下策略应对 AI/脚本：

| 机制                                   | 说明                                       | 可借鉴性                                      |
| -------------------------------------- | ------------------------------------------ | --------------------------------------------- |
| **Random Number Generator (RNG) 认证** | 使用区块链 RNG，公开可验证，非确定性洗牌   | ⭐⭐⭐ 核心差异化，可结合区块链筹码系统       |
| **玩家行为分析**                       | 检测异常操作速度（低于人类极限的操作间隔） | ⭐⭐ 实施成本中等，可考虑                     |
| **强制随机等待时间**                   | 每手牌有最小思考时间，防止秒fold/call      | ⭐⭐⭐⭐ 简单有效，与本项目 Pot Odds 机制兼容 |
| **CAPTCHA 挑战**                       | 异常行为触发 CAPTCHA 验证                  | ⭐⭐ 体验影响较大，可选                       |
| **多账号检测**                         | IP/设备指纹关联检测                        | ⭐⭐ 需要额外基础设施                         |

### CoinPoker 产品亮点

| 功能             | 说明                                         |
| ---------------- | -------------------------------------------- |
| **匿名游戏**     | 可选匿名模式（用户名隐藏），降低社交压力     |
| **HUD 自由**     | 不禁止第三方 HUD（差异化策略 vs PokerStars） |
| **加密货币原生** | USDT 充值 + 区块链提现，15分钟到账           |
| **快速桌**       | 30秒/手，适合休闲玩家                        |
