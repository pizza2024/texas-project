## 竞品调研 — 本轮专题：CoinPoker + 888poker UX 对比

### 背景

上轮报告深度分析了 GGPoker。本轮聚焦两个差异化平台：

- **CoinPoker**（2023年上线）：加密原生扑克，USDT 充提，HDL 代理
- **888poker**（2010年至今）：传统扑克老牌，2023年重新设计 UI

---

### CoinPoker UX 特征

#### 1. 加密原生体验
- **USDT 钱包内置**：无需第三方充值，平台内直接兑换筹码
- **TRC20 充值**：链上确认后即时到账，无银行卡限制
- **匿名桌**：支持无用户名桌台（仅显示匿名标识）
- **HDL 代理系统**：去中心化代理，返利 55%+

#### 2. 游戏桌面 UI
- **简洁暗色主题**：黑色背景 + 绿色/金色强调，与本项目视觉方向相似
- **Pot Odds**：Call 按钮旁显示百分比（本项目已实现 ✅）
- **手牌历史**：每局结束后显示详细摊牌数据
- **倒计时**：数字倒计时，无 GGPoker 的彩色弧形进度条

#### 3. 与本项目功能对比

| 功能 | CoinPoker | 本项目 | 差距 |
|------|-----------|--------|------|
| USDT 充提 | ✅ 原生内置 | ✅ 原生内置 | 无 |
| TRC20/ERC20 | ✅ 支持 | ✅ 支持 | 无 |
| Pot Odds HUD | ✅ | ✅ | 无 |
| 轮次指示器 | ✅ 简单文字 | ✅ 小标签 | 视觉相当 |
| All-in 确认 | ⚠️ 弱（无强制弹窗） | ❌ 无 | 中等 |
| Sit-Out 菜单 | ✅ 自动+座位保留 | ⚠️ 需重构 | 中等 |
| 匿名桌 | ✅ | ❌ 无 | 可探索 |
| HDL 代理返利 | ✅ 55%+ | ❌ 无 | P3 |

---

### 888poker UX 特征（2023 redesign）

#### 1. SNAP 闪电扑克
- **即时入场**：点击即入池，无需等待牌局结束
- **自动发牌**：离场后自动回池，无需手动返回
- **适合休闲玩家**：低压力，高频决策

#### 2. 视觉设计
- **现代化暗色 UI**：卡片式布局，圆角按钮
- **人像模式**：6人桌默认显示头像，9人桌俯视
- **色调**：深蓝 + 白色文字，与 PokerStars 接近
- **移动端**：与桌面端一致的 UI 体验

#### 3. 与本项目功能对比

| 功能 | 888poker | 本项目 | 差距 |
|------|----------|--------|------|
| SNAP 闪电扑克 | ✅ 核心模式 | ❌ 无 | 可探索 |
| Pot Odds | ✅ | ✅ | 无 |
| Sit-Out | ✅ 座位保留 | ⚠️ 需重构 | 中等 |
| 9人桌支持 | ✅ | ⚠️ 最多 9 人 | 无（已支持） |
| 表情互动 | ✅ | ❌ 无 | 规划中 |
| 俱乐部系统 | ✅ | ❌ 无 | 规划中 |
