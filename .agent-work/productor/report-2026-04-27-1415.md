## 本轮调研主题：GGPoker SPINS / 即时赛事深度分析

### 什么是 SPINS？

SPINS（Spin & Go）是 GGPoker 最成功的即时赛产品，特点：

| 特征 | 规格 |
|------|------|
| 参赛人数 | 3 人 |
| 赛制 | 随机抽取代币（token），决定奖池倍数 |
| 奖池倍数 | x2 / x5 / x10 / x25 / x100 / x1000 / x10000 |
| 门票价格 | $1 / $3 / $5 / $10 / $25 / $50 / $100 / $250 / $500 / $1000 |
| 单局时长 | 3-5 分钟 |
| 结构 | 极速赛（Fast），无重买 |

### SPINS 奖池分布（官方概率）

| 奖池倍数 | $1 门票 | $3 门票 | $5 门票 | $10 门票 |
|---------|---------|---------|---------|---------|
| x2 | 75% | 72% | 70% | 68% |
| x5 | 20% | 22% | 24% | 25% |
| x10 | 4% | 5% | 5% | 6% |
| x25 | 1% | 0.8% | 0.8% | 0.8% |
| x100 | — | 0.2% | 0.2% | 0.2% |
| x1000 | — | — | — | 超稀有 |

### 关键产品设计洞察

1. **随机性即娱乐** — 核心乐趣不是技术对决，而是"开奖"瞬间的情绪波动
2. **极速闭环** — 3分钟一局，满足移动端碎片时间
3. **社交传播性** — "我刚中了 x100！"天然适合分享传播
4. **低进入门槛** — 小额门票 + 高中奖率，入门无压力
5. **波动性货币** — 对于休闲玩家，x10 的期待价值 > 稳定的小额返还

**为什么不先做 MTT（多桌赛）:**
- MTT 复杂度是 SNG 的 10x+（牌桌分配、座位平衡、淘汰通知、奖金分配树）
- SPINS 用现有 cash table 逻辑可改造，MTT 需要全新架构
- SPINS 是流量产品，MTT 是留存产品——应先拉新
