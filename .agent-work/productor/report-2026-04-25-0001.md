| 选项                         | 描述                                                                                           |
| ---------------------------- | ---------------------------------------------------------------------------------------------- |
| **Option A（WSOP 风格）**    | 确认弹窗：显示"你确定要 All-in？" + 当前手牌 + 底池赔率（Pot Odds）+ 预计赢率（Hand Strength） |
