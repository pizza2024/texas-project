## 竞品分析

### 1. GGPoker（GGNetwork）

**概况**：全球最大线上扑克平台之一，主打亚洲市场。

| 维度 | 详情 |
|------|------|
| **UI/UX** | 现代深色主题，卡片式布局，多桌标签页 |
| **核心功能** | SnapCam 表情、实时聊天表情、HH足球动画、ALL-IN Insurance |
| **Tournament** | SNG（6/9-max）、Jackpot SNG（随机奖池最高 $100K）、MTT（多桌赛、Satellites） |
| **Rake** | 5%~7% 按锅底，抽水较高 |
| **独特功能** | 智能投币（Smart Coin）、皇家同花顺保险（Royal Jackpot）、AHK脚本检测 |
| **移动端** | iOS/Android 原生 App，体验流畅 |
| **商业模式** | 纯游戏收费（抽水）+ 赛事门票 |

**优势**：功能最全面、赛事最丰富、平台最成熟
**劣势**：抽水较高、界面复杂新手门槛高

### 2. WSOP（Caesars Entertainment）

**概况**：WSOP 品牌线上平台，美国市场领先。

| 维度 | 详情 |

| **UI/UX** | 蓝金配色，WSOP 品牌元素，卡通风格 |

| **Rake** | 约 5%~6.5% |

| **移动端** | iOS/Android，App Store 高评分 |
| **商业模式** | 抽水 + 虚拟货币（Play Money）|

**优势**：WSOP 品牌效应、免费赛吸引流量、社交性强
**劣势**：仅限美国市场、功能较 GGPoker 少

### 4. CoinPoker（Crypto Poker）

**概况**：加密货币原生扑克平台，区块链透明性为卖点。

| 维度 | 详情 |
|------|------|
| **UI/UX** | 深色科技风，加密货币元素 |
| **核心功能** | USDT 充值、链上随机数（Provably Fair）、CHP Token |
| **Tournament** | SNG、CHP Satellites、MTT |
| **Rake** | 约 4%~5% |
| **独特功能** | 区块链验证手牌随机性、CHP Token 经济体系、USDT 快速充值 |
| **移动端** | Web-based（H5）+ 原生 App |
| **商业模式** | 抽水 + CHP Token（平台币）+ 游戏内购 |

**优势**：USDT 充值快、链上透明、加密用户友好
**劣势**：波动性（CHP Token 涨跌）、非加密用户门槛高
