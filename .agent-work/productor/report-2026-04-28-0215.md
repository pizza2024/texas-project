## 本轮调研 — P2-VERIFIABLE-SHUFFLE 竞品研究

### CoinPoker RNG/Fairness 机制研究

CoinPoker 官网 `coinpoker.com/rng/` 详细披露了其发牌机制：

#### Fisher-Yates Shuffle + crypto.getRandomValues

CoinPoker 采用 Fisher-Yates shuffle（与本项目相同）：
1. 从52张标准牌开始（工厂顺序 A♥ 起）
2. 从最后一张牌开始，生成 `1~52` 随机位置
3. 将最后一张牌与随机位置牌交换
4. 对倒数第二张、第三张...重复，直到所有牌随机换位
5. **每手牌全新洗牌**，完全独立于前一手

#### 关键结论

| 特性 | CoinPoker | 本项目（table-game-logic.ts:26-33） |
|------|-----------|-------------------------------------|
| 算法 | Fisher-Yates | ✅ Fisher-Yates |
| 随机源 | certified RNG | ✅ `crypto.getRandomValues` |
| 牌靴 | 每手新牌 | ✅ `this.shuffle(this.initDeck())` |
| 第三方认证 | MKT RNG 认证 | ❌ 无外部认证 |
| 公开验证 | 无（闭源） | ❌ 无链上验证 |

**发现：本项目 shuffle 算法与 CoinPoker 一致，均使用 Fisher-Yates + crypto.getRandomValues。**

#### 本项目当前 shuffle 实现（已确认）
```typescript
// table-game-logic.ts:26-33
/** Fisher-Yates shuffle using crypto.getRandomValues for cryptographically secure randomness. */
shuffle(deck: string[]): string[] {
  const d = [...deck];
  const random = new Uint32Array(1);
  for (let i = d.length - 1; i > 0; i--) {
    crypto.getRandomValues(random);
    const j = random[0] % (i + 1);
    [d[i], d[j]] = [d[j], d[i]];
  }
  return d;
}
```

### Fisher-Yates 对齐 CoinPoker — 值得宣传

- 考虑获取第三方 RNG 认证（如 eCOGRA、iTech Labs）
