## 竞品调研：Rakeback/VIP 系统对比

### 竞品功能矩阵

| 平台          | Rakeback 系统      | 可视化             | 兑换机制    | 特色               |
| ------------- | ------------------ | ------------------ | ----------- | ------------------ |
| **GGPoker**   | Fish Buffet 分级   | ✅ 进度条 + 百分比 | ✅ 直充账户 | 丰富奖励，多种等级 |
| **Ignition**  | Casino Credits     | ✅ 余额显示        | ✅ 自动入账 | 无分级，简单直接   |
| **CoinPoker** | CHP 代币           | ✅ 代币余额        | ✅ 实时兑换 | 加密原生，透明     |
| **888poker**  | 888 Club 分级      | ✅ 等级 + 进度     | ✅ 奖金     | 传统扑克品牌       |
| **BetOnline** | 简单返利           | ❌ 无专门UI        | ✅ 自动返利 | 传统平台           |
| **本项目**    | BRONZE/SILVER/GOLD | ❌ 零展示          | ❌ 无兑换   | 后端 MVP 完成      |

### GGPoker Fish Buffet UX 亮点

1. **进度条可视化**: 显示当前等级、距下一等级所需 rake、剩余时间
2. **多任务系统**: 每日/每周任务完成可加速升级
3. **奖励预览**: 可预览各等级对应奖励内容
4. **一键领取**: 简单操作即可兑换

### 竞品启示

| 启示                      | 对本项目建议                                 |
| ------------------------- | -------------------------------------------- |
| Rakeback 必须是"活的"数字 | 前端实时显示 rakebackBalance，增长时动画反馈 |
| 分级制度提升长期留存      | BRONZE/SILVER/GOLD tier 视觉化（卡牌/徽章）  |
| 兑换要简单                | "Claim" 按钮一键完成，无需复杂流程           |
| 透明沟通                  | 显示"你本周 rake $X，已返 $Y"                |
