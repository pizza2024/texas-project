---

### 核心产品特性对比

#### 1. 首充红利（First Deposit Bonus）

| 平台          | 方案                         | 详情                                   |
| ------------- | ---------------------------- | -------------------------------------- |
| **888poker**  | 100% 匹配最高 $888           | 分 8 周释放（每周 12.5%），需 30天完成 |
| **CoinPoker** | 200% 匹配最高 $2000 + 3 USDT | 通过 CHP 代币生态实现                  |
| **WSOP**      | 100% 匹配最高 $1000 金币     | 线下赛资格为激励核心                   |
| **本项目**    | ❌ 空缺                      | P1 待立项                              |

**888poker 设计亮点**：

- 释放周期 **8 周**（而非一次性），显著提升留存
- 匹配比例 **100%**（心理锚点明确：充多少送多少）
- 最高上限 $888（低于 WSOP 的 $1000，但足够吸引）

**本项目建议**：参考 888poker 8 周分期释放机制，而非 CoinPoker 的一次性释放。

#### 2. SNAP Mode（FastFold）

| 特性     | 888poker 实现                               |
| -------- | ------------------------------------------- |
| 核心机制 | 弃牌后 **立即换到新桌**，无需等待当前局结束 |
| 加速倍率 | 平均每小时手牌数提升 3-4 倍                 |
| 适合场景 | 多桌玩家、刷手牌量（rake 贡献）             |

**竞品对比**：

- **BetMGM**: 即时坐下（Instant Seating）类似概念
- **CoinPoker**: 也有快速弃牌模式
- **本项目 P3**: FastFold 在规划中（Snap Mode 名称参考 888poker）

#### 3. 每日登录奖励（Daily Rewards）

| 平台          | 机制                                            |
| ------------- | ----------------------------------------------- |
| 888poker      | 连续登录奖励（7天循环），VIP 等级越高奖励越丰厚 |
| WSOP          | 第1/3/7天不同档位，连续中断重置                 |
| CoinPoker     | CHP 代币每日签到（区块链概念）                  |
| **本项目 P3** | 待开发（每日签到奖励）                          |

#### 4. VIP 忠诚计划

888poker 的 **Poker 888 Club** VIP 系统：

- 基于每周 rake 贡献分级（1-10 级）
- 每周结算，返利比例 **10%-40%**
- 可兑换现金或锦标赛门票
- **本项目 P3 Rakeback** 对应此功能（`User.totalRake` 字段已就位）

---

### 888poker UX 设计亮点

1. **SNAP Mode 一键切换** — 牌桌内提供 SNAP 按钮，随时进出快速模式
2. **多桌排列** — 支持同时开 4+ 桌，标签页切换
3. **手牌历史 HUD** — 每局结束显示赢率统计
4. **7天循环签到** — 连续登录激励比单次奖励更有效
5. **锦标赛天天有** — 降低参与门槛（$1/$3 买入），提升 DAU

| 平台       | 匹配比例 | 上限  | 释放周期  | 门槛 |

| CoinPoker  | 200%     | $2000 | 立即+分期 | 无   |

- 触发门槛：**最低 $10**（避免小额刷 bonus）
