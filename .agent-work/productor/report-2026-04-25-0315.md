### SNG 核心机制分析

```
SNG (Sit-and-Go) = 单桌淘汰赛
格式: 6-max / 9-max / 2-max
开始: 座位满员后立即开始（无需等待定时）
淘汰: 最后一名出局后其余按排名分配奖金
结构: 50%/30%/20% 或 60%/40% 常见

Jackpot SNG 变体（BetOnline / CoinPoker）:
- 3人极速桌
- 奖池随机: 2x ~ 10,000x 买入
- 每小时可打 20+ 桌（极高留存）

SNG Phase 1 MVP 建议（针对本项目）:
- 买入: 固定筹码 (如 1000 chips)
- 座位: 6-max
- 速度: Turbo (5min 盲注升级)
- 奖金分配: 第1名 60%, 第2名 30%, 第3名 10%
- 不需要 Rakeback 改动
```

### Tournament 产品阶段规划（参考竞品）

```
Phase 1: SNG 单桌赛（最简 MVP）
├── 6-max, 固定买入
├── 盲注每 5 分钟升级
├── 奖金分配: 60/30/10
└── 复用现有 room/table 基础设施

Phase 2: Jackpot SNG（高留存驱动）
├── 3人极速
├── 随机奖池: 2x-10000x 买入
└── 高频短局，留存极佳

Phase 3: MTT 多桌赛
├── 多桌并行，座位平衡
├── 牌桌合并 (table redraw)
└── 晋级通道

Phase 4: Satellite 晋级赛
└── 低买入 → 高买入 通道
```
