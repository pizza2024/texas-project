## 竞品调研：CoinPoker 深度分析

### CoinPoker — 加密扑克平台

**平台概况**
| 维度 | 数据 |
|------|------|
| 成立时间 | 2023年 |
| 核心技术 | 区块链 + 加密货币（USDT/ETH/BNB）充值 |
| 特色 | 匿名交易 + 加密货币钱包集成 |
| 玩家规模 | 中型（估算 5-10万 MAU） |

**核心产品特性**
| 功能 | 实现方式 | 对本项目启示 |
|------|----------|--------------|
| USDT 充值 | TRC20/ERC20 直接充值，即时到账 | ✅ 本项目已有 deposit 模块 |
| 加密提现 | 链上提现，15分钟处理 | ✅ 本项目已有 withdraw 模块 |
| 匿名扑克 | 无 KYC 要求 | 差异化竞争点 |
| Jackpot Sit & Go | 3人极速桌，奖池随机倍增 | P3 — Jackpot 模式 |
| 好友对战 | 私人邀请链接 | ✅ Club Phase2 已覆盖 |
| RB (Rakeback) | 周度 rakeback 5-30% | P3 — Rakeback 体系 |
| 锦标赛 | 卫星赛晋级体系 | P3 — Tournament |

**UX 设计亮点**

1. **加密原生体验** — 钱包余额实时显示，链上交易记录内嵌
2. **极速充值** — USDT 充值30秒到账，无等待感
3. **私人桌密码** — 好友间私密对战，Club 私人局参考
4. **快速弃牌 (FastFold)** — 一键进入新桌，保持游戏节奏

**商业模式对比**
| 平台 | 抽水 | Rakeback | 首充红利 |
|------|------|----------|----------|
| 888poker | 3-5% | 5-30% 周度 | 100% 匹配 |
| CoinPoker | 3-5% | 5-30% 周度 | 200% 首充 |
| WSOP | 3-5% | VIP 忠诚计划 | 15金币/仅移动端 |
| **本项目** | 2.5-5% (tier-based) | P3 待实现 | P1 待立项 |
