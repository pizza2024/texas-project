### Rooms 页面 UX 专题（本轮深入）

经过连续多轮观察，Rooms 页面（`apps/web/app/rooms/page.tsx`，1198行）仍有以下产品改进空间：

**体验问题（优先级排序）：**

| #   | 问题                         | 现状                       | 影响                    |
| --- | ---------------------------- | -------------------------- | ----------------------- |
| 1   | 首次加载无骨架屏/加载动画    | 空白或闪白                 | 高 — 感知差             |
| 2   | 房间卡片无 tier 分级视觉标签 | 仅显示盲注数字             | 中 — 玩家难快速识别档次 |
| 3   | 无在线人数显示               | 无任何人数指标             | 高 — 无法判断房间热度   |
| 4   | 无"我的筹码是否足够"突出提示 | Chips vs 准入 散落在各卡片 | 中 — 转化率影响         |
| 5   | 快速匹配 Dialog 无倒计时     | 匹配中一直等待             | 低 — 可加 loading 动效  |

**优化建议：**

1. **加载态**：房间列表获取期间显示骨架卡片（3-4个占位），而非空白
2. **Tier 标签**：每张卡片左上角加彩色徽章（MICRO=灰、LOW=绿、MEDIUM=蓝、HIGH=橙、PREMIUM=紫）
3. **在线人数**：卡片底部加 `👥 X/9` 格式显示当前/最多人数（竞品标配）
4. **准入高亮**：筹码不足的房间卡片显示为灰色 + `🔒 筹码不足`标签，而非让用户一个个点进去看
5. **匹配动画**：Quick Match Dialog 增加转盘/脉冲动画，降低等待焦虑

### 亚洲市场关键洞察

1. **私人局是核心差异**：PPPoker/Pokerrr/PokerBros 的核心竞争力是"和认识的人玩"，而非随机匹配
2. **Club 系统是护城河**：三级 Club（总代理→代理→玩家）体系 + 抽水分成，是亚洲平台的核心商业模式
3. **皮肤/主题付费**：PPPoker 和 PokerBros 提供付费皮肤，是重要收入补充
4. **USDT 充值是空白**：三大亚洲平台均无加密货币充值，这是本项目的绝对差异化优势
5. **Web 端是蓝海**：PPPoker/Pokerrr/PokerBros 均为纯移动端，Web 端是本项目的独有入口

### 定位建议

本项目 vs 亚洲私人局平台：

```
本项目优势：       本项目劣势：
✅ USDT 合规充值   ❌ 无私人局/Club 体系
✅ Web 端入口      ❌ 无表情互动
✅ ELO 智能匹配    ❌ 无皮肤系统
✅ 西方市场定位    ❌ 品牌认知度低
```

**战略建议**：在亚洲市场，本项目应主打"USDT 充值 + Web 端 + ELO 匹配"差异化；在西方市场主打"无 HUD + 匿名桌 + USDT"。
