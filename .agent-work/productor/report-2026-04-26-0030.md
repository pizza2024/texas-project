## 竞品调研更新

### 本轮聚焦：主流平台忠诚度与留存机制深度分析

#### GGPoker FishBuff 计划

GGPoker 的 FishBuff 是行业最复杂的忠诚度系统之一：

| 维度     | 内容                                                       |
| -------- | ---------------------------------------------------------- |
| 返水比例 | 最高可达 60%（Diamond + Royal 等级）                       |
| 返水计算 | 基于净输赢（net loss/gain）的 rake 返还                    |
| 抽签系统 | FishBuff 抽签 — 每周都有机会抽中奖金                       |
| 任务系统 | 完成捕鱼任务获得额外奖励                                   |
| 贵宾升级 | 6级（Copper → Silver → Gold → Platinum → Diamond → Royal） |
| 特色     | 专为"鱼类"玩家设计，降低入坑门槛                           |

**关键洞察：** GGPoker 返水最高 60% 来自 Royal 等级（顶级），普通玩家约 30-40%。任务系统和抽签机制让留存更具游戏化。

#### WSOP PlayWSOP myGP 系统

| 维度         | 内容                                  |
| ------------ | ------------------------------------- |
| myGP         | 基于游戏时间 + 买入金额的积分系统     |
| 积分兑换     | 可兑换锦标赛门票、现金奖励、WSOP 商品 |
| Gameshare    | 与朋友分享手牌/战绩到社交媒体         |
| Caesars 打通 | 线下赌场积分与线上互通                |

**关键洞察：** WSOP 的竞争优势在于线下联动（Caesars Rewards），而非单纯 rakeback。对于纯线上玩家吸引力较弱。

#### CoinPoker CHP 代币经济学

| 维度     | 内容                             |
| -------- | -------------------------------- |
| 代币名称 | CHP (CoinPoker Hub)              |
| 质押收益 | 通过质押 CHP 获得平台收益分成    |
| 充值优惠 | 持有 CHP 可获得额外 rakeback     |
| VIP 分级 | 4级会员体系（非传统 rakeback）   |
| 匿名游戏 | 支持匿名德州扑克桌（差异化功能） |

**关键洞察：** CoinPoker 的代币经济是行业独特，但代币价格波动可能导致玩家收益不稳定。适合加密原生用户。

## 本轮新增洞察

**GGPoker FishBuff 的启示：**

- 行业顶级 rakeback (60%) 是留住高额桌玩家的核心
- 抽签机制比固定返水更有"惊喜感"，提升日活
- 任务系统让玩家有明确目标，而非单纯刷牌

**CoinPoker 匿名游戏桌的启示：**

- 隐私保护是部分玩家的强需求
- 匿名桌可作为差异化功能，低成本实现
