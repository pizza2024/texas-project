### 2. 本项目 TournamentCard 现存问题

#### P1 问题：奖池信息缺失

**问题：** `TournamentCard` 只显示 `prizeDistribution: [number, number, number]`（百分比），没有显示**总奖池金额**。

```tsx
// 当前实现 (TournamentCard.tsx:198-202)
<div>
  <span className="text-gray-400">Prize: </span>
  <span className="text-white font-medium">
    {prize[0]}% / {prize[1]}% / {prize[2]}%
  </span>
</div>
```

**竞品做法：**

- GGPoker: `$10,000 GTD`（Guarantee）+ 总奖池金额
- PokerStars: 显示总奖池 + ITM 人数
- CoinPoker: 显示 USDT 奖池金额

**改进建议：** `ScheduleEntry` 类型需要增加 `prizePool: number`（总奖池 chips/USD），TournamentCard 显示 `prizePool` 而非仅百分比。对于 GTD 赛事，显示"保证奖池"标签。

#### P2 问题：注册人数不透明

**问题：** 只显示 `maxPlayers`，不显示当前已注册人数。

**改进建议：** `ScheduleEntry` 增加 `registeredCount: number`，TournamentCard 显示 `registeredCount/maxPlayers` 格式（如 `42/100`）。
