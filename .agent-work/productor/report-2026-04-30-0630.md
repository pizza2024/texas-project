## 本轮产品调研：锦标赛盲注结构可视化

**调研目标:** 德州扑克竞品如何展示锦标赛盲注结构（Blind Structure），作为 P1-TOURNAMENT-001 的参考

### 竞品分析

| 平台 | 盲注结构展示 | 特色 |
|------|------------|------|
| **WSOP** | ✅ 每个锦标赛详情页显示完整盲注时间轴 | 级别时长 + 升盲比例，Antique/Classic 主题色 |
| **GGPoker** | ✅ 锦标赛列表卡片显示当前/下一级别 | 简洁，紧凑 |
| **888poker** | ✅ 锦标赛桌面显示盲注时钟 | 倒计时风格 |
| **CoinPoker** | 部分展示 | 仅显示当前级别，无结构预览 |
| **当前系统** | ❌ TournamentCard 无盲注结构 | 缺 `smallBlind`/`bigBlind` 字段 |

### WSOP 盲注结构设计

WSOP 锦标赛详情页的盲注结构展示特点：
1. **时间轴可视化**: 横轴为时间，纵轴为盲注级别
2. **每级别信息**: 盲注大小、ante、持续时间
3. **颜色编码**: 不同级别用颜色区分（如 early/late stage）
4. **当前进度指示**: 标注当前处于哪个级别
5. **Break 标注**: 显示休息时间

### 当前系统 TournamentCard 现状

根据 `TournamentCard.tsx` 和 `ScheduleEntry` 接口：
- ✅ `smallBlind` 字段存在于接口
- ✅ `prizeDistribution` 已展示（前三名分配比例）
- ✅ `totalPrize` / `isGuarantee` 已展示
- ❌ 盲注结构（levels + intervals）未展示
- ❌ 无倒计时到下一盲注级别的显示
- ❌ 无升盲时间轴

### 建议实现：Tournament Blind Structure 组件

**优先级:** P2（建议）

**功能点:**
1. `TournamentBlindStructure` 组件 — 展示盲注级别时间轴
2. `nextBlindLevel` 计算 — 根据当前时间推算下一级别
3. `blindLevelClock` — 桌面内实时倒计时
4. TournamentCard 增强 — 显示下一级别信息

**数据需求:**
```typescript
interface BlindLevel {
  level: number;
  smallBlind: number;
  bigBlind: number;
  ante: number;
  durationSeconds: number;
}
```
