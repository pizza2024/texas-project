### GGPoker 关键 UX 差异化功能

1. **智能操作建议** — 根据手牌强度 + pot odds 给出折叠/跟注/加注提示（可关闭）
2. **Snapshotted Cards** — showdown 时手牌以动画方式亮起高光
3. **Player Notes** — 长期记录对手习惯（标注颜色/笔记）
4. **Bad Beat Beat Jackpot** — 特定冤家牌池累积奖金
5. **Card Runbacks** — 河牌发完后再发一次公共牌计算两次结果（娱乐模式）

### 本项目可借鉴的 UI 改进

1. **手牌强度指示器** — 跟注/加注区域显示当前手牌相对强度（基于模拟）
2. **对手历史标注** — 鼠标悬停显示该对手过去10局表现（松/紧/激进）
3. **思考时间图表** — 复盘时叠加各玩家思考时长折线
4. **池大小动态动画** — 有人加注时 pot 数字有弹跳动画
