## 本轮调研 — P1-BLAST-001 规格修正（第2轮）

### 规格参数对齐 GGPoker SPINS

基于275轮调研结论，本轮进一步细化规格修正：

#### 1. 报名费级别修正

| 当前规格                | GGPoker SPINS                          | 建议修正                    |
| ----------------------- | -------------------------------------- | --------------------------- |
| `$1/$3/$7/$20/$50/$100` | `$0.25/$1/$3/$5/$10/$25/$50/$100/$250` | `$1/$3/$5/$10/$25/$50/$100` |

**理由：**

- 移除 `$7`（国内无此档位，主流为 $5）
- 增加 `$5/$25`（GGPoker 主力档位，国内玩家习惯）
- 最高档位暂时用 `$100`，`$250` 可后续按需扩展

#### 2. 倍数上限修正

| 当前规格     | GGPoker SPINS | 建议修正         |
| ------------ | ------------- | ---------------- |
| 最高 50,000X | 最高 12,000X  | 最高 **10,000X** |

**理由：** 50,000X 奖池（如 $100 × 50,000 = $5,000,000）超出常规运营预期，易引发投诉。建议上限 10,000X（$100 × 10,000 = $1,000,000 已属大奖级别）。

#### 3. 超时时间修正

| 当前规格 | GGPoker/PokerStars | 建议修正 |
| -------- | ------------------ | -------- |
| 30秒     | 60秒               | **60秒** |

#### 4. 赛事时长修正

| 当前规格 | 建议修正  | 理由                                               |
| -------- | --------- | -------------------------------------------------- |
| 3分钟    | **4分钟** | 3分钟过短，轮盘动画(3-5s) + 结算 + 出桌至少需4分钟 |

### 修正后规格摘要

```typescript
// P1-BLAST-001 核心参数

const BLAST_MAX_MULTIPLIER = 10_000;
```
