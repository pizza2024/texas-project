## 竞品调研（SNAP Fast-Fold 深度分析）

### 888poker SNAP Poker（Fast-Fold）机制

| 维度 | 详情 |
|------|------|
| 核心机制 | 玩家 fold 后立即换到另一桌，无需等待当前局结束 |
| 目标用户 | 追求高频 action 的碎片时间玩家 |
| 增长数据 | 888poker SNAP 活跃用户增长 18%，行业 Quick Game 最高增速 |
| 牌桌规模 | 通常 6-max（也有 4-max/9-max 变体） |
| 盲注结构 | 与常规 Cash Game 相同，无特殊盲注 |
| 匹配逻辑 | 按 ELO/ stakes 自动匹配，跳过当前桌玩家 |
| 收入模式 | Rake（与常规 Cash Game 一致） |

**888poker SNAP 对本项目的借鉴：**
- Fast-Fold 核心价值：消除等待时间，提升局/小时（hands per hour）
- UI 提示：「你的手牌已 Fold，正在为你寻找新桌…」
- 技术难点：TableManager 需要维护多个可用牌桌的 pool，而非单一 room
- 匹配策略：ELO + 盲注级别双重匹配

### SNAP Fast-Fold vs 其他 Quick Game 形式

| 形式 | 代表产品 | 特点 |
|------|---------|------|
| SNAP Fast-Fold | 888poker | fold 后立即换桌，最高 H/H |
| Beat the Clock | 888poker | 倒计时开赛，微型 SNG |
| Zone Poker | PartyPoker | 每局后换位置但不换桌 |
| Fast Forward | partypoker | 换桌但保留座位 |

**结论**：SNAP Fast-Fold 是 hands-per-hour 最高的格式，适合碎片时间玩家。本项目 P2-FASTFOLD-001 规划为 Phase 3，在 SNG/MTT 之后实施。
