### CoinPoker Web3 功能深度分析

CoinPoker 是目前最大的加密货币德州扑克平台，其 USDT 金融体系设计如下：

#### 充值流程对比

| 步骤     | 本项目                    | CoinPoker                              |
| -------- | ------------------------- | -------------------------------------- |
| 生成地址 | HD 钱包派生专属地址       | 独有 CHP 钱包系统，自动生成 TRC20 地址 |
| 确认等待 | 轮询扫块，依赖 ScanCursor | 实时 WebSocket 推送，链上确认即到账    |
| 到账反馈 | 玩家需刷新页面            | 推送 + 弹窗 + 余额自动更新             |
| 交易记录 | 数据库记录                | 链上哈希 + 浏览器链接                  |
| 最小充值 | 无明确提示                | $10 等值 USDT 门槛                     |

#### 提现流程对比

| 步骤     | 本项目                    | CoinPoker                           |
| -------- | ------------------------- | ----------------------------------- |
| 提现门槛 | 无明确展示                | $20 等值 USDT                       |
| 处理时间 | BullMQ 后台处理           | 即时处理（站内）/ 15-30分钟（链上） |
| 手续费   | 未明确展示                | 固定 1 USDT 或 1%                   |
| 进度追踪 | 状态：PENDING → CONFIRMED | 状态 + 链上交易哈希 + 浏览器链接    |
| 实时通知 | 无                        | WebSocket 实时推送                  |

#### 链上透明化功能

CoinPoker 公开的功能：

1. **CHP（CoinPoker Currency）** — 平台内加密货币，可直接用于游戏
2. **链上随机数（CSRNG）** — 公开可验证的随机数生成，扑克牌发牌透明
3. **区块历史** — 所有游戏操作记录在区块链上，可查询
4. **冷钱包公示** — 大额资产冷存储，公开钱包地址
5. **审计报告** — 第三方审计财务报表

本项目缺失的关键透明化功能：

- **Pr Cooling（冷静期）** — 大额提现有 24-48h 延迟，防止洗钱同时公示给玩家
- **KYC 简化** — 小额免 KYC，大额需 KYC，公开政策
- **Proof of Reserve** — 储备金证明，玩家可验证
