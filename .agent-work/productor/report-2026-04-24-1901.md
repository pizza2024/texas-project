#### 关键留存机制详解

##### 1. WSOP（品牌授权模式）

**新人引导**:
- 首局强制教程（skip-able），覆盖 fold/call/raise/bet
- 引导期间 AI 对手，失败不扣筹码
- 引导完成后发送 50K 起始筹码

**留存机制**:
- 每日签到：连续7天阶梯奖励，第7天给大奖（100K+）
- 每周 Ring Game 任务（完成 X 手牌）
- Club 系统（类似本项目 ClubGG 调研）
- WSOP 名额争夺作为核心钩子（年度活动）

**首充策略**:
- 首次购买额外 +100% 加赠（行业最高之一）
- 最低 $4.99 起充，降低门槛

##### 2. 888poker（rakeback 导向）

**差异化**——Rakeback 透明:
- 9级 VIP 制度，每级对应固定 rakeback 比例
- Level 1: ~10% → Level 9: ~50%
- 玩家可计算预期收益，降低流失

**PokerCam**: 允许玩家上传真实头像，增强社交粘性

**欢迎赛**: 新用户自动注册免费锦标赛，赢了有真实奖金

##### 3. Zynga Poker（社交+习惯驱动）

**连胜奖励（Hot Streak）**:
- 3连胜：额外 50K 筹码
- 5连胜：额外 200K 筹码
- 7连胜：额外 500K + 专属徽章

**每日奖励轮盘**: 登录即转，最低保底 10K，上限 jackpot

**礼物系统**: 好友之间可赠送筹码，增强社交绑定

**Streak 丢失机制**: 连续 48 小时不登录，连胜计数归零 ——制造紧迫感

##### 4. CoinPoker（Crypto 原生）

**Crypto 入场优势**:
- USDT 充值即时到账，无 KYC
- 充值 bonus：首充 100% match（上限 $500）
- 持有 CHP（平台 token）享受额外 rakeback

**Rakeback via CHP**: 每周按持有量分配平台收益，持有即有收益

**缺点**: 社交功能薄弱，留存依赖收益而非情感

##### 5. PlayWSOP（Facebook 社交）

**社交导入**: Facebook 好友列表直接导入
**每日任务**: "赢得 3 手牌" / "连续 5 手不下桌" 等
**成就徽章**: 几百种成就，满足收集欲

## 竞品调研小结

**留存系统核心洞察**:
- 新人教程是所有主流扑克 app 标配，本项目缺失
- 连胜奖励（Zynga）是制造"上头"的经典设计
- 首充 +100% 是行业标准，本项目已有但需验证转化率
- 每日任务（PlayWSOP/WSOP）引导玩家目标，减少无聊流失
- Rakeback（888poker/CoinPoker）是最直接的留存钩子

**本项目差异化**: 无 KYC + Web3 充值 + Club 系统 = 隐私优先 + 加密原生 + 社交绑定，三位一体。与主流错位竞争。
