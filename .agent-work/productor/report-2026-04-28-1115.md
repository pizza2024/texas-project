## 本轮调研专题：德州扑克 NFT/Token 化趋势 + Play-to-Own 经济模型

### 行业背景

2025-2026 年，crypto-native 扑克平台（CoinPoker、Sportsbet.io、True Poker）正在探索 **Play-to-Own (P2O)** 经济模型，与传统 F2P 平台的差异日益显著。

### Play-to-Own vs Play-to-Earn 差异

| 维度 | Play-to-Earn (P2E) | Play-to-Own (P2O) |
|------|-------------------|-------------------|
| 核心逻辑 | 玩游戏赚钱（可持续性差） | 通过游戏行为积累资产所有权 |
| Token 用途 | 主要为激励，无实际治理 | 治理投票 + 功能性用例（质押/手续费折扣） |
| 经济稳定性 | 通胀压力大，Token 价格易崩 | 需求驱动，Token 价值与平台活跃度绑定 |
| 代表平台 | Axie Infinity (已衰落) | CoinPoker, STEPN |
| 玩家感知 | "打工"感强 | "股东"感强 |

### CoinPoker 匿名模式深度解析

**为什么匿名桌受欢迎：**
1. **Anti-HUD 保护** — 阻止对手使用追踪软件获取数据
2. **公平竞争** — 新手不被数据歧视
3. **隐私保护** — 无需暴露真实战绩

**本项目 Anonymous Mode 规格（已有基础）：**
- `Room.isAnonymous: boolean` — 房间标记
- 匿名桌：隐藏所有玩家的历史战绩、ELO、真实姓名
- UI 需显示：座位号 + 筹码量（无昵称 or 固定匿名昵称）

**实施缺口：**
- Backend: ✅ `isAnonymous` 字段 + 房间过滤
- Frontend: ❌ 匿名桌特殊 UI（无战绩标签、无玩家卡片详情）
- WebSocket: ❌ 匿名模式下 `game-state` 事件需过滤敏感字段
