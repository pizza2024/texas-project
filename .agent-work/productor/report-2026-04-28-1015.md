### 社交系统优先级建议

基于竞品分析，社交功能对留存有显著正向影响：

**P0（立即实施）：**
1. **好友申请通知** — 当前 Friend 模块已有，但无实时通知气泡
2. **牌桌内快速添加好友** — 点击对手头像 → "添加好友"按钮

**P1（下个 sprint）：**
3. **好友观战列表** — 好友进入牌桌时显示"正在游戏中"可跳转观战
4. **牌桌表情同步** — GGPoker 风格的击掌/挥手/祝贺动画（已有 emoji 基础）
5. **玩家称号/等级** — 基于手牌数/胜率/参赛数的可见称号（初学者/常客/鲨鱼）

**P2（后续迭代）：**
6. **私人房间邀请码** — 生成邀请链接，好友可一键加入
7. **俱乐部/公会系统** — 创建战队，举办内部赛（参考 ACR 的线上战队联赛）

参考 Bovada 7天循环签到，设计本项目方案：

```
第1天：500 chips
```
