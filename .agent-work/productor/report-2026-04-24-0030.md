## 本轮竞品调研 — Tournament Structure 深度分析

### 竞品赛事系统对比

| 功能                          | CHIPS | WSOP | 888poker | CoinPoker | Bet365 Poker |
| ----------------------------- | ----- | ---- | -------- | --------- | ------------ |
| SNG（Sit & Go）               | ❌    | ✅   | ✅       | ✅        | ✅           |
| MTT（Multi-Table Tournament） | ❌    | ✅   | ✅       | ❌        | ✅           |

| 实时筹码排行榜                | ❌    | ✅   | ✅       | ✅        | ✅           |
| Tournament lobby              | ❌    | ✅   | ✅       | ⚠️        | ✅           |

### CHIPS 赛事功能差距

CHIPS 当前仅有 **Cash Game**（常规桌），完全没有赛事体系。这是一个重大功能缺口：

1. **用户留存角度**：Tournament 是扑克应用的核心留存机制，WSOP 85% 的 DAU 来源于赛事
2. **收入角度**：Buy-in + Rebuy + Add-on 是主要收入模型
3. **差异化角度**：CHIPS 的 Web3/透明特性在 Tournament 中同样适用（智能合约分发奖金）

**建议**：

- P2 优先级：实现简单 SNG（6-max, single table）
- P3 优先级：实现 MTT 基本框架

| ------------------ | ------- | ---- | -------- | --------- |

| 玩家头像           | ⚠️ 基础 | ✅   | ✅       | ✅        |
| 聊天/快捷短语      | ⚠️ 基础 | ✅   | ✅       | ❌        |
