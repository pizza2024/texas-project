| 项目 | 内容 |
|------|------|
| **ID** | P2-BREAKING-001 |
| **位置** | `club.service.ts:698` → `club.controller.ts:255` |
| **问题** | `joinByCode` 返回类型从 `Promise<{ clubId, clubName, role }>` 变更为 `Promise<ClubMember>` |
| **影响** | ClubMember 包含 id（内部UUID）、clubId、userId、role、joinedAt |

| **建议** | 确认前端 joinByCode 调用处是否需要适配新返回类型 |
