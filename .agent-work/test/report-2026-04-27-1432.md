## 2. Web TypeScript 编译错误 ⚠️

与上轮相同 3 个错误，未修复：

```
components/blast/MatchingOverlay.tsx(3,41): error TS2307: Cannot find module 'framer-motion'
components/blast/SpinWheel.tsx(4,38): error TS2307: Cannot find module 'framer-motion'
hooks/useBlastSocket.ts(2,10): error TS2459: Module '"@texas/shared/socket"' declares 'socket' locally, but it is not exported.
```

**根因分析：**

| 错误 | 原因 | 修复方案 |
|------|------|---------|
| framer-motion 缺失 ×2 | `apps/web/package.json` 中无 `framer-motion` 依赖 | `pnpm add framer-motion` |
| socket 未导出 | `packages/shared/src/socket.ts` 导出的是 `getSocket()` 函数而非 `socket` 实例 | `useBlastSocket.ts` 改用 `getSocket()` |

**packages/shared/src/socket.ts 导出确认：**
```typescript
export function getSocket(...): Socket { ... }  // ✅ 存在
// socket 实例未直接导出
```

**useBlastSocket.ts 当前错误导入：**
```typescript
import { socket } from "@texas/shared/socket";  // ❌ socket 不存在
```

**正确导入应为：**
```typescript
import { getSocket } from "@texas/shared/socket";
const socket = getSocket();  // ✅
```
