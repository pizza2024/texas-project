### ⚠️ P2-NOTIFY-001-001 — NotificationGateway room join 缺失（发现）

**文件:** `apps/backend/src/notification/notification.gateway.ts:76`

```typescript
emitToUser(userId: string, notification: unknown) {
  this.server.to(`user:${userId}`).emit('notification', notification);
}
```

**问题:** `emitToUser` 使用 `this.server.to('user:${userId}')` 发送通知，但 `handleConnection` 中从未调用 `client.join('user:${userId}')`。结果：用户连接后没有加入个人 room，push 通知无法送达。

**对比:** `WebSocketManager.emitToUser`（正确模式）通过遍历所有 socket 的 `socket.data.user?.sub` 找到目标用户并直接发送。

**建议:** 在 `handleConnection` 末尾添加：
```typescript
await client.join(`user:${payload.userId}`);
```

**影响范围:** 所有实时通知（好友上线等）无法通过 `NotificationGateway` 送达。
