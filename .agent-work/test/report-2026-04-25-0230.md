### P2 — TypeScript 编译错误（阻塞构建）

#### P2-TS-002 — app.gateway.ts:544 action 类型不兼容

**文件**: `apps/backend/src/websocket/app.gateway.ts:544`

```typescript
@SubscribeMessage('player_action')
async handlePlayerAction(
  @ConnectedSocket() client: Socket,
  @MessageBody()
  data: { action: unknown; amount?: unknown; roomId?: unknown },  // ← 这里是 unknown
) {
  return handlePlayerAction(this, client, data);  // ← 传入的 data 是 unknown
}
```

**错误**:
```
src/websocket/app.gateway.ts:544:45 - error TS2345: Argument of type
'{ action: unknown; amount?: unknown; roomId?: unknown; }' is not assignable
to parameter of type '{ action: "fold" | "call" | "raise" | "check" |
"straddle" | "sit-out" | "allin"; amount?: number | undefined; roomId?: string | undefined; }'.
```

**根本原因**：`@MessageBody() data` 默认类型为 `unknown`，但 `handlePlayerAction` 期望具体类型联合。Gateway 层数据验证应在 handler 内部进行，gateway 层应做类型守卫或类型断言。

**建议修复**: 在 gateway 层添加运行时验证或类型断言，而非让 `unknown` 流入 handler。
