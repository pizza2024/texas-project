## CodeReview 发现

### 🔴 P1 — 新发现：Private Blast Lobby 无法被加入

| ID                    | 问题                                                                                        | 文件                            | 优先级 | 状态       |
| --------------------- | ------------------------------------------------------------------------------------------- | ------------------------------- | ------ | ---------- |
| P1-BLAST-PRIVATE-JOIN | `joinBlastLobby` 端点缺少 `password` 参数，但 `createBlastLobby` 支持创建密码保护的私人房间 | tournament.controller.ts:80-100 | 🔴 P1  | **待修复** |

**问题分析：**

- `POST /rooms/blast` 接受可选 `password` 字段创建私人房间
- `POST /rooms/blast/:id/join` **不接受** `password` 参数
- 结果：私人 Blast 房间创建后，**无法被其他玩家加入**

**修复建议：**

```typescript
// tournament.controller.ts — joinBlastLobby 补充 password 参数
async joinBlastLobby(
  @Param('id') lobbyId: string,
  @Body('password') password: string | undefined,  // ← 添加
  @Req() req: AuthenticatedRequest,
)
```
