| ID | 严重度 | 问题 | 位置 |

| P2-WEB-JEST-001 | Medium | `jest.mock('@texas/shared')` 解析失败 — moduleNameMapper 缺少映射 | `apps/web/jest.config.js` |

**问题分析:**
```
Cannot find module '@texas/shared' from 'jest.setup.ts'
```
- `jest.setup.ts` 中 `jest.mock('@texas/shared', ...)` 失败
- tsconfig.json 有 path mapping 但 jest.config.js 的 `moduleNameMapper` 没有对应映射
- 当前 `moduleNameMapper` 只映射 `@/(.*)$`，缺失 `@texas/shared` → `../../packages/shared/src`

**影响:** 2 个 spec 文件无法运行（socket-session-provider.spec.tsx, 另一个）
