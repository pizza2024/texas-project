## 🔴 P2-NEW-001: app.gateway.ts 生产构建仍阻塞

**严重程度**: P2（阻塞生产构建）

**问题描述**:
Coding 第88轮对 `app.gateway.ts` 的类型修复不完整。测试通过但构建失败：

```
src/websocket/app.gateway.ts:544:45 - error TS2345:
  Argument of type '{ action: "fold" | "call" | "raise" | "check" | "straddle" | "sit-out" | "allin";
    amount?: unknown; roomId?: unknown; }'
  is not assignable to parameter of type '{ action: "fold" | "call" | "raise" | "check" | "straddle" | "sit-out" | "allin";
    amount?: number | undefined; roomId?: string | undefined; }'.
  Types of property 'amount' are incompatible.
```

**根因分析**:

| 位置                           | 当前类型                                | 期望类型                       |
| ------------------------------ | --------------------------------------- | ------------------------------ |
| `app.gateway.ts:542` (调用方)  | `amount?: unknown`                      | `amount?: number \| undefined` |
| `game.handler.ts:241` (接收方) | `PlayerActionInput` (`amount?: number`) | —                              |

`@MessageBody()` 装饰的 `data` 对象中 `amount` 被推断为 `unknown`，但 `handlePlayerAction` 期望 `PlayerActionInput`，其 `amount` 字段类型为 `number | undefined`。

**修复方案**:

```typescript
// apps/backend/src/websocket/app.gateway.ts:541-542
@MessageBody()
data: { action: 'check' | 'fold' | 'call' | 'raise' | 'allin' | 'straddle' | 'sit-out'; amount?: number; roomId?: string },
```

将 `amount?: unknown` 改为 `amount?: number`，`roomId?: unknown` 改为 `roomId?: string`。
