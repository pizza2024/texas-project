#### ✅ `user.controller.ts` / `user.service.ts` — 实时在线人数 API 实现

```typescript
// user.service.ts
async getOnlinePlayerCount(): Promise<{ online: number; playing: number; total: number }> {
  const [online, playing] = Promise.all([
    this.prisma.user.count({ where: { status: 'ONLINE' } }),
    this.prisma.user.count({ where: { status: 'PLAYING' } }),
  ]);
  return { online, playing, total: online + playing };
}

// user.controller.ts
@Get('stats/online')
async getOnlinePlayerStats() { return this.userService.getOnlinePlayerCount(); }
```

| 检查项   | 结果         | 说明                                                 |
| -------- | ------------ | ---------------------------------------------------- |
| SQL 注入 | ✅ 安全      | Prisma 参数化查询                                    |
| 权限控制 | ⚠️ 需确认    | 建议加 `@UseGuards(AuthGuard('jwt'))` 或明确公开意图 |
| 索引     | ✅ 有索引    | `User.status` 已有索引（见 schema）                  |
| 性能     | ✅ 2次 count | Promise.all 并行，无 N+1                             |

**小风险**: `/user/stats/online` 无鉴权，暴露在线人数。建议确认是否为产品预期（竞品均公开此数据）。
