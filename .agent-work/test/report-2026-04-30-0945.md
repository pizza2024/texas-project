| 组件 | 状态 | 说明 |

| `initRedis()` | ✅ 正确 | `lazyConnect=true`，`enableOfflineQueue=false`，3s timeout，non-fatal |
| `redisSub.subscribe` | ✅ 正确 | 订阅 `ws:broadcast` channel，`on('message')` 转发至本地 |
| `redisPub.publish` | ✅ 正确 | 仅在 `status === 'ready'` 时发布，本地投递完成后异步执行 |
| `registerSocket` | ✅ 正确 | `handleConnection` 中调用，构建 localSockets 索引 |
| `unregisterSocket` | ✅ 正确 | `handleDisconnect` 中调用，清理 socketId |

| `initRedis()` 调用位置 | ✅ 正确 | `AppGateway.onModuleInit()` 中调用 |
| 单实例 fallback | ✅ 正确 | Redis 不可用时降级为纯本地投递，完全向后兼容 |
