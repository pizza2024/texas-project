### 专项检查：P1-WALLET-001（exchangeBalanceToChips 非原子）

**文件:** `wallet.service.ts:338-380`

三处独立 DB 调用，仍无 `$transaction` 包裹：

```ts
await this.prisma.wallet.upsert({ ... });     // ✅ 成功
await this.prisma.user.update({ ... });        // ❌ 失败 → wallet 更新，user 未更新
await this.prisma.transaction.create({ ... }); // ❌ 失败 → 静默丢失
```

**风险:** 用户充值时，wallet 扣款成功但 chips 未到账，或 chips 到账但 USDT 未扣款。

**对比:** `exchangeChipsToBalance` (line 276) 正确使用了 `$transaction`。

**建议:** 立即用 `$transaction` 包裹，与 `exchangeChipsToBalance` 模式一致。
