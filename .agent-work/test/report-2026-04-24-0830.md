## 本轮新发现问题

### W-013: Math.random() 在生产代码中使用 [P2]

**文件**: `apps/web/app/rooms/page.tsx:233`

```typescript
name: `Table ${Math.floor(Math.random() * 1000)}`,
```

**问题**: `Math.random()` 不是加密安全的随机数生成器。虽然此处于 UI 层用于创建房间的默认名称（非关键随机性），但为保持代码一致性（后端 Bot/Matchmaking 已改用 `crypto.randomInt()`），建议统一替换。

**影响**: 低 — 仅用于 UI 默认名称，无安全敏感度

**建议**: 使用 `crypto.randomInt()` 或 UUID 生成默认房间名

**修复示例**:

```typescript
import { randomInt } from 'crypto';
name: `Table ${randomInt(1000)}`,
```
