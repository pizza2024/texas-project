## 🔴 P0 — 新发现：TypeScript 编译错误（commit b827c8b）

| ID           | 问题                                                                            | 文件                                                                   | 优先级 | 状态       |
| ------------ | ------------------------------------------------------------------------------- | ---------------------------------------------------------------------- | ------ | ---------- |
| P0-BUILD-001 | `getRoomStatus()` 返回值包含 `gameState` 字段，但 `RoomStatus` 接口未定义该字段 | table-manager.service.ts:619 + packages/shared/src/types/game.ts:44-49 | 🔴 P0  | **待修复** |

**错误信息：**

```
src/table-engine/table-manager.service.ts:619:7 - error TS2353:
Object literal may only specify known properties, and 'gameState'
does not exist in type '{ roomId: string; currentPlayers: number;
maxPlayers: number; isFull: boolean; }'.
```

**根因分析：**

- `b827c8b` 在 `table-manager.service.ts:getRoomStatus()` 的返回值中增加了 `gameState` 字段
- 但 `packages/shared/src/types/game.ts:RoomStatus` 接口（line 44-49）**未同步更新**
- `RoomStatus` 接口仅有：`roomId, currentPlayers, maxPlayers, isFull`

**修复建议：**

```typescript
// packages/shared/src/types/game.ts — RoomStatus 接口补充 gameState 字段
export interface RoomStatus {
  roomId: string;
  currentPlayers: number;
  maxPlayers: number;
  isFull: boolean;
  gameState?: "waiting" | "playing"; // ← 添加（可选，保持向后兼容）
}
```
