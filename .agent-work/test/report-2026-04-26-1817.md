### JWT 存储：P2-JWT-LOCALSTORAGE 仍是 XSS 向量

**状态:** P2，建议升 P1

`packages/shared/src/auth.ts` 定义了 `StorageAdapter` 接口，web 端使用 `localStorage`。`login/page.tsx:28` 确认 `setToken` → `localStorage`。

**风险:** 任何 XSS 漏洞可直接读取 JWT token，导致账户接管。

**建议:** 迁移到 HttpOnly cookie（`auth.service.ts` + API 层），避免 token 被页面 JS 访问。
