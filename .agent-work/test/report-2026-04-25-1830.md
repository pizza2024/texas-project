| P1-NEW-003 | TypeScript 构建失败: `pa.totalBet` 属性不存在 | `hand-history.service.ts:135` | 🔴 阻塞 |

**问题详情:**

```
src/table-engine/hand-history.service.ts:135:58 - error TS2339: Property 'totalBet' does not exist on type '{ bet: number; fold: boolean; allIn: boolean; }'.
```

**根因分析:**

`playerActions` Map 的 value 类型为 `{ bet: number; fold: boolean; allIn: boolean }`（第112-115行），但第135行访问 `pa.totalBet`。该属性从未被写入，仅在 `ParsedPlayer` 类型（第78-82行）存在。

正确字段应为 `pa.bet`。这与已修复的 P1-BACKEND-001（`netProfit` 用 `pa.bet` 而非 `pa.totalBet`）模式相同——说明 hand-history.service.ts 中存在同类遗留 bug。

**影响:**
- 阻塞 `npm run build`（后端编译失败）
- 手牌历史/复盘页面数据可能不正确（净收益计算用错字段）

**建议修复:**
```typescript
// hand-history.service.ts:135
const netProfit = (settlement?.amount ?? 0) - pa.bet;  // pa.totalBet → pa.bet
```
