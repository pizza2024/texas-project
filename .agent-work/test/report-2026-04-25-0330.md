## P1-002 Jest Worker 泄漏 — 根因分析建议

```
A worker process has failed to exit gracefully and has been force exited.
Active timers can also cause this, ensure that .unref() was called on them.
```

**可能位置**：

1. `app.gateway.ts` — `afterInit()` 中的 Socket.io 定时器未在 `onModuleDestroy` 清理
2. `table-manager.service.ts` — `Set` 清理回调中 `setTimeout` 未 `unref()`
3. `deposit.service.ts` — 轮询 `setInterval` 未 `unref()`

**验证步骤**：
```bash
cd apps/backend
npm test -- --detectOpenHandles --forceExit 2>&1 | grep -A5 "detectOpenHandles"
```
