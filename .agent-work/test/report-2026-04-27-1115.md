### 1.2 P2-NEW-014 `broadcastTableState` 异步广播 — 🔍 审查中

**文件:** `apps/backend/src/websocket/broadcast.service.ts:18-63`

当前实现：

```ts
for (const [userId, userSockets] of Array.from(socketsByUser.entries())) {
  const view = userId ? table.getMaskedView(userId) : table.getMaskedView("");
  for (const socket of userSockets) {
    socket.emit("room_update", view); // 顺序同步 emit
  }
}
```

**分析:**

- 每次 emit 是同步的，房间玩家越多延迟越高
- `getMaskedView()` 每个 userId 只调用一次（已优化）
- 无并发 bug，但可以优化为 `Promise.all` 并行 emit
- **建议:** 降级为 P3 优化项，不阻塞功能

**风险:** Medium-Low（不影响正确性，仅性能）
