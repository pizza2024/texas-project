```ts
shuffle(deck: string[]): string[] {
  const random = new Uint32Array(1);
  for (let i = d.length - 1; i > 0; i--) {
    crypto.getRandomValues(random);
    const j = random[0] % (i + 1);
    [d[i], d[j]] = [d[j], d[i]];
  }
  return d;
}
```

- **密码学安全随机数**（Web Crypto API）✅ — 优于 `Math.random()`
