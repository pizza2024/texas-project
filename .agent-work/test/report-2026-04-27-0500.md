### P1-BACKEND-005 — `TournamentType` 未导入导致编译失败 🔴

**文件**: `apps/backend/src/tournament/tournament.service.ts:58`

**问题**: `createSngConfig()` 方法使用了 `TournamentType.SNG`，但文件中**未导入** `TournamentType`。

```ts
// tournament.service.ts 第14-23行 — 导入了这些:
import {
  SngConfig, BlindLevel, createDefaultBlindSchedule,
  SNG_MAX_PLAYERS, SNG_INITIAL_CHIPS, SNG_BLIND_DURATION_SECONDS,
  PrizeDistributionResponse, PrizePosition,
} from '@texas/shared/types/tournament';

// 但 TournamentType 也需要导入，文件中却缺失
return {
  type: TournamentType.SNG,  // ← TS2304: Cannot find name 'TournamentType'
  ...
};
```

**影响**: `npx tsc --noEmit` 失败，阻塞后端编译和所有测试运行。

**修复**: 添加 `TournamentType` 到 import 列表:
```ts
import {
  TournamentType,  // ← 添加这一行
  SngConfig, BlindLevel, ...
} from '@texas/shared/types/tournament';
```

**优先级**: P1（阻塞编译）
