return {
  availableChips: available,

**缺陷:** `networkFeeUsdt` 硬编码为 `1` — 无法反映实际 gas fee 波动。建议后续改为动态计算（通过 TRON API 获取实际转账 gas）。但竞品均无动态 fee，可接受当前实现。
