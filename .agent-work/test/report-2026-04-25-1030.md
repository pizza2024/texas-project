### P2-NEW-006 详情

**文件**: `apps/web/app/club/join/page.tsx:130`
**问题**: `Button` 组件（`components/ui/button.tsx`）未定义 `loading` prop，但 club/join 页面传入了 `loading={joining}`。

**当前 ButtonProps 接口**:
```ts
export interface ButtonProps
  extends React.ButtonHTMLAttributes<HTMLButtonElement> {
  variant?: 'default' | 'outline' | 'ghost' | 'destructive'
  size?: 'default' | 'sm' | 'lg' | 'icon'
}
```

**调用处**:
```tsx
<Button
  onClick={handleJoin}
  loading={joining}    // ← TS2322: 'loading' does not exist
  ...
/>
```

**修复建议**（二选一）:
1. **推荐**: 在 `ButtonProps` 添加 `loading?: boolean`，渲染时叠加 `disabled` 样式
2. **临时**: 移除 `loading={joining}`，改用 `disabled={joining}`（当前已正确处理 disabled 状态）
