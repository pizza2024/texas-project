#### P2-CODE-033 — Blast 清理间隔可能过长

**位置:** `apps/backend/src/tournament/blast.service.ts:95`

```typescript
this.cleanupInterval = setInterval(() => {
  this.sweepExpiredGames();
}, BLAST_TOTAL_DURATION_MS); // sweep once per full game duration
```

**问题:** 清理间隔 = 完整游戏时长（可能数小时）。高频 Blast 场景下，内存可能持续增长到数小时后才清理。

**建议:** 考虑缩短间隔至 `BLAST_TOTAL_DURATION_MS / 2` 或固定值（如 30 分钟），在 `sweepExpiredGames()` 中正确处理 `Date.now() > game.endsAt` 即可。

**严重程度:** 低
