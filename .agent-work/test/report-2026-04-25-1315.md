### 🟡 P2 — 新发现问题

**P2-CLUB-007**: `createRoom` 缺少 `isClubOnly + clubId` 一致性强校验

- **位置**: `apps/backend/src/room/room.service.ts:15-34`
- **问题**: DTO 接受 `isClubOnly` 和 `clubId` 字段并透传到 Prisma，但 `createRoom` 方法未校验：
  - `isClubOnly=true` 时必须提供有效 `clubId`
  - `clubId` 存在时 `isClubOnly` 必须为 `true`
- **风险**: 用户可创建 `isClubOnly=true` 但 `clubId=null` 的房间，导致数据不一致
- **建议**: 在 `createRoom` 方法开头添加一致性校验：

```ts
if (data.isClubOnly && !data.clubId) {
  throw new BadRequestException("clubId required when isClubOnly is true");
}
if (data.clubId && !data.isClubOnly) {
  data.isClubOnly = true; // 隐式设置
}
```
