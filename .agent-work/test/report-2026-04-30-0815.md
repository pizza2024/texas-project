### ⚠️ P2-NOTIFY-001-001 (P1) — NotificationGateway room join 缺失（未修复）

**文件:** `apps/backend/src/notification/notification.gateway.ts:76`

```typescript
emitToUser(userId: string, notification: unknown) {
  this.server.to(`user:${userId}`).emit('notification', notification);
}
```

**问题确认:** `handleConnection` 中从未调用 `client.join('user:${userId}')`，WebSocket 通知无法送达。

**变通:** `app.gateway.ts` 好友上线通知使用 `wsManager.emitToUser`（直接 socket 发送），绕过 NotificationGateway，所以实际通知功能仍可用。但 `NotificationGateway.emitToUser` 独立使用时会失效。

**建议:** 在 `handleConnection` 末尾添加：
```typescript
await client.join(`user:${payload.userId}`);
```
