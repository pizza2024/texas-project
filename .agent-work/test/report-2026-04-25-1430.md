### 🔴 建议升 P1 (3项)

#### P2→P1-NEW-002 — `client.leave()` 在 DB 更新之后（竞态条件）

**文件:** `apps/backend/src/websocket/game.handler.ts:330-340`

```typescript
// line 330-335: leaveCurrentRoom 更新 DB
const result = await gateway.tableManager.leaveCurrentRoom(userId);
// ...
gateway.matchmakingService.recordPlayerLeft(roomId, userId);
client.leave(roomId);  // ← 在 DB 更新之后
```

**问题:** 如果客户端在 `leaveCurrentRoom`（更新DB）和 `client.leave()`（socket离开）之间断连，另一个玩家可能在此窗口内加入房间，导致该玩家仍在房间里但socket已离开。

**建议修复顺序:**
```typescript
client.leave(roomId);  // 先离开 socket room
gateway.matchmakingService.recordPlayerLeft(roomId, userId);
await gateway.tableManager.leaveCurrentRoom(userId);  // 再更新 DB
```

#### P2→P1-NEW-003 — `handlePlayerAction` table-not-found 静默失败

```typescript
const table = await gateway.tableManager.getTable(roomId);
if (!table) return;  // ← 静默返回，无任何事件通知
```

**问题:** 玩家发起操作但 table 不存在时，完全没有反馈（无 `error` event，无 `action_rejected`）。玩家界面上操作"消失"，无法判断是网络问题还是牌桌问题。

**建议:** 应 emit `action_rejected` 或 `error` 事件，让玩家知情。

#### P2→P1-NEW-001 — `generateInviteCode()` 使用 `Math.random()`

**文件:** `apps/backend/src/club/club.service.ts:77`

```typescript
private generateInviteCode(): string {
  let code = '';
  for (let i = 0; i < 6; i++) {
    code +=
      this.INVITE_CHARS[Math.floor(Math.random() * this.INVITE_CHARS.length)];
  }
  return code;
}
```

**问题:** `Math.random()` 不是密码学安全的随机数生成器。邀请码可预测，攻击者可暴力枚举 6 位字母数字空间（36^6 ≈ 21 亿）。

**建议:** 使用 `crypto.randomInt()` 或 `randomBytes()` 替代。
