```
apps/web/app/rakeback/page.tsx(458,59): error TS7006: Parameter 't' implicitly has an 'any' type.
```

**根因：** `apps/web/tsconfig.json` 缺少 `paths` mapping for `@texas/shared`。
`jest-e2e.json` 的 `moduleNameMapper` 修复只覆盖了测试运行时，不影响 `tsc --noEmit`。
`packages/shared` 导出正常（backend 编译无错误）。

**修复方案：** 在 `apps/web/tsconfig.json` 的 `compilerOptions.paths` 添加：

```json
"@texas/shared": ["../../packages/shared/src"]
```

**严重度：** P2（不影响运行时，Jest 测试全通过，Next.js 构建时 Next.js 自己解析路径）
