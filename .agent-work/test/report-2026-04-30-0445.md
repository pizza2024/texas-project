### ⚠️ 遗留未关闭问题

#### 1. `BOT_ID_PREFIX` 前缀碰撞 — P1 建议

**文件:** `apps/backend/src/wallet/wallet.service.ts:95`

```typescript
if (userId.startsWith(BOT_ID_PREFIX)) {
  return [walletOp]; // Skips coinBalance sync for "bots"
}
```

**风险:** 真实用户 ID 以 `"bot_"` 开头时，`User.coinBalance` 永远不会被 `setBalances` 同步，`getAvailableBalance` 返回错误可用余额。

**建议:** 使用 `User.role === 'BOT'` 判断替代字符串前缀匹配。

**优先级:** P1（用户资金正确性受损，当前无阻塞 P0/P1）

#### 2. `client.leave()` 在 `leaveCurrentRoom` 之前 — P2

**文件:** `apps/backend/src/websocket/game.handler.ts:387-391`

```typescript
return gateway.withRoomLock(roomId, async () => {
  client.leave(roomId);                         // ← socket 离开 room
  const result = await gateway.tableManager.leaveCurrentRoom(userId);  // ← DB 更新之后
```

**风险:** 两步之间如有异常，socket 已离开 room 但 DB 未更新，极小 race window。

**建议:** 用 try/finally 确保一致性，或将 `client.leave()` 移至 `leaveCurrentRoom` 成功之后。

**优先级:** P2（race window 极小）
