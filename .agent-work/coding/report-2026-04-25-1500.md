## 改动说明

### `GET /hands/:id/replay` — Phase 1 后端实现

**新增文件/模块:**
- `HandController` (`table-engine.controller.ts` 新增)
  - 路由: `GET /hands/:id/replay`
  - 调用 `HandHistoryService.getHandReplay()`
  - 无认证（公开接口，可按需加 JWT）

**新增方法:**
- `HandHistoryService.getHandReplay(handId)` (`hand-history.service.ts`)
  - 根据 `handId` 查询完整 `Hand` 记录（含 `table.room`、`actions`、`settlements`）
  - 解析 `stateSnapshot` 获取:
    - 每位玩家的手牌 (`holeCards`)
    - 公共牌 (`communityCards`)
    - 盲注大小 (`smallBlind`/`bigBlind`)
  - 构建 `ReplayActionNode[]` 时间轴，每节点包含:
    - `playerId`、`playerNickname`、`action`、`amount`
    - `totalBet`（本局累计投入）、`remainingStack`（剩余筹码）
    - `potAfter`（当时底池大小）、`communityCards`（当时可见公共牌）
    - `stage`（preflop/flop/turn/river/showdown）、`description`（中文描述）
  - 插入阶段转换节点（翻牌/转牌/河牌/摊牌）
  - 返回 `HandReplayData`:
    - `handId`、`roomName`、`date`
    - `smallBlind`、`bigBlind`、`initialPot`、`finalPot`
    - `players[]`（手牌 + 净盈亏 + 最终牌型）
    - `communityCards`、`winnerId`
    - `timeline`（完整事件时间轴）

**模块注册:**
- `TableEngineModule` — 新增 `HandController`
