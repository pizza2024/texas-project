**修复:**

1. **`table-manager.service.ts`** — 在 `persistSettlementRecords()` 成功分支中（结算事务提交后）触发四个 mission 事件：

2. **依赖注入** — `MissionService` + `DepositService` 作为 constructor 参数注入 `TableManagerService`

3. **Module wiring** — `MissionModule` + `DepositModule` 添加到 `TableEngineModule` imports

4. **代码清理** — `timer.service.ts` 移除未使用的 `MISSION_KEY_FIRST_HAND` import
