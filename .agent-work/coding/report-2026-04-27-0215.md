### P2-CODE-PATTERN — mission.service.ts Promise.allSettled 优化 ✅

**文件:** `apps/backend/src/mission/mission.service.ts:170`

**问题:** `seedMissions()` 使用 `Promise.all`，一个 upsert 失败会导致整体 reject，无法记录部分成功。

**修复:** 改用 `Promise.allSettled`，记录失败数量并逐条打印错误原因。

```typescript
// 修复前
await Promise.all(MISSION_DEFINITIONS.map(...));  // 一个失败全部 reject

// 修复后
const results = await Promise.allSettled(MISSION_DEFINITIONS.map(...));
const failures = results.filter((r) => r.status === 'rejected');
if (failures.length > 0) {
  this.logger.error(`seedMissions: ${failures.length}/${MISSION_DEFINITIONS.length} upserts failed`);
  for (const f of failures) { this.logger.error((f as PromiseRejectedResult).reason); }
} else {
  this.logger.log(`Missions seeded (${MISSION_DEFINITIONS.length} upserts)`);
}
```
