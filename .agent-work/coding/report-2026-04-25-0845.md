  - `code` 字段 UNIQUE index 防重

#### Backend — API 层

- **ClubController** 新增5个端点：
  | 方法 | 路径 | 权限 | 说明 |
  |------|------|------|------|
  | POST | `/clubs/:id/invite-codes` | Owner/Admin | 创建邀请码 |
  | GET | `/clubs/:id/invite-codes` | Owner/Admin | 列出所有邀请码 |
  | DELETE | `/clubs/:id/invite-codes/:codeId` | Owner/Admin | 撤销邀请码 |
  | POST | `/clubs/join-by-code` | 登录用户 | 凭码加入 Club |
  | GET | `/clubs/validate-code` | 公开 | 验证邀请码有效性 |
- **ClubService** 新增方法：
  - `generateInviteCode()`: 6位字符，字符集 ABCDEFGHJKMNPQRSTUVWXYZ23456789（排除0/O/1/I/L）
  - `createInviteCode()`: 生成唯一码，DB unique index + 重试
  - `listInviteCodes()`: 返回码列表（含使用情况）
  - `deleteInviteCode()`: 撤销码
  - `validateInviteCode()`: 公开验证（防枚举错误提示）
  - `joinByCode()`: 加入 Club + 原子 increment usedCount
- **DTOs**:
  - `CreateInviteCodeDto`: maxUses(1-100, 默认5), expiresInHours(1-168, 默认72)
  - `JoinByCodeDto`: 6位码正则校验
