## 代码改动详情

### 1. ActionBar.tsx — 新增 Pot-Relative 预设按钮行

在 Fold/Check/Call 主操作按钮行和 Raise Input 行之间，新增了一行 4 个快捷按钮：

| 按钮 | 计算公式 | 样式 |
|------|---------|------|
| **Min** | `min(minRaiseTo, myPlayerStack)` | 琥珀色边框，黑色背景 |
| **½ Pot** | `min(floor(pot × 0.5), myPlayerStack)` | 琥珀色边框，黑色背景 |
| **¾ Pot** | `min(floor(pot × 0.75), myPlayerStack)` | 琥珀色边框，黑色背景 |
| **All-in** | `myPlayerStack` | 红色边框/背景（激活时） |

**行为**：
- 点击按钮：直接执行加注（`setRaiseAmount` + `handleAction('raise', amount)`）
- 按钮-disabled 条件：`preset < minRaiseTo`（低于最低加注）或 `preset > myPlayerStack`（超出玩家筹码）
- 仅在 `isMyTurn` 时显示
- 每个按钮显示两个标签行：主标签（如 "Min"）+ 金额（如 "200"）

### 2. ActionBar.tsx — Raise Input 上限保护

```tsx
// Before

// After

// Display clamped value
value={Math.min(raiseAmount || minRaiseTo, myPlayerStack)}
```
