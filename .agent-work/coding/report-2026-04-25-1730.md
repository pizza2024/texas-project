## 本轮处理

### P1-HANDREPLAY-002 — Hand Replay Phase 2 ✅ 已实现

根据 Productor 第147轮规格，实现了以下5个组件：

#### 1. EquityCurveChart (`apps/web/components/replay/EquityCurveChart.tsx`)
- Recharts `LineChart` 折线图，展示各玩家 equity 随回合变化
- 颜色方案：amber/blue/green/red/purple/pink/cyan/orange 8色轮换
- X轴：PREFLOP → FLOP → TURN → RIVER → SHOWDOWN
- Y轴：0-100% equity
- 折叠面板，点击 header "📊 Equity" 按钮展开/收起

#### 2. PotOddsTooltip (`apps/web/components/replay/PotOddsTooltip.tsx`)
- 悬停浮窗，显示 pot odds 计算和跟注建议
- 参数：Call金额、当前pot、Odds百分比
- 颜色编码：≤33%绿（✓ Good call）、34-50%黄（◐ Marginal）、>50%红（✗ Overpriced）
- 基于 `anchorRef` 自动定位或手动 position

#### 3. PlaybackControls (`apps/web/components/replay/PlaybackControls.tsx`)
- 播放/暂停按钮（圆角大按钮，amber色）
- 快退/快进按钮（‹‹ ››，跳5步）
- 集成于 AutoPlayPanel 内部（不从 ReplayTimeline 替换）

#### 4. SpeedSelector (`apps/web/components/replay/SpeedSelector.tsx`)
- 0.5× / 1× / 2× / 4× 速度选择（单选按钮组）
- amber 高亮当前选中速度
- 导出 `PlaybackSpeed` 类型

#### 5. AutoPlayPanel (`apps/web/components/replay/AutoPlayPanel.tsx`)
- 自动播放 toggle 开关（滑动样式）
- 速度选择器集成
- "⏩ To Showdown" 按钮：跳至最后一帧并停止自动播放
- 自动播放 interval 基于速度：0.5×=2s, 1×=1s, 2×=500ms, 4×=250ms

#### ReplayModal 集成
- 新增 state：`isPlaying`, `playbackSpeed`, `showEquityChart`, `hoveredActionNode`
- `onHoverNode` 传入 ReplayActionLog（新增 prop）
- Header 新增 "📊 Equity" 按钮
- `handleIndexChange` 支持 functional setState（供 AutoPlayPanel 使用）

#### ReplayActionLog 改动
- 新增 `onHoverNode?: (node: ReplayActionNode | null) => void` prop
- `onMouseEnter`/`onMouseLeave` 事件触发 hover 状态
