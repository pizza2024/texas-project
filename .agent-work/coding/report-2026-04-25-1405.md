### P2-CLUB-003: Club 专属房间前端权限过滤缺失 ✅ 已修复

**问题描述:**

- `/rooms` 页面会展示所有房间（包括 Club 专属房间 `isClubOnly=true`）
- 非 Club 成员可以看到 Club 专属房间，但无法加入（后端 WebSocket 层正确拦截）
- 体验问题：用户看到房间却无法加入

**修复内容:**

#### 1. `packages/shared/src/types/game.ts` — Room 接口扩展

```ts
export interface Room {
  // ...existing fields
  isClubOnly?: boolean; // 新增
  clubId?: string; // 新增
}
```

#### 2. `apps/backend/src/room/room.controller.ts` — 无需修改

- `findAll` 和 `findOne` 通过 `...room` spread 已自动包含 `isClubOnly` 和 `clubId` 字段

#### 3. `apps/web/app/rooms/page.tsx` — 核心过滤逻辑

- 新增 `myClubIds` state，存储用户所属 Club ID 列表
- 并行请求 `GET /clubs/me/clubs` 获取用户俱乐部列表
- `filteredRooms` 增加过滤：Club 专属房间且用户不在该 Club → 不展示

```ts
// 过滤逻辑
.filter((room) => {
  if (room.isClubOnly && room.clubId && !myClubIds.includes(room.clubId)) {
    return false;
  }
  return true;
})
```
