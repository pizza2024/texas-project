**问题:** `handleEmojiReaction` broadcast 的 payload 中缺少 `roomId` 字段，导致前端多房间场景下 emoji 过滤失效。

**改动文件:** `apps/backend/src/websocket/game.handler.ts:607`

**改动内容:** 在 `emoji-reaction` broadcast payload 中添加 `roomId` 字段，使前端能正确过滤多房间场景。

```ts
// Before
gateway.server.to(roomId).emit('emoji-reaction', {
  id: crypto.randomUUID(),
  userId,
  username,
  emoji: validated.emoji,
  timestamp: Date.now(),
});

// After (添加 roomId)
gateway.server.to(roomId).emit('emoji-reaction', {
  id: crypto.randomUUID(),
  userId,
  username,
  emoji: validated.emoji,
  roomId,          // ← 新增
  timestamp: Date.now(),
});
```
