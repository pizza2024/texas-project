### P2-CODE-033 — Blast 清理间隔过长 ✅ 已修复

**文件:** `apps/backend/src/tournament/blast.service.ts:95-97`

**问题:** `cleanupInterval` 使用 `BLAST_TOTAL_DURATION_MS`（180秒），等于完整游戏时长。实际游戏往往提前结束（如首轮全押），导致内存清理延迟过长。

**修复:** 将清理间隔从 180 秒缩短至 30 秒：
```typescript
// P2-CODE-033: sweep every 30s instead of full game duration (180s)
// to reclaim memory faster for short-lived games (e.g., all-in early)
this.cleanupInterval = setInterval(() => {
  this.sweepExpiredGames();
}, 30_000); // sweep every 30 seconds
```
