### ✅ P0-NEW-001 — TooManyRequestsException 编译失败
**文件:** `apps/backend/src/auth/auth.service.ts`

**问题:** `@nestjs/common@11.1.14` 未导出 `TooManyRequestsException`，运行时抛出 `undefined is not a constructor`

**修复:**
- 移除 `TooManyRequestsException` 导入
- 替换为 `BadRequestException`（语义相近，HTTP 429 → 400）

```typescript

// After
throw new BadRequestException('...');
```
