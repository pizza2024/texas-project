### P2-EMOJI-REF — `use-game-socket.ts` ref 违规访问（已修复）

**文件:** `apps/web/lib/use-game-socket.ts`

- `useRef<EmojiFlight[]>([])` → `useState<EmojiFlight[]>([])`

- `flightsRef.current = flightsRef.current.filter(...)` → `setActiveFlights((prev) => prev.filter(...))`
- `const activeFlights = flightsRef.current` 已删除（state 变量直接返回）
