## 本轮修复 — P2-EMOJI-REF

**Bug:** `use-game-socket.ts` 中 `activeFlights` 从 `useRef` 读取，但 ESLint `react-hooks/refs` 规则禁止在渲染期间访问 ref — 导致 `flightsRef.current` 始终返回初始空数组，emoji reaction UI 从不显示。

**Fix:** 将 `useRef<EmojiFlight[]>([])` 替换为 `useState<EmojiFlight[]>([])`，在 socket handler 中使用 `setActiveFlights` 更新状态，每次增删 flight 都触发正确的 React 重渲染。

**文件:** `apps/web/lib/use-game-socket.ts`

- 移除 `useRef`，导入 `useState`
- `activeFlights` 从 state 读取
- `setActiveFlights` 在 handler 和 `removeFlight` 中调用
