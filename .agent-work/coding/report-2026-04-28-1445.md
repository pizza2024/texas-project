### P1-BLAST-4-SOCKET — socket 未导出 ✅ 已修复

**问题:** `useBlastSocket.ts` 使用 `import { socket } from "@texas/shared/socket"`，但 `socket.ts` 只导出 `getSocket()`，不导出裸 `socket`。

**根因:** `@texas/shared/socket` 中的 `socket` 是内部模块变量（`let socket: Socket ...`），不对外暴露。

**修复:**

- `apps/web/hooks/useBlastSocket.ts` — 将 `socket` import 替换为 `getSocket` from `@/lib/socket` + `getStoredToken` from `@/lib/auth`
- `packages/shared/src/socket.ts` — `ClientToServerEvents` 添加 `join-blast-lobby` 和 `leave-blast-lobby` 类型（解决 TS 报错 "Argument of type 'join-blast-lobby' is not assignable to parameter of type ..."）

**关键 pattern:** Web 侧所有组件都通过 `@/lib/socket` 调用 `getSocket(token)` 获取 socket 实例，而非直接用内部单例。
