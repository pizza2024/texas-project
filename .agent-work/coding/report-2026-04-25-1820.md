## 处理的任务

### P1-BACKEND-003 ✅ 已修复
**文件**: `apps/backend/src/websocket/game.handler.ts:290-296`

**问题**: `player-action` 处理中，当 `roomId` 为 null（用户不在任何房间）时，错误消息为 "Invalid action or roomId"，无法区分是 action 非法还是用户未入座。前端无法针对性处理。

**修复**:
```typescript
// 修复前
if (!action || !roomId) {
  client.emit('error', { message: 'Invalid action or roomId' });
  return;
}

// 修复后
if (!action) {
  client.emit('error', { message: 'Invalid action' });
  return;
}
if (!roomId) {
  client.emit('error', { message: 'not_in_any_room' });
  return;
}
```

**效果**: 前端可区分 action 校验失败 vs 用户未入座，针对性显示提示。
