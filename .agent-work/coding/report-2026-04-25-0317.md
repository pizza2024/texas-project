**分析过程**:
1. 运行 `npm test -- --detectOpenHandles` — 238 tests 全部通过，警告间歇出现（有时有，有时无）
2. 扫描 `apps/backend/src` 中 `setTimeout`/`setInterval` 使用 — 0 个直接调用（全部通过 `timer.service.ts` 注入）

**结论**: P1-002 为非阻塞问题，间歇性出现，不影响测试覆盖率和功能正确性。
