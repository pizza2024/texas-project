### ✅ P2-WEB-JEST-001 — Web jest `@texas/shared` mock 解析失败

**问题:** `jest.mock('@texas/shared')` 在 jest.setup.ts 中无法解析。tsconfig.json 的 `paths` 映射 (`@texas/shared` → `../../packages/shared/src`) 仅作用于 TypeScript 编译器，Jest 的 `moduleNameMapper` 需要显式配置。

**根因:** `apps/web/jest.config.ts` 的 `moduleNameMapper` 缺少 `@texas/shared` 映射。

**修复:**

```diff
  moduleNameMapper: {
    '^@/(.*)$': '<rootDir>/$1',
+   '^@texas/shared$': '<rootDir>/../../packages/shared/src/index.ts',
    '\\.(css|less|scss|sass)$': 'identity-obj-proxy',
  },
```
