### P2 — [W-003] 房间密码暴力破解防护 — 已完成

**文件**:

- `apps/backend/src/websocket/app.gateway.ts` — 新增 brute-force 保护方法
- `apps/backend/src/websocket/game.handler.ts` — 在 join_room handler 中集成检查

**问题**: `wrong_password` 事件无日志无封禁，攻击者可无限尝试房间密码。

**修复**:

1. **In-memory 追踪器** (`AppGateway`):
   - `checkPasswordAttemptLimit(ipHash, roomId)` — 5分钟内超过5次错误密码则 ban
   - `clearPasswordAttempts(ipHash, roomId)` — 成功验证后清除计数器
   - 使用 `ipHash = matchmakingService.hashIp(rawIp)` 而非原始 IP（保护隐私）
   - 日志前缀 `[SECURITY-BRUTE-FORCE]` 便于 SIEM 告警规则匹配

2. **集成点** (`handleJoinRoom`):
   - 在 bcrypt 密码比对前先检查 brute-force 状态
   - 密码正确 → `clearPasswordAttempts` 清除计数器
   - 密码错误 → 计数器累加

**参数**:

- 窗口: 5分钟
- 最大尝试: 5次
- 封禁时长: 5分钟（窗口重置）
