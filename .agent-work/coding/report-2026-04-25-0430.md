## 处理的任务

### ✅ P2-NEW-002 — creditRakeback 事务语法修复

**文件**: `apps/backend/src/rakeback/rakeback.service.ts`

**问题**: Prisma `$transaction([])` 数组语法已废弃，应使用 callback 语法。

**改动**:
```typescript
// Before（废弃语法）
await this.prisma.$transaction([
  this.prisma.user.update({ ... }),
]);

// After（推荐语法）
await this.prisma.$transaction(async (tx) => {
  await tx.user.update({ ... });
});
```

**验证**: ✅ 17/17 rakeback tests passed
