### ✅ P1-CHAT-INJECTION — Chat Username 身份冒充（已修复）

**文件:** `apps/backend/src/websocket/game.handler.ts` 行 556-559

**问题:** `handleChatMessage` 中 `username` 直接取自 `client.data.user.username`（JWT payload），攻击者可伪造任意用户名冒充他人身份。

**修复:**
```ts
// 修复前（不安全）
const username =
  (client.data.user as { username?: string } | undefined)?.username ??
  '未知玩家';

// 修复后（安全）
const dbUser = await gateway.userService.user({ id: userId });
const username = dbUser?.username ?? '未知玩家';
```
