**Migration**: `202604250138_add_first_deposit_bonus`

**Service** (`deposit.service.ts` - `checkAddressDeposits`):

- 存款入账后检查 `hasReceivedFirstDepositBonus`
- 未领取过则计算 bonus：`min(depositUsdt, 100) * 100` chips
- 信用 bonus 筹码并标记用户为已领取
- 记录 `Transaction` type=`'BONUS'`

**逻辑**:

```typescript
if (user && !user.hasReceivedFirstDepositBonus) {
  const bonusUsdt = Math.min(amount.toNumber(), 100);
  const bonusChips = bonusUsdt * USDT_TO_CHIPS_RATE;
  await walletService.setBalance(userId, currentBalance + bonusChips);
  await prisma.user.update({ data: { hasReceivedFirstDepositBonus: true } });
  await prisma.transaction.create({
    data: { type: "BONUS", amount: bonusChips },
  });
}
```
