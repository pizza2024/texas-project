## 处理的任务

### ✅ P1-BLAST-PRIVATE-JOIN — Private Blast Lobby 无法被加入

**问题:** `POST /rooms/blast/:id/join` 缺少 `password` 参数。`CreateBlastLobbyDto` 支持创建密码保护的私人房间（`password?: string`），但 `joinBlastLobby` 无法接收密码，导致私人房间永远无法被其他玩家加入。

**修复:**

1. **tournament.controller.ts** — `joinBlastLobby` 添加 `@Body('password') password: string | undefined`，传递给 service
2. **tournament.service.ts** — `joinBlastLobby` 添加 `password?: string` 参数，在允许加入前校验密码：
   - 若 `lobby.password !== undefined`（私人房间），则 `password === lobby.password` 才允许加入
   - 密码错误返回 `null`，controller 抛出 `BadRequestException`

**验证:**

- ESLint: ✅ passed
- Tests: ✅ 62 tournament tests passed (tournament.service.spec.ts + blast.service.spec.ts)
- `BlastLobby.password` 字段已存在于 `@texas/shared` 类型定义中，create 时已正确存储
